# /agency-hire

## Purpose
Build a complete hiring plan for the user's next agency hire.

## Skills Required
Load in order: `scaling-team`, `delivery-operations`

## Step-by-Step Execution

### Step 1: Gather Context
Ask the user:
1. Current team size (full-time + freelancers) and their roles
2. Current monthly revenue
3. Number of active clients
4. What is your biggest bottleneck? (delivery / sales / admin / ops)
5. What tasks are consuming most of YOUR time right now?
6. Budget for this hire (monthly)
7. Preference: full-time, part-time, or freelancer?
8. Do you already have a specific role in mind?

### Step 2: Function Chart Analysis
Apply Dee Deng's function chart logic:
- List all functions the user is currently performing (strategic leadership, finance, operations, admin, payroll, client services, sales, marketing)
- Identify which function creates the most leverage if delegated
- Apply Donvesh's first-hire principle: first hire should always be a VA/admin to free up time for revenue-generating activities
- Then use Dee's framework to identify which role creates the most leverage for the business

### Step 3: Role Recommendation
Based on analysis, recommend:
- **If pre-$10K/month:** VA or freelancer (Donvesh's first-hire principle)
- **If $10K-$30K:** Role that fills biggest skill gap, complementary to founder (Shan's principle)
- **If $30K+:** Follow function chart priorities, hire missionary for core function (Dee's framework)
- **If sales bottleneck:** SDR with commission structure (Donvesh's tiered model)
- **If founder spends 50%+ on admin:** VA/Admin regardless of revenue stage

Determine hire type:
- Overflow/project work -> Mercenary (freelancer/contractor)
- Core ongoing function -> Missionary (full-time, culture-fit)

Validate financial feasibility:
- Does profit-per-team support this hire? (reference agency-finance)
- Will cash buffer remain at 3+ months after hire?

### Step 4: Generate JD
Use Shan's JD structure:
1. Job title and overview
2. About the agency (2-3 sentences: mission, culture)
3. What you will do (5-7 specific responsibilities)
4. About you (requirements and nice-to-haves)
5. Application questions (2 questions that filter for quality and reveal thinking)
6. Practical task (test of relevant skills -- must take 30-60 minutes and mirror real work)

### Step 5: Application Task Design
Design a practical assessment (Shan's approach):
- Must test actual ability, not just CV claims
- Should take 30-60 minutes to complete
- Should mirror real work they would do in the role
- Examples by role type:
  - Lead gen exec: Source 3 prospects and write an outreach email for each
  - Media buyer: Create a 30-day campaign strategy for a specific brand
  - Copywriter: Write 3 subject lines and opening paragraph for a prospect
  - VA/Admin: Organize a messy spreadsheet and flag issues
  - SDR: Record a mock cold call for a specific prospect type
  - Designer: Create a 1-slide mockup presenting a specific metric

### Step 6: Interview Process
Use Dee Deng's 6-step framework:
1. **Application screening** -- Filter for effort, research into company, quality of task
2. **Culture fit interview** (standardized questions):
   - "What can we do to be part of your personal growth journey?"
   - Handling disagreements question
   - Work environment preferences
   - Role-specific values question
3. **Skills assessment** -- Real-world challenge, time-boxed 24-48 hours
4. **Tandem interview** -- 2 interviewers (one asks, one notes)
5. **Offer + onboarding** -- Clear offer with KPIs, responsibilities, development pathway
6. **30/60/90-day probation** with structured check-ins

Include red flags to watch for (Shan's bad hiring lesson: corporate backgrounds often cannot adapt to agency pace; test skills, not interview performance).

### Step 7: Platform Recommendation
Based on role type (from Shan's 20-platform list):
- **VA/Admin:** OnlineJobs.ph (Filipino talent, free up to 3 jobs/month)
- **Core team (US/UK/EU):** LinkedIn (min $5/day to promote)
- **Remote roles in specific locations:** Indeed
- **Project-based freelancers:** Fiverr, Upwork
- **Top-tier specialists:** Toptal (top 3% freelancers)
- **Sales/SDR:** Upwork at $10-$15/hour (Donvesh), Overpass
- **Creative/Design:** Dribbble Jobs
- **Commission closers:** Closify (no upfront cost)

### Step 8: Onboarding & Trial Plan
Generate:
1. **Trial period** (1-2 weeks, paid): specific deliverables, evaluation criteria, decision point
2. **Team welcome email** using Shan's template (tool access, Slack, Asana task list, first-day schedule)
3. **First 2 weeks onboarding checklist:**
   - Day 1: Welcome email, tool access, team introductions
   - Week 1: Manager 1-1, initial project, SOP review, culture orientation
   - Week 2: First real deliverable, feedback session, 30-day goal setting
4. **30/60/90-day milestones** for probation review

### Step 9: Output Complete Hiring Plan
Deliver as a structured document:
1. **Recommended role** with justification (function chart analysis)
2. **Hire type** (missionary vs mercenary) with reasoning
3. **Financial validation** (affordability check)
4. **Full JD** (ready to post)
5. **Application task**
6. **Interview framework** (screening + culture + skills assessment)
7. **Platform recommendation** with posting instructions
8. **Trial period structure** with evaluation criteria
9. **Onboarding checklist** (first 2 weeks)
10. **30/60/90-day milestones**
11. **Estimated timeline:** Posting -> Screening -> Interviews -> Trial -> Decision (typically 3-6 weeks)

If relevant, note the talent-sales staircase dynamic: hiring temporarily increases costs (under-capacity phase), so sales pressure increases to fill the gap. Advise the user to plan for this.
