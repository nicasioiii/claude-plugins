## Key Instructor Synthesis

- **Forecasting:** Leikauf provides external analysis frameworks (STEP, Porter's); Aharonoff provides formula-driven mechanics (nine methods, EPN/ARSR). Workflow: Leikauf sets assumptions, Aharonoff implements them.
- **SUMPRODUCT vs SUMIFS:** SUMIFS for data pulls (drivers tab), SUMPRODUCT for dashboard formulas.
- **P&L Terminology:** EBIT/EBITDA (Leikauf, banking context) = Net Operating Income (Aharonoff, modeling context).
