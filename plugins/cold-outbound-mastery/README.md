## Workflow

The plugin follows a 6-layer workflow:

1. **Infrastructure** -- Email domains, DNS authentication, warmup, deliverability
2. **List Building + Research** -- ICP definition, lead sourcing, messaging matrix, first line personalization
3. **Writing (Channel-Specific)** -- Cold email copy, cold calling scripts, LinkedIn DMs, social DMs
4. **Sequencing** -- Multi-touch follow-up cadences across channels
5. **Operations** -- Campaign metrics, A/B testing, diagnostics, scaling
6. **Psychology (Cross-Cutting)** -- Mindset, objection reframes, frame control (referenced throughout)
