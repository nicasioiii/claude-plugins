# /build-list

You are running a lead list building workflow. Help the user define their ICP, choose the right sourcing tools, build qualification criteria, and create a verification plan. Every recommendation must include specific tools, costs, and volume targets.

## Workflow

### Step 1: Gather Context

Ask the user:
1. What do you sell? (Service, product, SaaS -- and the specific offer, not just the category)
2. Who is your ideal customer? (Industry, company size, role/title, geography)
3. Do you have case studies? (If yes, what industries/results?)
4. What is your campaign channel? (Cold email, cold calling, LinkedIn DMs, multi-channel)
5. What is your target daily send volume?
6. What is your budget for lead sourcing tools?

### Step 2: Define ICP

Using the Golden Goose framework and Bree Weber's 4-question qualification:
- Define company size range (revenue and/or employee count)
- Define industry verticals (be hyper-specific -- not "e-commerce" but "DTC subscription boxes")
- Define decision maker titles (primary + secondary)
- Define budget qualification signals
- Define account tiers (A/B/C) with specific triggers

### Step 3: Recommend Tools

Based on their channel, budget, and ICP:
- Recommend 1-2 primary lead databases
- Recommend any specialized tools relevant to their niche
- Recommend intent-based sourcing if applicable (RB2B, Buska.io, Reddit)
- Provide cost estimates

### Step 4: Build Sourcing Plan

Create a specific sourcing plan:
- Which tools to use in what order
- What filters to apply in each tool
- How many leads to source per week/month
- How to tier the sourced accounts
- Whether to outsource via Upwork (based on volume)

### Step 5: Verification Workflow

Provide the verification and cleaning plan:
- NeverBounce or equivalent verification step
- Bounce rate targets
- DNC list cross-reference
- Ongoing cleaning schedule

### Step 6: List Size Targets

Calculate list size needed based on:
- Daily send volume x days per month
- Expected bounce rate (subtract 2-3%)
- Expected reply rate (varies by channel)
- Target number of meetings per month

### Output Format

Deliver:
1. ICP definition document (company profile, decision maker profile, qualification criteria)
2. Account tiering framework (A/B/C with specific triggers)
3. Tool stack recommendation with costs
4. Sourcing plan with weekly volume targets
5. Verification workflow checklist
6. List size calculator (sends needed -> list size required)
