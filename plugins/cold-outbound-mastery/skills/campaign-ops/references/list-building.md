## Step-by-Step: Lead Generation Process

### Step 1: Define ICP (Ideal Customer Profile)

Before you source a single lead, define your target:

**Company Level:**
- [ ] Company size: Employee count (e.g., "50-500")
- [ ] Industry/Vertical: (e.g., "SaaS", "Ecommerce", "Healthcare Tech")
- [ ] Geography: (e.g., "US only", "US + Canada")
- [ ] Revenue range: (if known, e.g., "$1M-10M ARR")
- [ ] Recent signals: (e.g., "Recently funded", "Hiring", "Public SEC filings")

**Role Level:**
- [ ] Job title(s): (e.g., "CMO", "Head of Sales", "Ops Manager")
- [ ] Department: (e.g., "Sales", "Marketing", "Operations")
- [ ] Experience level: (e.g., "VP+ only", "Manager and up")

**Use Case Level:**
- [ ] Problem they have: (e.g., "Low email deliverability", "High CAC")
- [ ] Related activities: (e.g., "Using Salesforce", "Hiring SDRs", "Recent website changes")

**Example ICP:**
"CMOs and Marketing Directors (3+ years experience) at SaaS companies ($5M-50M ARR) in the US, founded 2010+, who've recently hired marketing or changed marketing tech stack"

### Step 2: Choose Lead Source

**Recommended flow:**
1. **UpLead (if targeting by company attributes)**
   - Best for: Finding all engineers in Toronto, or all CEOs at Series B companies
   - Process: Use UpLead filters, export CSV, validate

2. **Apollo.io or Seamless (if bulk prospecting)**
   - Best for: Finding 5000+ leads matching criteria
   - Process: Set filters, export, validate in NeverBounce

3. **LinkedIn Sales Navigator (if warm relationship focus)**
   - Best for: Finding specific people you want to research deeply
   - Process: Manual search, view profiles, take notes

4. **Manual LinkedIn (if niche/relationship-based)**
   - Best for: Ultra-targeted lists, founder outreach, relationship building
   - Process: Search, view 100-200 profiles, export names manually

### Step 3: Source Leads (Platform-Specific Steps)

#### Using UpLead

**Pros:** Verified database, transparent pricing, high accuracy
**Cons:** Per-lead cost adds up at scale

**Process:**
1. Go to UpLead.com → Create account
2. Set filters:
   - Company size (e.g., "50-500 employees")
   - Industry (e.g., "Software")
   - Location (e.g., "United States")
   - Job title (e.g., "Marketing Director")
3. Search and view list count (e.g., "15,000 results")
4. Select leads (usually charged per lead, $0.50-1.50 each)
5. Export as CSV with columns: First name, Last name, Email, Title, Company, Phone
6. Download and validate

#### Using Apollo.io

**Pros:** Unlimited leads once subscribed ($99/mo), integrates with Zapier/Make, native CRM integration
**Cons:** Less accurate than UpLead, many invalid emails (need validation)

**Process:**
1. Go to Apollo.io → Create account ($99/mo)
2. Click "People Search"
3. Add filters:
   - Title (e.g., "Marketing Director")
   - Company size (e.g., "50-500 employees")
   - Location (e.g., "United States")
   - Industry (optional)
4. View total matches (e.g., "25,000 results")
5. Click "Save to a list" or "Export"
6. Can export directly to CRM (HubSpot, Salesforce) or download CSV
7. Move to validation step (next)

#### Using LinkedIn Sales Navigator

**Pros:** Best research quality, see engagement signals, very targeted
**Cons:** Manual process, slow, limited exports (50 per day)

**Process:**
1. Subscribe to Sales Navigator ($99-165/mo)
2. Search people by:
   - Title (e.g., "Marketing Director")
   - Company (e.g., "Acme Corp")
   - Location (e.g., "San Francisco Bay Area")
   - Add to list
3. View each profile for engagement/research signals
4. Can take notes on profile
5. Export to CRM or note down manually
6. Use for relationship-based outreach

#### Using Inflect.com (for DM campaigns)

**Pros:** Fast scraping of Instagram audiences, cheap ($99/mo)
**Cons:** Lower email quality, Instagram focus, some invalid addresses

**Process:**
1. Go to Inflect.com → Tools → Instagram Scraper
2. Enter Instagram profile or hashtag to scrape
3. Set filters (location, engagement, follower count)
4. Scrape followers/commenters
5. Download CSV with Instagram handles + any available email
6. Append emails if needed (see enrichment step below)
7. Validate in NeverBounce before sending

### Step 4: Append Emails (If Missing Email Addresses)

**Situation:** You have names + company, but no email

**Process using Hunter.io:**
1. Go to Hunter.io
2. Bulk email finder (pay per lookup)
3. Upload CSV with: First name, Last name, Company domain
4. Run lookup (e.g., 1000 lookups @ ~$0.20 each)
5. Download results with emails

**Alternative using Apollo:**
1. Upload CSV with: First name, Last name, Company
2. Apollo auto-matches to their database
3. Appends email if found

### Step 5: List Cleaning & Validation

**Critical:** Never send to unvalidated list (will damage deliverability)

#### Using NeverBounce

**When to use:** After sourcing, before campaign launch

**Process:**
1. Go to NeverBounce.com → Start → New job
2. Upload CSV of emails (or paste emails)
3. Select email validation level:
   - "Standard" = fast, good for most campaigns
   - "Plus" = slower, more accurate
4. Run job (takes 1-4 hours depending on size)
5. Download results: Shows each email as VALID, INVALID, or RISKY
6. Action:
   - VALID: Keep these emails
   - INVALID: Remove before sending
   - RISKY: Remove or segment separately (lower volume)

**Expected results:**
- Fresh list from Apollo/Seamless: 15-25% invalid/risky
- UpLead list: 5-15% invalid/risky
- Old list (>30 days): 30-50% invalid/risky
- Scraped list: 40-60% invalid/risky

**Pro tip:** Only send initial email to VALID emails. Use RISKY for follow-up sequence only after confirmed engagement.

#### Using ZeroBounce (Alternative)

**When to use:** If you need abuse detection + bounce checking

**Process:**
1. Go to ZeroBounce.net → Sign up
2. Upload list → Select scoring level
3. Run validation
4. Download results: Shows valid, invalid, catch-all, abuse
5. Action: Remove abuse + invalid, validate/tag risky

## Building List in Apollo.io (Step-by-Step Example)

**Goal:** Build list of 500 marketing directors at SaaS companies, US-based, 50-500 employees

**Step 1: Apollo Filters**
1. Title = "Marketing Director" (or similar)
2. Company size = 50-500 employees
3. Location = United States
4. Industry = Software/SaaS (optional but helps)

**Step 2: Review Results**
- Apollo shows: "3,247 results"
- Review first 50 to verify quality (are they actual marketing directors?)
- If good, proceed. If bad, adjust filters.

**Step 3: Create List**
1. Name list: "SaaS Marketing Directors - US - 50-500 employees"
2. Add to list (Apollo lets you select all or add individually)
3. Export CSV (includes: First name, Last name, Email, Title, Company, Phone)

**Step 4: Validate in NeverBounce**
1. Upload CSV to NeverBounce
2. Run validation (standard mode)
3. Download results
4. Filter for "VALID" only
5. Expected: ~2,500 valid emails from original 3,247

**Step 5: Segment**
1. Take top 100-200 valid emails
2. Send initial batch with A/B testing
3. Let first batch finish follow-up sequence (10 days)
4. Send next 100-200 batch
5. Scale gradually: 100 → 200 → 500 → 1000

## List Segmentation Best Practices

### Why Segment?
- Deliverability: Different segments need different volumes
- Messaging: Copy tailored to each segment converts better
- Testing: Segment-specific A/B testing is more accurate

### Recommended Segments

**Segment 1: By Role (Most Important)**
- CMO
- VP Marketing
- Marketing Manager
- Growth Manager
- Demand Gen Manager

**Action per role:** Write role-specific first line and case study
- CMOs care about: Revenue impact, board-level metrics, strategic outcomes
- Managers care about: Tactical execution, quick wins, team efficiency

**Segment 2: By Company Size**
- 10-50 employees (startups)
- 50-500 employees (growth stage)
- 500-5000 employees (mid-market)
- 5000+ employees (enterprise)

**Action per segment:** Price, use case, implementation timeline will differ

**Segment 3: By Engagement Level**
- Cold (never interacted)
- Warm (opened previous email from you)
- Hot (replied to previous email)

**Action per segment:**
- Cold: Longer sequence, lower frequency, different copy
- Warm: Can increase frequency, shorter sequence
- Hot: Move to phone/meeting immediately, different script

**Segment 4: By Source**
- Apollo list
- LinkedIn Research
- UpLead list
- Existing customer referrals

**Action per segment:** Each source has different average quality and response rate; track separately

**Segment 5: By Geographic/Business Signals**
- Recently funded (use Crunchbase, news)
- Recently hired (use LinkedIn hiring indicator)
- New job change (use LinkedIn alerts)

**Action per segment:** Time-sensitive relevance; prioritize these

### Segmentation Process

```
Raw list of 5000 prospects
├─ Segment by role (CMO, VP, Manager)
├─ Within each role: Segment by company size
├─ Within each role + size: Segment by engagement level
├─ Track open/reply rate per segment
├─ Identify best-converting segment
└─ Double down on best segment; improve others
```

## List Maintenance & Hygiene

### Ongoing List Health

**Weekly:**
- [ ] Track bounce rate (should stay <8%)
- [ ] Track unsubscribe rate (should stay <0.5%)
- [ ] Monitor spam complaint rate (should stay <0.1%)

**Monthly:**
- [ ] Remove engaged (100% open, 50%+ reply) from follow-up sequence
- [ ] Remove non-responsive after 4 follow-ups (sunset policy)
- [ ] Segment new list sources separately
- [ ] Review best-converting segments; prioritize similar profiles

**Quarterly:**
- [ ] Clean entire list in NeverBounce again
- [ ] Remove spam trap addresses (ZeroBounce abuse flag)
- [ ] Remove duplicate emails
- [ ] Merge duplicate contacts (same person from different sources)

### When to Retire a List

- Bounce rate consistently >15%
- Open rate declining >30% over 2 weeks
- Unsubscribe rate >1%
- Compliance issues (hit TCPA limits, spam complaints)
- Age >90 days with no fresh segments

**Action:** Pause sending, clean aggressively, or start new list

## Build vs. Buy Decision Matrix

### When to BUILD your own list (Manual)

**Best for:**
- Ultra-specific niche (fewer than 500 total prospects)
- Relationship-based selling (need detailed research per prospect)
- Premium positioning (fewer high-value targets)
- Long sales cycle B2B (need customization)

**Process:**
1. LinkedIn Sales Navigator (buy accounts for team)
2. Manual search + profile review (1-2 per day per SDR)
3. Notes/research in spreadsheet or CRM
4. LinkedIn InMail or personalized email (highly customized)

**Cost:** Team time (slow but high quality)
**Timeline:** Slower build but higher conversion rates

### When to BUY a list (Apollo/UpLead)

**Best for:**
- Mid-size campaigns (1000-5000 prospects)
- Fast scaling needed (need volume quickly)
- Clear ICP filters (can define in tool filters)
- Cost-sensitive (cheaper per prospect than manual)

**Process:**
1. Define filters in Apollo/UpLead
2. Export list
3. Validate in NeverBounce
4. Send campaign

**Cost:** $99-300/month for tools + validation
**Timeline:** Fast but lower personalization

### Hybrid Approach (RECOMMENDED)

- **30% warm/researched** (manual or LinkedIn, highest conversion)
- **70% bulk** (Apollo/UpLead, volume to hit KPIs)

**Benefit:** Balance speed with quality

## Troubleshooting Common List Issues

### Problem: 20% Bounce Rate (Very High)

**Likely causes:**
- Using old list (>60 days old)
- Sourced from low-quality provider
- Emails were scraped/not verified
- Wrong email format (e.g., "firstname.lastname@company.com" when actual is "firstname@company.com")

**Solution:**
1. Check email format consistency
2. Run through NeverBounce (expect to lose 40-50%)
3. Or: Get fresh list from UpLead/Apollo
4. Going forward: Always validate with NeverBounce

### Problem: 30% Open Rate (Very Low)

**Likely causes:**
- Deliverability issue (emails in spam)
- Subject line weak
- List fatigue (same list used multiple times)
- ICP mismatch (wrong role/industry)

**Solution:**
1. Check deliverability (test with new email accounts)
2. If deliverability good: A/B test subject lines
3. If list fatigue: Get fresh list
4. If ICP mismatch: Re-segment or get different list

### Problem: List Too Expensive

**Options:**
1. UpLead too expensive? Try Apollo/Seamless ($99/mo unlimited)
2. Need smaller list? Do manual LinkedIn research (30-50 people)
3. Use free tools: LinkedIn search + Hunter.io lookup ($0.20 per email)

## Complete List Building Checklist

Before launching campaign:
- [ ] ICP defined (company size, role, industry, geography)
- [ ] List source chosen (Apollo, UpLead, LinkedIn, or manual)
- [ ] List sourced (100-500 initial batch)
- [ ] Emails validated in NeverBounce (removed invalid)
- [ ] List segmented by role/company size
- [ ] Domain authenticated (SPF, DKIM, DMARC)
- [ ] Domain warmed (14 days low volume if new domain)
- [ ] Initial email ready (personalized first line)
- [ ] Follow-up sequence mapped (day 3, 5, 8, 12)
- [ ] CRM set up (track opens, replies, meetings)
- [ ] KPIs defined (open rate target, reply target, booking target)

Once all checked: Ready to send.
