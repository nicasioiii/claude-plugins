- **Facebook native:** Works directly in Messenger

- **Tagging system:** Built for the 8-12 tag methodology
- **Filters:** Powerful filters to prioritize inbox

## THE 8-12 TAG SYSTEM (CORE PRIORITIZATION)

**Philosophy:** Use ONLY conversation progress tags, not source tags. Simplicity beats features.

### Core Tags (Required)

| Tag | Purpose | Action When Filtered |
|-----|---------|---------------------|
| **OPEN_ASCENT** | Initial outreach sent, waiting for response | Wait 24-48 hours before follow-up |
| **COLD** | Early stage, just connected, building rapport | Continue Phase 1 (rapport building) |
| **WARM** | Engaged, showing interest, pain discovered | Move to Phase 3 (invitation) if Level 2+ pain found |
| **HOT** | High intent, ready for call, moving to booking | Send calendar link immediately |
| **CALL_PROPOSED** | Offer made, waiting for their decision | Follow up daily until they decide |
| **CALL_BOOKED** | Call is scheduled on calendar | Send pre-call materials, reminder 24hrs before |
| **CALL_ATTENDED** | Call completed, waiting for next steps | Follow up based on call outcome |
| **QUALIFIED_LEAD** | Meets your niche/budget requirements | Priority for follow-up and outreach |
| **NOT_NOW** | Interested but timing wrong | Set reminder to follow up in 30-60 days |
| **NOT_FIT** | Doesn't match your ideal customer profile | Archive or keep for future reference |

### Optional Tags (Add if Needed)

- **WARM_AUDIENCE** – Content follower, organic connection (vs cold outreach)
- **CLOSED_DEAL** – Became a paying client
- **IN_TRIAL** – Client in trial/test phase
- **OBJECTION_[TYPE]** – Tag specific objections for tracking patterns

**Golden Rule:** Max 12 tags. Any more = complexity kills prioritization.

## DAILY CRM WORKFLOW (Step-by-Step)

### Morning Startup (Before Outreach Begins)

```
1. Open GroupTrack in Facebook Messenger
   Link: https://www.facebook.com/messages/t

2. Review Calendar
   - Check today's booked calls
   - Prepare any call materials
   - Note: Update CALL_BOOKED tags if any are missing

3. Check CRM: "Unread" Messages Only Filter
   - Are there any urgent messages overnight?
   - Respond to critical messages ASAP
```

### Core Daily Workflow (5-6 Hours)

#### STEP 1: Inbox Management (20 minutes)
```
Filter: "Unread messages only"

Action:
- Respond to ALL unread messages you haven't replied to
- Rule: If message is read, it must be responded to or tagged appropriately
- Don't let unread messages pile up

Time: 20 minutes max
Expected: 10-30 messages typically
```

#### STEP 2: Warm Follow-Ups (1.5-2 hours)

**2A: Past Calendar Events (30 minutes)**
```
Filter: Tag "CALL_ATTENDED" (calls from this week/month)

Action for each contact:
- Did they say they'd get back to you?
- Did they ask any questions?
- Send follow-up based on call outcome

Script Example:
"Hey! Just following up on our call yesterday. How are you feeling about it?
Do you have any other questions about what we discussed?"

After response:
- Move to CALL_PROPOSED (if you made an offer) or WARM (if still in discussion)
```

**2B: Calls Proposed Yesterday (20 minutes)**
```
Filter: Tag "CALL_PROPOSED"

Action for each contact:
- Did they book the call?
- Did they respond to the calendar link?

Script Example:
"Hey, did you have any trouble finding a time that works for you?"

After response:
- If booked: Move to CALL_BOOKED, set reminder
- If declined: Move to NOT_NOW, schedule follow-up in 7-30 days
- If silent: Stay CALL_PROPOSED, will follow up tomorrow
```

**2C: New Facebook Group Members (15 minutes)**
```
Action:
- Add 3-5 new group members as friends (if they're in group request queue)
- Then message them warm (they're more likely to respond)

Script:
"Hey! Thanks for joining [GROUP NAME]! Would you be interested in the free course
I put together for members? [YES/NO]"

Tag all new messages: OPEN_ASCENT
```

**2D: Story Engagement Responses (15 minutes)**
```
Filter: Messages from story poll voters or story responders

Action:
- Respond to each person who engaged with your story
- Ask follow-up question to move conversation forward

Script Example:
"Thanks for voting! Yeah, [respond to their choice] ...
So what's your take on that?"

Tag as: COLD or WARM depending on engagement quality
```

**2E: Action Post Commenters (20 minutes)**
```
Filter: Messages or comments from people who requested your resource

Action:
- Send the resource they asked for
- Follow up with a question about it

Script:
"Awesome, here you go: [LINK]

What caught your attention most about it?"

Tag as: OPEN_ASCENT
```

**2F: Recent Post Engagers (15 minutes)**
```
Action:
- Message people who liked or commented on your most recent post
- Focus on story engagers first (higher intent)

Script:
"Hey! Thanks for the engagement on that post. Curious what resonated with you most?"

Tag as: COLD
```

#### STEP 3: Cold Outreach (30 min-1 hour, if space available)

**Only do if you've completed all warm follow-ups above**

```
Filter: Create filter for "NOT tagged" or "Not contacted in 60+ days"

Sources to message:
- Old group members (no recent contact)
- Older friends (extended time since last message)
- Content engagers (past likers)
- Story viewers (previous story watchers)

Daily quota: 20-30 new cold opens
(These are longer conversations, so fewer volume than warm)

Tag all new cold outreach: OPEN_ASCENT
```

#### STEP 4: Follow-Up Sequences (2-3 hours)

**4A: 24-Hour Follow-Up (Scroll Back 24 Hours)**
```
Filter: All messages from past 24 hours

Action:
- Message everyone who didn't move forward
- Don't repeat yourself; add NEW value or question

Rule: 1 follow-up = 1 new question or value add, NOT a reminder

Priority Order:
1. HOT leads first
2. WARM leads second
3. COLD leads third

Expected response: 40-60% of warm leads respond to follow-up
```

**4B: Hot Tag Follow-Ups (20 minutes)**
```
Filter: Tag "HOT" in GroupTrack

Action:
- These are your highest priority people
- Check each one: what do they need NOW?

Script Example:
"Hey, checking in. Do you have any questions about what we discussed?
Ready to book that call?"

Urgency level: Respond ASAP, same day
```

**4C: 3-Day Scroll Back Follow-Ups (30 minutes)**
```
Action:
- Scroll back 3 days in your message history
- Find people you didn't hear back from
- Send one strategic follow-up

Script Example:
"Hey! Just checking in. Still interested in exploring this?
Let me know if timing isn't right and we can touch base later."

Expected response: 20-30% of cold leads
```

**4D: 7-Day Scroll Back (20 minutes, 2x per week)**
```
Action:
- Scroll back 7 days
- People who've gone silent
- One more attempt at reengagement

Script Example:
"Hey, haven't heard from you in a bit! How have you been?
Still interested in that quick call? [LINK]"

Expected response: 10-20%
```

**4E: 30-Day Scroll Back (30 minutes, 1x per week)**
```
Action:
- Scroll back 30 days
- People from a month ago who never moved forward
- Fresh angle or permission to move on

Script Example:
"Hey [NAME]! Long time—how have you been? I was just thinking about your goal of [THEIR GOAL].
Have you been able to make progress, or is it still on the back burner?
If you want to explore some options, I'm still here. [LINK]"

Expected response: 5-10%
```

**4F: 60-Day Follow-Ups (1x per week, if capacity)**
```
Action:
- People from 2 months ago
- Quick check-in
- Often these re-engage when circumstances change

Script:
"Hey! Haven't talked in a couple months. What's new?
Still interested in [TOPIC]?"
```

### End of Day Checklist (30 minutes)

```
☐ Clear inbox to zero (all messages responded to or tagged)
   - No unread messages should remain
   - Every message should have a tag

☐ Update CRM tags based on conversation progress
   - New conversations get OPEN_ASCENT or COLD
   - Responses move to WARM
   - Pain-discovered conversations move to HOT

☐ Fill KPI tracker
   - Number of new conversations started: _____ (target 20-50)
   - Number of follow-ups sent: _____ (target 100+)
   - Number of calls booked: _____ (target 3)
   - List of calls booked (names, times)

☐ Review: Did we hit daily goals?
   - 20-50 conversations? ✓ / ✗
   - 100+ follow-ups? ✓ / ✗
   - 3 calls booked? ✓ / ✗

☐ Send end-of-day report (if you have a team)
   - Total conversations: X
   - Hot leads created: Y
   - Calls booked: Z
```

## WEEKLY CRM MAINTENANCE (1-2 hours/week)

### Monday: Setup Week
```
☐ Review past week's performance
   - Which openers worked best?
   - Which follow-up times were most effective?
   - Any patterns in what converts to calls?

☐ Tag cleanup
   - Move people from WARM → HOT if pain discovered
   - Move people from HOT → NOT_NOW if they went silent
   - Archive NOT_FIT people or keep for future reference
```

### Wednesday: Mid-Week Review
```
☐ Check all CALL_PROPOSED contacts
   - Have they booked? If not, send reactivation follow-up
   - Any patterns in objections?

☐ Review HOT list
   - Send value to anyone waiting on your next message
   - Any calls ready to be scheduled?
```

### Friday: End-of-Week Cleanup
```
☐ Scroll back 30 days
   - Update tags on anyone who went silent
   - Move old OPEN_ASCENT to COLD if no response

☐ Prepare for next week
   - What content will you post Monday-Friday?
   - Any special outreach angles for next week?

☐ Analytics review
   - Total conversations this week: _____
   - Total follow-ups this week: _____
   - Total calls booked this week: _____
   - Conversion rate: _____ % (calls booked / conversations)
   - Target: 15-20%
```

## GROUPTRACK SETUP INSTRUCTIONS

### Installation & Initial Setup

```
Step 1: Go to GroupTrack website
https://www.grouptrackcrm.com?fpr=jason38

Step 2: Sign up / Log in
- Create account or log in
- Choose Basic plan ($59/month) or higher

Step 3: Install Chrome extension

Step 4: Set up your Facebook account
- Connect to your Facebook Business account

- Verify it's working in Messenger

Step 5: Create your tags
- In GroupTrack dashboard: Settings > Tags
- Create the 8-12 tags (see tag list above)
- Assign colors:
  - RED = HOT / CALL_PROPOSED
  - YELLOW = WARM
  - BLUE = COLD
  - GREEN = QUALIFIED_LEAD / CLOSED_DEAL
  - GRAY = NOT_FIT / NOT_NOW

Step 6: Test with one conversation
- Send yourself a test message
- Tag it in GroupTrack
- Verify the interface works
```

### Essential Features to Use

```
✓ MESSAGE TAGGING
  - Right-click on any message in Messenger
  - Select "Add Tag" in GroupTrack
  - Tag all conversations

✓ FILTERS (Most Powerful Feature)
  - Create custom filters for daily workflow
  - Examples:
    - "Unread messages only"
    - "Tag = HOT"
    - "Tag = CALL_PROPOSED"
    - "Last activity > 24 hours"
    - "Multiple tags"

✓ ADVANCED FILTERS (Premium, Optional)
  - Sort by oldest activity
  - Show unread messages only
  - Show read but not responded
  - Show messages with specific tags
  - (These are GOLD for prioritization)

✗ MESSAGE TEMPLATES (Optional)
  - Available in GroupTrack
  - BUT: Manual copy/paste performs better
  - Prospects can tell when using CRM templates
  - Better to have templates in Google Doc and copy/paste manually
```

```
Platform Link:
Use: https://www.facebook.com/messages/t
NOT: Facebook Messenger app
(Website version is more stable)

Notification Settings:
```

## COMMON GROUPTRACK MISTAKES TO AVOID

```
❌ Creating 20+ tags
- Too many tags = decision paralysis
- Stick to 8-12 max

❌ Using CRM automations for all messages
- Prospects detect automation
- Manual messages perform 2-3x better
- Only automate action post comments (acceptable)

❌ Not filtering at all
- Leaving CRM as plain inbox
- GroupTrack's power is in filters
- Use filters to prioritize every single day

❌ Tagging inconsistently
- One week WARM means one thing, next week it's different
- Define tag definitions clearly
- Review tags weekly for consistency

❌ Not clearing unread messages
- "Unread" should always be processed daily
- If read, must be responded to or archived
- Don't let unread messages pile up

❌ Mixing lead source tags with progress tags
- Wrong: Tag by "Instagram_Lead" vs "Facebook_Lead"
- Right: Tag by "COLD" vs "WARM" vs "HOT"
- Source doesn't matter; progress is what matters

❌ Ignoring the HOT tag
- If someone is HOT, they're ready for a call
- Don't leave HOT people hanging
- Respond to HOT same day
```

## SCALING THE CRM (When You Hire a VA)

### What a VA Can Do

```
✓ Inbox management (respond to unread)
✓ Tag management (organize conversations)
✓ Follow-up sequences (pre-written, if supervised)
✓ Calendar scheduling (book calls, send links)
✓ Lead scraping (Inflect.com, manual list building)
✓ Prep work (research, material prep for calls)

✗ Do NOT automate all messaging
✗ Do NOT let VA completely take over conversations
✗ Do NOT remove you from relationship-building

Best practice: VA handles 50% of follow-ups and inbox,
you focus on 50% hot leads and phone conversations
```

### Training Your VA on CRM

```
1. Show them the tag system (8-12 tags)
2. Show them the daily workflow (inbox → warm → cold → follow-ups)
3. Show them what "ready for call" looks like (HOT tag criteria)
4. Have them shadow for 3 days
5. Let them take over 20% of daily work
6. Gradually increase to 50%
7. Always review their tagging decisions

Key: VA is force multiplier, not replacement.
Your personal touch drives conversions.
```
