# Write Headlines

This command generates a collection of high-converting headline variations, each using different proven formulas and scored for clarity, benefit communication, and curiosity.

## What You'll Get

- 10+ headline options (10-15 typically)
- Each scored on 3 dimensions: Clarity (1-10), Benefit (1-10), Curiosity (1-10)
- Mix of formula types: benefit-driven, curiosity gaps, number-based, question-based, pattern interrupt
- Ranked by overall score
- Usage guidance for each type

## Questions I'll Ask

1. **What product or offer are we writing for?**
   - Name and one-sentence description

2. **What's the main benefit?**
   - What does the customer get? What problem is solved?

3. **Who is the audience?**
   - Job title, pain point, aspirational outcome

4. **What type of page is this for?**
   - Ad, landing page, email subject line, sales page hero, blog post

## My Process

I'll use the **headline-hook-engine** skill to generate variations across multiple proven formulas:

- Direct benefit statements
- Curiosity gaps (incomplete info that demands answer)
- Number-based headlines
- Question-based hooks
- Problem/agitation headlines
- Pattern interrupt (unexpected angle)
- Fear/risk reversal
- Specificity + promise

## Output Format

Each headline will be displayed as:

```
[Rank]. [Formula Type] — [Headline]
   Clarity: 8/10 | Benefit: 9/10 | Curiosity: 7/10 | Overall: 8.0/10
   Why it works: [Brief explanation of effectiveness]
```

Final section includes guidance on A/B testing and context for deployment.
