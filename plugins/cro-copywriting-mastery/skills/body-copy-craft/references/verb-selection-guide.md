# Verb Selection Guide

## THE 3-STEP PROCESS

Every verb in your copy should pass through this filter:

### Step 1: What IMAGE am I trying to convey?
Choose from: Speed, Volume, Ease, Passivity, Aggression, Precision, Weight, Lightness, Stealth, Force, Grace, Chaos, Order

### Step 2: What FEELING do I want to trigger?
Choose from: Fear, Excitement, Relief, Confidence, Urgency, Curiosity, Comfort, Disgust, Awe, Anger, Hope, Pride

### Step 3: Find a verb that conveys BOTH simultaneously.

The verb must paint a picture AND trigger an emotion in the same word. If it only does one, keep searching.

---

## DEAD VERBS — ELIMINATE THESE

These verbs are vague, fuzzy, and static. They convey no image and trigger no emotion:

| Dead Verb | Problem |
|-----------|---------|
| Have | No action, no image |
| Do | Completely generic |
| Get | The weakest verb in the English language |
| Show | Vague, passive feeling |
| Use | No sensory information |
| Make | Generic creation |
| Put | No image of how |
| Go | No speed, no direction quality |
| Take | No force, no method |
| Give | No manner of giving |
| Is/Was/Are | State of being, zero motion |
| Seem | Uncertainty without purpose |
| Feel | Telling, not showing |

## REPLACEMENT BANK BY IMAGE CATEGORY

### Speed Verbs (Fast → Slow)
| Fast | Medium | Slow |
|------|--------|------|
| Sprint, bolt, rocket | Walk, pace, cruise | Crawl, creep, inch |
| Rush, dash, blast | Stroll, wander, drift | Plod, trudge, lumber |
| Surge, shoot, zip | Glide, flow, stream | Drag, slog, grind |
| Slam, strike, whip | Move, travel, shift | Linger, hover, loiter |

**Example transformations:**
- "He went to the meeting" → "He bolted to the meeting" (urgency) / "He trudged to the meeting" (reluctance)
- "Sales grew" → "Sales rocketed" (excitement) / "Sales crawled" (frustration)

### Volume Verbs (Large → Small)
| Large Volume | Medium | Small Volume |
|-------------|--------|-------------|
| Flood, drown, overflow | Fill, supply, deliver | Trickle, drip, seep |
| Swamp, overwhelm, deluge | Stream, flow, pour | Leak, ooze, bleed |
| Explode, erupt, cascade | Spread, expand, grow | Whisper, hint, nudge |
| Avalanche, swarm, surge | Gather, collect, build | Sprinkle, scatter, dust |

**Example transformations:**
- "Get more leads" → "Flood your business with leads" (volume + excitement)
- "Reduce costs" → "Slash costs to the bone" (force + relief)

### Force Verbs (Strong → Gentle)
| Strong Force | Medium | Gentle |
|-------------|--------|--------|
| Crush, smash, shatter | Push, press, drive | Touch, caress, brush |
| Pummel, demolish, obliterate | Strike, hit, knock | Tap, nudge, graze |
| Slam, blast, detonate | Pull, yank, tug | Lift, ease, glide |
| Ram, bulldoze, pulverize | Grip, hold, clench | Cradle, cushion, nestle |

**Example transformations:**
- "Beat the competition" → "Crush the competition" (dominance)
- "Touch their emotions" → "Grip their emotions" (intensity)

### Stealth/Secrecy Verbs
| Verb | Connotation |
|------|-------------|
| Lurk | Hidden with menacing intent |
| Sneak | Moving without detection |
| Slip | Smooth, unnoticed movement |
| Creep | Slow, deliberate stealth |
| Smuggle | Moving something forbidden |
| Conceal | Active hiding |
| Whisper | Quiet, intimate sharing |
| Uncover | Revealing what was hidden |
| Expose | Forceful revealing |
| Unmask | Dramatic revealing of truth |

**Example:** "He was behind the counter" → "He lurked behind the counter" (fear/hiding)

### Growth/Transformation Verbs
| Verb | Connotation |
|------|-------------|
| Bloom | Natural, beautiful growth |
| Explode | Sudden massive growth |
| Evolve | Gradual, intelligent growth |
| Multiply | Compounding growth |
| Ignite | Starting with energy |
| Skyrocket | Dramatic upward trajectory |
| Catapult | Sudden launch to new level |
| Transform | Complete fundamental change |
| Metamorphose | Deep structural change |
| Flourish | Healthy, abundant growth |

### Destruction/Pain Verbs
| Verb | Connotation |
|------|-------------|
| Drain | Slow, exhausting loss |
| Bleed | Painful, visible loss |
| Erode | Gradual wearing away |
| Devour | Consuming completely |
| Strangle | Suffocating, choking |
| Cripple | Disabling capability |
| Sabotage | Deliberate undermining |
| Corrode | Slow chemical damage |
| Hemorrhage | Rapid, dangerous loss |
| Suffocate | Being smothered |

**Example:** "Losing money" → "Hemorrhaging cash" (urgency + danger)

## SENSORY VERB CATEGORIES

### Sight Verbs
Gleam, glitter, shimmer, flash, glow, sparkle, shine, beam, radiate, illuminate, reveal, expose, spotlight, showcase, display, peek, glimpse, gaze, stare, scan

### Sound Verbs
Whisper, roar, thunder, hum, buzz, crackle, pop, sizzle, ring, echo, boom, crash, murmur, chatter, scream, shriek, rumble, purr, hiss, clang

### Touch Verbs
Caress, grip, clutch, stroke, scratch, poke, squeeze, press, crush, embrace, tickle, pinch, slap, rub, scrape, jab, prod, knead, clench, smooth

### Taste/Smell Verbs
Savor, devour, gulp, sip, nibble, feast, gorge, swallow, inhale, breathe in, sniff, reek, stink, perfume, fragrance, waft, infuse, saturate, permeate

### Motion Verbs
Surge, pulse, vibrate, oscillate, spiral, tumble, plunge, soar, dive, glide, bounce, ricochet, orbit, rotate, pivot, weave, zigzag, meander, swirl, cascade

## ADVERB + WEAK VERB → SINGLE STRONG VERB

This is the fastest way to improve any draft. Find every adverb + verb combination and replace with one powerful verb.

| Adverb + Weak Verb | Strong Verb Replacement |
|--------------------|----------------------|
| Slowly walk | Stroll, saunter, amble, trudge |
| Quickly run | Sprint, dash, bolt, race |
| Gently touch | Caress, brush, graze |
| Loudly yell | Scream, bellow, roar |
| Vigorously push | Shove, ram, thrust |
| Quietly say | Whisper, murmur, breathe |
| Angrily look | Glare, scowl, glower |
| Carefully examine | Scrutinize, inspect, probe |
| Happily accept | Embrace, welcome, celebrate |
| Sadly decline | Mourn, grieve, lament |
| Strongly believe | Know, trust, champion |
| Easily achieve | Breeze through, nail, ace |
| Barely survive | Scrape by, cling to life |
| Completely destroy | Annihilate, obliterate, erase |
| Slowly decrease | Dwindle, erode, wither |
| Rapidly increase | Surge, skyrocket, explode |

## PRESENT TENSE CONVERSION TABLE

Use present tense for immediacy and energy:

| Past Tense (Weak) | Present Tense (Strong) |
|-------------------|----------------------|
| Grew | Growing |
| Scaled | Scaling |
| Drove | Driving |
| Saved | Saving |
| Built | Building |
| Generated | Generating |
| Converted | Converting |
| Doubled | Doubling |
| Transformed | Transforming |
| Eliminated | Eliminating |

**Example:** "Our system grew revenue by 40%" → "Our system is growing revenue by 40% right now"

## THE VERB AUDIT PROCESS

When editing a draft:

1. **Highlight every verb** in the piece
2. **Circle dead verbs** (have, do, get, show, use, make, put, go, take, give, is, was, are)
3. **For each dead verb**, run the 3-step process: Image → Feeling → Replacement
4. **Check adverb+verb combos** and replace with single strong verbs
5. **Read aloud** — does each verb paint a picture in your mind? If not, replace it
6. **Final check:** Can you SEE, HEAR, or FEEL each verb? If it is purely abstract, make it sensory

**Target:** Replace at least 70% of dead verbs in any draft. The remaining 30% can stay if replacing would feel forced or unnatural.
