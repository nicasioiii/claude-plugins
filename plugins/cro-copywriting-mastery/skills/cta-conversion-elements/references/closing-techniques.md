# Closing Techniques

**Sources:** Adil Amarsi (Encyclopedia of Copy), Todd Brown (A-Z Workshop), Copy Hackers (10x Landing Pages), Arman Assadi (7-Figure Copywriting)

---

## The Three-Part Close (Arman Assadi)

Every sale requires a structured close. Even free ebooks need closing copy.

### Part 1: Re-Summarize What They Get

Use bullet points for scanners. This is NOT the time for new information -- it's the time to remind them of everything they're getting.

**Template:**
```
Here's everything you get when you [action] today:

[checkmark] [Main Product] -- [one-line benefit] (Value: $X)
[checkmark] [Bonus 1] -- [one-line benefit] (Value: $X)
[checkmark] [Bonus 2] -- [one-line benefit] (Value: $X)
[checkmark] [Bonus 3] -- [one-line benefit] (Value: $X)

Total Value: $[sum of all values]

Your Investment Today: Just $[actual price]
```

**Rules:**
- List every deliverable, even obvious ones (the longer the list, the higher the perceived value)
- Attach a dollar value to EVERY item (including bonuses)
- The total value should be 5-10x the actual price
- Use checkmarks or bullet points for visual scanning
- Bold the product names and benefits

### Part 2: Apply the Closing Formula

Choose one or combine:

**Elements-of-the-Offer Close (Copy Hackers):**
1. State exactly what they get
2. Establish the total value ($X,XXX)
3. Offer a conditional bonus ("When you order today, you also get...")
4. Reveal the price (after anchoring)
5. Trivialize the price ("That's less than $X per day")
6. State the guarantee
7. Reverse the risk ("If you don't see [result], you pay nothing")
8. Add scarcity ("Only available until [date/condition]")

**Consequences-of-Not-Acting Close (Copy Hackers):**
After the final CTA, add one section answering: "What is the consequence of NOT acting today?"

**Template:**
```
Still on the fence?

Consider what happens if you do nothing:

- [Consequence 1]: [Specific pain that continues]
- [Consequence 2]: [Specific opportunity that passes]
- [Consequence 3]: [Specific cost that accumulates]

Now imagine the alternative:

[Future-paced description of life with the product]

The only difference between these two futures?
The decision you make right now.
```

**The Takeaway Close (Todd Brown):**
Reverses the dynamic. The seller becomes the gatekeeper.

**Template:**
```
To be honest, [Product] isn't for everyone.

If you're looking for a magic bullet that requires zero effort,
this is not it.

If you're not willing to [minimum commitment],
you should probably pass.

But if you ARE the kind of [identity] who [positive trait],
and you're ready to [action]...

Then this was built for you.

[CTA Button]
```

### Part 3: First-Person CTA

The final element is always a button in first person:
- "Yes, I'm ready to [specific result]"
- "Give me instant access"
- "I want to start today"

## The Re-Close (Adil Amarsi)

A second closing section placed after the main close, for readers who scrolled past or needed more persuasion.

### When to Use
- Sales pages over 2,000 words
- Products priced above $97
- Audiences that are methodical researchers (they read everything before deciding)
- Pages with long FAQ sections (the FAQ can kill momentum; the re-close rebuilds it)

### Re-Close Template
```
[Restate the primary promise in different words]

[Strongest testimonial -- best result, most relatable person]

"[Testimonial quote with specific result]"
-- [Name, Title, specific detail]

Remember: [Guarantee statement restated]

If you don't [specific result] within [timeframe],
you don't pay. Simple as that.

[CTA Button: "Get [Product] Risk-Free"]

[Reassuring text: "Protected by our [X]-day guarantee"]
```

### Key Differences from the Main Close
- Shorter (1/3 the length of the main close)
- Uses different words for the same promise (don't copy-paste)
- Leads with the single strongest testimonial (not the full value stack)
- Restates the guarantee more concisely
- Feels like a friendly reminder, not a hard push

## P.S. Closing Formulas

The P.S. is one of the most-read parts of any sales letter or email. Use it strategically.

### P.S. Formula 1: Restate the Offer
```
P.S. When you grab [Product] today, you get:
- [Key item 1]
- [Key item 2]
- [Key item 3]
All protected by our [guarantee type] guarantee.
[Link: Claim your copy here]
```

### P.S. Formula 2: Add Urgency
```
P.S. This offer closes at midnight on [date].
After that, the price goes to $[higher price]
and [bonus] is no longer included.
[Link: Lock in today's price]
```

### P.S. Formula 3: Address the #1 Objection
```
P.S. Wondering if this will work for [their specific concern]?
Here's what [Name] said:
"[Testimonial addressing that exact concern]"
[Link: See for yourself -- risk free]
```

### P.S. Formula 4: Future Pace
```
P.S. 90 days from now, you'll either:
(a) Be in exactly the same spot you're in today, or
(b) Have [specific result] and be [vivid description of after state]

The only difference is the decision you make in the next 60 seconds.
[Link: Start your transformation]
```

### P.S. Formula 5: Scarcity + Social Proof
```
P.S. Only [number] spots remain this [month/quarter].
We've already welcomed [number] new members this week.
When they're gone, enrollment closes until [date].
[Link: Reserve your spot]
```

## Closing by Price Point

| Price Point | Close Strategy |
|------------|---------------|
| **Free (opt-in)** | Simple value statement + CTA. "Get your free [resource] now." No complex close needed. |
| **$1-$47** | Value stack + trivialize price + single CTA. Keep it short. Low price = low friction = short close. |
| **$47-$197** | Full three-part close. Value stack + guarantee + urgency. Consequences-of-not-acting section. |
| **$197-$997** | Extended close with re-close. Full value stack + multiple proof points + guarantee + urgency + P.S. |
| **$997-$2,997** | Full close + re-close + FAQ section between them. Takeaway positioning. Multiple P.S. formulas. |
| **$2,997+** | Application/qualification close. Takeaway heavy. Call/consultation CTA instead of direct purchase. |

## Close + Guarantee Integration

The guarantee is part of the close, not separate from it. Place it WITHIN the closing sequence.

### Guarantee Placement Options

**Option A: Before the CTA (risk reversal as final push)**
```
[Value stack]
[Price reveal]
[Guarantee: "Try it for 60 days. If you're not thrilled, get a full refund."]
[CTA button]
```

**Option B: After the CTA (reinforcement)**
```
[Value stack]
[Price reveal]
[CTA button]
[Guarantee: "Remember, this is completely risk-free. Your purchase is protected by our 60-day guarantee."]
```

**Option C: Both (belt and suspenders)**
```
[Value stack]
[Price reveal]
[Brief guarantee mention]
[CTA button]
[Detailed guarantee with process]
```

**Rule:** For products above $97, use Option C. The guarantee should appear at least twice in the close.

## Agreement Sequencing Before the Close

Drive mental "yes" responses before presenting the CTA. Each "yes" makes the next "yes" easier.

**Template:**
```
Do you want [desirable outcome 1]?
Do you want [desirable outcome 2]?
Are you ready to stop [pain point]?

Then [Product] is one of the smartest investments you can make.

Because [reason-why proof linking to the outcomes above].

[CTA Button]
```

**Rules:**
- Use 3-5 yes-questions (fewer feels rushed, more feels manipulative)
- Each question should reference a different benefit or pain point
- The final CTA should feel like the natural conclusion to all the "yeses"
- Questions should be rhetorical (the answer is obviously yes)

## Common Closing Mistakes

1. **Not asking for the sale.** Many pages present the offer but never actually close. Always include an explicit ask.

2. **Closing too early.** If the marketing argument isn't complete, the close feels premature. Make the full case first.

3. **Only closing once.** Sales pages need 3-5 CTA placements. A single close is not enough.

4. **No consequences of inaction.** If there's no cost to waiting, there's no reason to act now.

5. **Guarantee hidden or absent.** The guarantee is a closing tool. Feature it prominently.

6. **New information in the close.** The close re-summarizes and pushes for action. It does not introduce new features or benefits.

7. **Weak P.S.** The P.S. is prime real estate. Using it for throwaway copy wastes one of the highest-attention areas of the page.
