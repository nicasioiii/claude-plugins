# Value Stacking Worksheet

## The 5-10x Rule

The perceived value of everything the customer receives should be 5-10x what they pay. If the offer is $100, show $500-$1,000 in total value. This is not deception -- it is accurate representation of the genuine stand-alone value of each component.

## STEP 2: ASSIGN DOLLAR VALUES

For each component, determine a legitimate stand-alone price using one or more of these methods:

### Method A: Market Comparable
What would it cost to buy this component separately in the market?
- Search for stand-alone products/services that match each component
- Use the average market price as your value

### Method B: Time-Cost Equivalent
What would it cost in time to create this yourself?
- Estimate hours to produce x hourly rate of someone qualified
- Example: 20 hours of content at $100/hr consultant rate = $2,000 value

### Method C: Outcome Value
What is the value of the result this component produces?
- If one template generates $500 in revenue, the template library is worth at least $500
- If the system saves 10 hours/week at $50/hr, annual value = $26,000

### Method D: Previous Pricing
Have you sold this component separately before?
- Use the actual price you charged (or currently charge) for it
- This is the most credible method

## STEP 3: DESIGN YOUR BONUSES

Add 2-4 bonuses that follow these rules:

### Bonus Selection Criteria

1. **Relevance:** Must directly relate to the core offer (irrelevant bonuses are psychologically worthless)
2. **Acceleration:** The best bonuses help them get results FASTER
3. **Complementary:** Fills a gap the core offer does not cover
4. **Low delivery cost:** Should not significantly increase your cost of goods

### Bonus Types That Work

| Bonus Type | Example | Why It Works |
|-----------|---------|-------------|
| Quick-start guide | "The 15-Minute Fast Track Guide" | Removes implementation friction |
| Templates/Swipe files | "47 Copy-and-Paste Email Templates" | DFY component; saves time |
| Checklist | "The Launch Checklist" | Creates confidence; nothing missed |
| Access to expert | "30-Min Strategy Call with [Name]" | High perceived value; personal touch |
| Community/group | "Private Mastermind Group Access" | Ongoing value; social proof |
| Tool/calculator | "ROI Calculator Spreadsheet" | Practical, immediate utility |
| Case study/example | "5 Detailed Case Studies with Breakdowns" | Proof + education combined |
| Additional training | "Bonus Masterclass: Advanced [Topic]" | Extends the core offer |

### Bonus Anti-Patterns

- Irrelevant bonus ("Buy my fitness course and get a free eBook on gardening")
- Bonus that duplicates what is in the core offer
- Too many bonuses (4+ starts feeling gimmicky)
- Bonus with no clear value explanation
- Bonus that takes attention away from the core offer

## COMMON MISTAKES IN VALUE STACKING

1. **Inflating values beyond credibility.** If a PDF guide is worth $997, nobody believes you. Use realistic, defensible numbers.

2. **Not explaining WHY each component is valuable.** A name and a price are not enough. State the specific result each component delivers.

3. **Listing too many micro-items.** 3-8 items is the sweet spot. More than 10 and it starts feeling like padding.

4. **Forgetting to include the total.** The visual contrast between total value and actual price is the entire point.

5. **Making all items the same value.** Vary the values. One anchor item should be worth more than the actual price on its own.
