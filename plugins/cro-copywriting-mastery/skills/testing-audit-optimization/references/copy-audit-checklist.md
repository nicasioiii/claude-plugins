# 20-Point Copy Audit Checklist

Synthesized from all 8 source courses. Score each element 0-5 (0 = missing, 1 = poor, 5 = excellent). Total possible: 100.

---

## FOUNDATION (Points 1-5)

### 1. One Reader (0-5)
**What to check:** Is the copy written for a single, specific reader segment?

| Score | Description |
|-------|-------------|
| 0 | No identifiable target reader. Copy speaks to "everyone." |
| 1 | Vague audience ("business owners," "marketers"). No specific segment. |
| 2 | Audience is defined but copy language is generic and interchangeable. |
| 3 | Clear target segment. Some language matches their vocabulary. |
| 4 | Strong target specificity. Language mirrors VOC data. Reader says "this is for me." |
| 5 | Ultra-specific. Copy reads like it was written for ONE person. Uses exact language from research. |

**Quick test:** Show the page to someone outside the target audience. If they feel it speaks to them, the targeting is too broad.

### 2. Awareness Match (0-5)
**What to check:** Does the page match the visitor's awareness stage?

| Score | Description |
|-------|-------------|
| 0 | No awareness consideration. Page is a feature dump. |
| 1 | Awareness mismatch (e.g., most-aware headline on a problem-aware audience). |
| 2 | Correct awareness stage but poor execution (right direction, wrong depth). |
| 3 | Good awareness match. Hook meets them where they are. |
| 4 | Strong match. Copy progresses from current stage to next stage naturally. |
| 5 | Perfect match. Every section calibrated to move from their current stage forward. |

**Quick test:** Identify the awareness stage of incoming traffic. Does the first sentence match where they are?

### 3. Message Match (0-5)
**What to check:** Does the page match the ad/email/link that brought them here?

| Score | Description |
|-------|-------------|
| 0 | No message matching at all. Page feels like a different conversation. |
| 1 | Same topic but different language, tone, or visual style. |
| 2 | Some matching words but overall tone or promise differs. |
| 3 | Good match. Same promise, similar language, consistent tone. |
| 4 | Strong match. Exact phrases from the ad appear on the page. Visual consistency. |
| 5 | Perfect match. Visitor feels no "jarring detour." Seamless continuation of the ad/email. |

**Quick test:** Place the ad and landing page side by side. Can you draw a straight line from ad promise to headline?

### 4. One Big Idea (0-5)
**What to check:** Is there a single, clear big idea? Or multiple competing ideas?

| Score | Description |
|-------|-------------|
| 0 | No clear idea. Page is a collection of disconnected claims. |
| 1 | Multiple ideas competing for attention. "Save time AND money AND reduce risk AND..." |
| 2 | Two ideas that could be reduced to one. Some dilution. |
| 3 | One primary idea with some secondary noise. |
| 4 | Clear single idea throughout. Minor distractions but core is strong. |
| 5 | One idea, hammered home from every angle. Every section supports the same thesis. |

**Quick test:** Ask "What is this page about?" If the answer takes more than one sentence, there are too many ideas.

### 5. One Offer (0-5)
**What to check:** Is there exactly one offer? No backup offers or competing CTAs?

| Score | Description |
|-------|-------------|
| 0 | Multiple offers competing. "Buy this OR sign up for that OR download this." |
| 1 | One primary offer with a distracting secondary offer (text link, popup). |
| 2 | One offer but presented with too many options (5+ pricing tiers, multiple bundles). |
| 3 | One clear offer with 2-3 clean options. |
| 4 | One offer, clearly presented, easy to act on. |
| 5 | One offer, one action, zero confusion. Every CTA drives the same action. |

---

## HOOK & ABOVE THE FOLD (Points 6-9)

### 6. Headline Clarity (0-5)
**What to check:** Can a stranger understand the value proposition in under 5 seconds?

| Score | Description |
|-------|-------------|
| 0 | No headline or headline is meaningless jargon. |
| 1 | Headline is clever but unclear. Reader doesn't know what the page offers. |
| 2 | Headline communicates the topic but not the value. |
| 3 | Clear headline. Value proposition is understandable. Could be sharper. |
| 4 | Strong headline. Clear, specific, benefit-driven. Passes the "blank is blank" kernel test. |
| 5 | Exceptional headline. Immediately clear, highly specific, emotionally compelling. |

### 7. 7-Second Test (0-5)
**What to check:** Does the page answer: What is it? Why should I care? What do I do next?

| Score | Description |
|-------|-------------|
| 0 | Fails all three questions. |
| 1 | Answers one of three questions. |
| 2 | Answers two of three questions. |
| 3 | Answers all three, but weakly or ambiguously. |
| 4 | Clearly answers all three. |
| 5 | Answers all three AND creates desire to continue reading. |

### 8. Unique Mechanism (0-5)
**What to check:** Is the mechanism named and positioned as different/superior?

| Score | Description |
|-------|-------------|
| 0 | No mechanism. Selling a commodity. |
| 1 | Generic approach described but not named or differentiated. |
| 2 | Named but not positioned as superior. Just a label on a commodity. |
| 3 | Named and somewhat differentiated. Reader sees it as different. |
| 4 | Named, differentiated, and supported with reason-why the mechanism is superior. |
| 5 | Named, differentiated, interrogated, and proven superior. Google Slap proof. |

### 9. Primary Promise (0-5)
**What to check:** Is there a big, bold, specific, measurable, envisionable promise?

| Score | Description |
|-------|-------------|
| 0 | No promise. Just features or vague claims. |
| 1 | Vague promise ("Get better results"). |
| 2 | Somewhat specific but not measurable ("Improve your marketing"). |
| 3 | Specific and measurable but not vivid ("Get 50% more leads"). |
| 4 | Specific, measurable, and envisionable ("Wake up to 12 new leads every morning"). |
| 5 | Big, bold, specific, measurable, AND backed by proof. Reader can see it in their mind's eye. |

---

## BODY COPY & ARGUMENT (Points 10-14)

### 10. Benefit Depth (0-5)
**What to check:** Are benefits dimensionalized across all three tiers?

| Score | Description |
|-------|-------------|
| 0 | Only features listed. No benefits at all. |
| 1 | Faux benefits ("Flush fat from your system"). One step removed from what they want. |
| 2 | Functional benefits only ("Saves time"). No dimensionalization. |
| 3 | Some dimensionalized benefits. Some remain surface-level. |
| 4 | Most benefits are dimensionalized. Emotional benefits present. |
| 5 | Full three-tier treatment: functional > dimensionalized > emotional. Reader FEELS the benefits. |

### 11. Proof Strategy (0-5)
**What to check:** Is every major claim supported with appropriate proof?

| Score | Description |
|-------|-------------|
| 0 | No proof anywhere. All claims unsupported. |
| 1 | Some generic testimonials dumped in one section. No strategic placement. |
| 2 | Proof exists but doesn't match claims. Right proof, wrong location. |
| 3 | Most major claims have some proof. Some gaps remain. |
| 4 | Strong proof strategy. Claim proof for mechanism, social proof for product. Matched testimonials. |
| 5 | Comprehensive. Power of Three on bold claims. Monster proof points. Every claim substantiated. |

### 12. Objection Handling (0-5)
**What to check:** Are the top 3-5 objections addressed before the close?

| Score | Description |
|-------|-------------|
| 0 | No objection handling. FAQ is absent or useless. |
| 1 | FAQ exists but doesn't address real objections. |
| 2 | 1-2 objections addressed, but major ones ignored. |
| 3 | Top 3 objections addressed somewhere on the page. |
| 4 | Top 5 objections addressed strategically at the right points. |
| 5 | All major objections handled preemptively throughout the copy. Reader never has to wonder. |

### 13. Story/Narrative (0-5)
**What to check:** Is there a compelling story element?

| Score | Description |
|-------|-------------|
| 0 | No story. Pure logical argument or feature list. |
| 1 | Brief "about us" section that's self-focused, not reader-focused. |
| 2 | Some narrative attempt but reader is not the hero. |
| 3 | Story present. Reader can see themselves in it. |
| 4 | Compelling narrative that positions reader as hero, product as guide. |
| 5 | Story-driven copy throughout. Emotional engagement from start to finish. |

### 14. Readability (0-5)
**What to check:** Is copy readable and scannable?

| Score | Description |
|-------|-------------|
| 0 | Wall of text. No formatting. Academic language. |
| 1 | Some paragraph breaks but sentences are too long and complex. |
| 2 | Readable but not scannable. No bold, bullets, or subheads. |
| 3 | Good readability. Some formatting. Flesch-Kincaid 8-10. |
| 4 | Strong readability. Short paragraphs, clear subheads, scannable. FK < 8. |
| 5 | Exceptional. Binary writing, strong verbs, sensory language, perfectly formatted for skimmers. |

## OFFER & CLOSE (Points 15-18)

### 15. Offer Presentation (0-5)
| Score | Description |
|-------|-------------|
| 0 | No clear offer section. Reader has to guess what they're buying. |
| 1 | Offer exists but is buried or confusing. |
| 2 | Offer is clear but no value stacking. Just price and product name. |
| 3 | Good offer presentation. Some value stacking. Deliverables listed. |
| 4 | Strong value stack. Everything listed with values. Clear and compelling. |
| 5 | Irresistible offer presentation. 5-10x value ratio. Crystal clear what they get and why it's a no-brainer. |

### 16. Price Justification (0-5)
| Score | Description |
|-------|-------------|
| 0 | Price stated cold with no context. Reader compares to $0. |
| 1 | Minor reframing but no real justification. |
| 2 | Some anchoring or comparison, but weak. |
| 3 | Decent price justification. Anchor + comparison + trivialize. |
| 4 | Strong justification. Multiple reframing techniques. Cost of inaction included. |
| 5 | Masterful. Reader feels the price is a steal. Investment framing with clear ROI. |

### 17. Guarantee (0-5)
| Score | Description |
|-------|-------------|
| 0 | No guarantee. |
| 1 | Guarantee mentioned in fine print but not featured. |
| 2 | Guarantee present but weak language or unclear terms. |
| 3 | Clear guarantee with decent positioning. |
| 4 | Strong guarantee. Prominent placement. Risk fully reversed. |
| 5 | Bold guarantee that builds confidence. Named guarantee. Dual structure (conditional + unconditional). |

### 18. CTA Strength (0-5)
| Score | Description |
|-------|-------------|
| 0 | No CTA or CTA says "Submit." |
| 1 | CTA exists but is generic ("Sign up"). No contrast, no reassurance. |
| 2 | Decent CTA copy. Some value. Not enough visual prominence. |
| 3 | Good CTA. Value-driven copy, decent contrast. |
| 4 | Strong CTA. First-person, value-driven, high contrast, reassuring text below. |
| 5 | Exceptional. Multiple placements, consistent action, perfect copy, reassurance, and contrast. |

---

## TECHNICAL & CONGRUENCY (Points 19-20)

### 19. Congruency (0-5)
| Score | Description |
|-------|-------------|
| 0 | Major contradictions between promise, proof, and guarantee. |
| 1 | Noticeable mismatches (e.g., "30-day results" with "60-day guarantee" unexplained). |
| 2 | Minor inconsistencies in tone or claims across sections. |
| 3 | Generally consistent. No major contradictions. |
| 4 | Strong consistency throughout. Ad-to-LP match is solid. |
| 5 | Perfect congruency. Every element supports every other element. Zero contradictions. |

### 20. Formatting & Friction (0-5)
| Score | Description |
|-------|-------------|
| 0 | Global nav on landing page. Slow load. Broken mobile. |
| 1 | Some friction elements. Coupon field visible. Too many links. |
| 2 | Decent but with unnecessary navigation or visual clutter. |
| 3 | Clean layout. Minimal friction. Mobile-functional. |
| 4 | Optimized. No global nav. Minimal form fields. Fast load. Good mobile. |
| 5 | Frictionless. Every element serves the conversion goal. Nothing extraneous. |

---

## Score Interpretation

| Total Score | Assessment | Action |
|------------|-----------|--------|
| 80-100 | Strong page | Test micro-optimizations (priority 5-7). Celebrate. |
| 60-79 | Good foundation | Fix the 3-5 weakest elements for significant lift. |
| 40-59 | Major gaps | Partial rewrite needed. Focus on priorities 1-3. |
| Below 40 | Fundamental issues | Start over. The page has strategic problems that tweaks cannot fix. |

---

## Priority Fix Guide

After scoring, identify the 3 lowest-scoring elements and fix them in this priority:

1. **Foundation issues (1-5)** -- Fix these FIRST. No amount of good writing fixes bad strategy.
2. **Hook issues (6-9)** -- Fix these SECOND. If people leave above the fold, nothing below matters.
3. **Argument issues (10-14)** -- Fix these THIRD. The body must convince.
4. **Close issues (15-18)** -- Fix these FOURTH. Good argument + weak close = missed conversions.
5. **Technical issues (19-20)** -- Fix these LAST. Important but less impactful than strategic/content fixes.
