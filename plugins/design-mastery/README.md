# Design Mastery Plugin

**Version:** 1.0.0
**Last Updated:** March 12, 2026

A comprehensive design knowledge system covering the complete visual design lifecycle—from discovery through implementation. Built from 20+ expert courses and design books.

---

## What This Plugin Covers

Design Mastery provides expert guidance across the entire design workflow:

- **Visual Design Fundamentals** — Typography, color systems, spacing, layout, visual hierarchy
- **UI Component Design** — Buttons, cards, forms, navigation, modals, and state management
- **Web & Responsive Design** — Website structures, layouts, responsive breakpoints, mobile-first strategy
- **SaaS & Dashboard Design** — Application interfaces, data visualization, dashboard patterns
- **Design Systems** — Figma workflows, component organization, design tokens, consistency
- **Design Critique** — Systematic review, identifying issues, improvement framework
- **Animation & Interactions** — Micro-interactions, scroll animations, motion timing
- **Platform Implementation** — Webflow building techniques, Shopify theme design
- **Logo & Branding** — Logo design process, brand identity systems, style guides

---

## The 11 Skills

### 1. **discovery-direction**
Project discovery, client needs assessment, scope definition, and design problem framing. Start here for new projects.

### 2. **visual-foundations**
Core design principles: typography, color theory, spacing systems, layout grids, and visual hierarchy. Foundation for all visual design.

### 3. **figma-workflows**
Figma file organization, design system setup, component creation, design tokens, and team collaboration practices.

### 4. **ui-components**
Design patterns for buttons, cards, forms, navigation, modals, and other reusable UI elements with variants and states.

### 5. **web-layout**
Website structure and responsive design: hero sections, feature layouts, footer patterns, and breakpoint strategies.

### 6. **dashboard-systems**
SaaS dashboard design: layouts, navigation patterns, data visualization, tables, metrics, and dense interface organization.

### 7. **web-design-audit**
Systematic design critique methodology, common mistakes checklist, and improvement framework with real examples.

### 8. **animation-interactions**
Micro-interactions, scroll animations, timing principles, motion design, and implementation across platforms.

### 9. **webflow-implementation**
Building in Webflow: layout techniques, responsive patterns, animations, CMS setup, and client handoff.

### 10. **shopify-design**
Shopify store design: theme customization, product displays, Liquid templating, and e-commerce patterns.

### 11. **logo-branding**
Logo design process, brand identity systems, style guide creation, and visual branding strategy.

## Slash Commands

Use these commands for quick access to the most common workflows:

### `/start-project`
**Launches:** discovery-direction
Launch a new design project with discovery questions, client questionnaire, and project complexity assessment.

### `/design-audit`
**Launches:** web-design-audit
Review and critique an existing design systematically with specific issues and recommended fixes.

### `/component-pattern`
**Launches:** ui-components
Show the best patterns for a specific component (button, card, form, navigation, modal, etc.) with variations.

### `/responsive-layout`
**Launches:** web-layout + webflow-implementation
Design a responsive website layout with desktop, tablet, and mobile breakpoints plus Webflow implementation guidance.

### `/create-logo`
**Launches:** logo-branding
Guide through the complete logo design process: research, sketching, vectorization, and brand system setup.

### `/dashboard-design`
**Launches:** dashboard-systems
Design a SaaS dashboard interface with structure, navigation, data display, and organization patterns.

### `/animation-help`
**Launches:** animation-interactions + webflow-implementation
Add animation and micro-interactions to design elements with timing, easing, and implementation details.

### `/design-system`
**Launches:** figma-workflows + visual-foundations
Set up a complete design system in Figma with file structure, components, tokens, and documentation.

## Quick Start

### For New Designers
1. Start with `/start-project` to understand the design process
2. Learn fundamentals via **visual-foundations** skill
3. Apply knowledge with `/component-pattern` and `/responsive-layout`

### For Design Reviews
1. Use `/design-audit` to systematically review a design
2. Jump to specific skills for fixes (e.g., **visual-foundations** for typography issues)

### For SaaS Projects
1. Use `/start-project` for discovery
2. Jump to `/dashboard-design` for app interface structure
3. Use `/animation-help` to enhance interactions

### For Logo Projects
1. Use `/create-logo` to follow the design process
2. Use `/design-system` to create brand style guides

### For Implementation
1. Use `/responsive-layout` or `/dashboard-design` to design
2. Use `/design-system` to set up Figma
3. Use `/animation-help` for motion design
4. Use `/webflow-implementation` or `/shopify-design` to build

## How It Works

Design Mastery is organized by **workflow stage**, not by tool. This means:

- **Skills are sequential**: Discovery → Design → Audit → Implement
- **Skills are modular**: Jump to any skill without prerequisites
- **Cross-references are smart**: Skills reference each other only when needed
- **Scope is clear**: In-scope (design), out-of-scope (copywriting, SEO)

Each skill includes:
- Core concepts and principles
- Quick reference tables and checklists
- Real-world examples
- Specific metrics and values
- Common mistakes to avoid
- Decision trees for ambiguous situations
- Cross-references to related skills
