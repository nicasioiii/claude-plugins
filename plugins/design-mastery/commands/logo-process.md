# /logo-process

You are guiding the user through the complete logo design process. Follow this workflow:

## Step 1: Research and Discovery

Ask the user:
1. **What is the brand?** (company name, industry, products/services)
2. **Who is the audience?** (demographics, values, aspirations)
3. **What personality should the logo convey?** (3-5 adjectives)
4. **What competitors exist?** (to differentiate from)
5. **Where will the logo appear?** (digital, print, signage, embroidery, favicon)
6. **Any constraints?** (must include specific icon, existing brand colors, text-only)

## Step 2: Explore Logo Types

Help the user understand options:

| Type | Description | Best For |
|------|-------------|---------|
| Wordmark | Full company name in custom type | Distinctive short names (Google, Coca-Cola) |
| Lettermark | Initials or monogram | Long company names (IBM, HBO) |
| Symbol/Icon | Abstract or pictorial mark | Established brands, app icons |
| Combination | Symbol + wordmark together | Versatile, works at any stage |
| Emblem | Text integrated into symbol | Heritage brands, badges |

Recommend a logo type based on the brand's stage, name length, and use cases.

## Step 3: Ideation and Sketching

Guide the sketching process:
1. Set a timer (15-30 minutes per round)
2. Sketch quickly and roughly -- quantity over quality at this stage
3. Explore 20-30 thumbnail concepts minimum
4. Focus on shapes and silhouettes, not details
5. Look for letterform manipulation opportunities (negative space, ligatures)
6. Evaluate sketches for: distinctiveness, simplicity, scalability, relevance

## Step 4: Vectorization

Move top 3-5 sketches to vector:
1. Set up artboard with grid (use golden ratio or geometric construction if appropriate)
2. Build with geometric shapes first, then refine curves
3. Ensure clean paths with minimal anchor points
4. Test at multiple sizes: favicon (16x16), social avatar (400x400), print (large)
5. Verify it works in single color (black on white, white on black)

## Step 5: Refinement

For each shortlisted concept:
- Optical corrections (adjust mathematically centered elements to look visually centered)
- Letter-spacing refinement for wordmarks
- Stroke consistency across all elements
- Test on colored backgrounds and photos
- Verify horizontal and stacked lockup variants work

## Step 6: Brand Identity Extension

Build the identity system around the chosen logo:
- Primary and secondary color palette
- Typography selection (heading and body fonts)
- Logo clear space rules (minimum padding around logo)
- Minimum size specifications
- Incorrect usage examples (stretch, recolor, rearrange)

## Step 7: Create Style Guide

Document the brand system:
- Logo versions (full color, reversed, monochrome, icon-only)
- Color specifications (hex, RGB, CMYK, Pantone)
- Typography specifications
- Usage rules and restrictions
- File format guide (SVG, PNG, EPS, PDF)

## Step 8: Client Presentation

Present 2-3 final concepts:
- Show logo in context (business card, website header, signage mockup)
- Explain the rationale behind each concept
- Present on neutral backgrounds first, then in mockups
- Walk through the brand story each concept tells

- If color palette needs formalizing → recommend `/color-palette`
- If web design follows → recommend `/homepage-structure`
