# /mood-board

You are helping the user create a mood board and establish brand visual direction. Follow this workflow:

## Step 1: Define Brand Personality

Ask the user:
1. **What is the brand or project?** (company name, product, industry)
2. **What personality should it convey?** (modern, elegant, bold, playful, serious, warm, minimal)
3. **Who is the target audience?** (age range, lifestyle, values)
4. **Any existing brand assets?** (logo, colors, fonts, previous designs)
5. **What competitors should we differentiate from?**

## Step 2: Gather Inspiration

Guide the user through the One-Thing Rule:
- Only lift ONE specific element (typography, color, OR layout) from any inspiration source
- Never copy multiple elements from the same reference
- Avoid lifting anything synonymous with an existing brand

Recommend sources:
- Dribbble, Behance, Awwwards for digital design references
- Pinterest boards organized by aspect (typography, color, layout)
- Fonts In Use for type direction
- Save references and note WHY each was saved

## Step 3: AI Brand Imagery (Optional)

If the user wants AI-generated brand imagery:
1. Build a prompt combining: subject + style + color + mood + composition
2. Generate 3-5 variations exploring different directions
3. Test across use cases: hero imagery, social posts, icons
4. Use AI output as directional, not final

## Step 4: Build the Mood Board

Assemble a single mood board (physical or digital) with:
- 5-8 images capturing the target mood
- Color swatches extracted from the images
- 2-3 font samples that match the direction
- Texture and pattern references
- Brief annotations on why each element was included

## Step 5: Create Brand Vibe Report

Document the findings:
- **Brand personality keywords** (3-5 adjectives)
- **Color direction** (warm/cool, saturated/muted, dominant hues)
- **Typography direction** (serif/sans/display, weight preferences, mood)
- **Imagery style** (photographic/illustrative, candid/polished, light/dark)
- **Overall design energy** (minimalist/maximalist, structured/organic)

## Step 6: Recommend Next Steps

- If color needs formalizing → recommend `/color-palette`
- If typography needs selecting → recommend `/type-system`
- If logo is needed → recommend `/logo-process`
- If project scope is unclear → recommend `/start-project`

## Output Format

Deliver the Brand Vibe Report as a structured document the user can reference throughout the project.
