# Logo & Brand Identity

The complete logo design process from research through delivery, synthesized from LogoCore (Kirill) and Will Patterson's Logo Launch. This skill covers every phase: research, ideation, sketching, vectorization, refinement, brand system creation, and client presentation.

**This skill runs as a parallel track** -- it can be activated alongside any layer in the design workflow.

- Designing a logo from scratch (research through delivery)

- Vectorizing hand-drawn sketches into production-ready logos
- Setting up logo color systems (RGB, CMYK, Pantone)
- Presenting logos and brand concepts to clients

- Exporting production-ready logo packages

## THE LOGO DESIGN WORKFLOW: 5 PHASES

### Phase 1: Research (1-2 weeks)

**Creative brief construction [Will Patterson]:**
- Write the brief FOR the client even if they provide one -- forces you to verify understanding
- Send it back for sign-off before starting design
- Use the client's exact language -- mirror their words back in presentations to build trust
- Key sections: about the client, due date, brand values/mission, target audience, project goals, competitors, deliverables
- Always add 2 extra weeks to any due date estimate

**Competitive/positioning mapping [Will Patterson]:**
- Map competitors on sliding scales: casual vs formal, premium vs affordable, modern vs traditional
- Reveals where visual territory is available and which archetype to pursue

**Mood boarding [Will Patterson]:**
- Find a feeling/atmosphere, not specific design elements
- Start from keywords derived from brief (e.g., "outdoors," "vintage," "signage")
- Look for color pattern emergence across images
- Pull palettes directly from mood board images using eyedropper
- See `mood-board-brand-vibe` for deeper brand direction workflow

**Mind mapping [Will Patterson]:**
- Write company name in center, branch to keywords, branch to visual associations
- Draw connection lines between branches that link
- Serves dual purpose: generates ideas AND creates the rationale for client presentations

**Stereotype research [LogoCore]:**
- Research AFTER designing, not before -- avoids subconscious copying
- Use Pinterest and Flat Icon to verify industry associations
- If your symbol triggers wrong industry, pivot specific elements

For detailed research frameworks, read `references/01-logo-process.md`.

### Phase 2: Ideation & Sketching

**Idea limitation rule -- max 2 ideas per logo [LogoCore]:**
A logo should express at most 2 core ideas. More creates an illustration, not a logo. Test: if a viewer has half a second, can they understand it?

**Fast sketching technique [Will Patterson]:**
- Use dot-grid sketchbook; give yourself 5 minutes total, ~1 minute per idea
- NO ruler, NO eraser, NO undos -- just get ideas out fast
- Fill in letter shapes by scribbling to see the final form
- Walk away after sketching; return fresh for refinement

**Kit bashing [LogoCore]:**
- Download free icons from Flat Icon/Noun Project to test compositions quickly
- Do NOT use downloaded icons in final logo -- only for experimentation
- Once you find a working composition, recreate from scratch

For complete sketching methodology, read `references/01-logo-process.md`.

### Phase 3: Vectorization

**Pen tool fundamentals [LogoCore]:**
- Use as FEW anchor points as possible -- more points = more errors and harder editing
- Click-and-drag for handles; Click-Alt to remove outgoing handle for sharp transitions
- The pen tool is "just a slower shape tool" -- prefer shape tools for geometric forms

**Key techniques:**
- Break logos into manageable pieces, not one continuous path; use Pathfinder to merge after
- Offset path technique: vector only innermost layer, then Object > Path > Offset Path for consistent stroke width (saves 50% of vectoring time)
- [CONTRARIAN] Never use Image Trace -- creates messy paths with unnecessary anchor points [Will Patterson]
- [CONTRARIAN] Non-linear workflow: avoid Object > Expand until final export to keep files editable [LogoCore]

For step-by-step vectorization, read `references/01-logo-process.md`.

### Phase 4: Production-Ready Refinement

**Testing protocol:**
- Mirror/reflect test: flip logotype horizontally so your brain stops reading and starts seeing shapes [Will Patterson]
- Blur/squint test: apply Gaussian Blur at high radius to reveal weight distribution imbalance [Will Patterson]
- Scale testing: favicon (64x64px), website header (~45px), business card, billboard [LogoCore]
- Vacuum-tight composition: less than 25% wasted space in the logo silhouette [LogoCore]

**Optical corrections:**
- Overshoot: curved shapes (O, U, C) must overshoot above/below guidelines to look aligned [Will Patterson]
- Boning effect: where straight meets curve, add anchor points and make transition more gradual [Will Patterson]
- Acute angles disappear at small sizes -- widen elements and add breathing room [LogoCore]

For refinement details and optical illusion corrections, read `references/02-logo-typography.md`.

### Phase 5: Brand System & Presentation

**Brand style guide [LogoCore]:**
- Purpose: prevent brand deterioration as more people use the logo
- Reuse your style guide template across clients
- Minimum sections: introduction, primary logo, variations, misuse rules, color palette, typography, dark/light versions

**Five keys to a winning presentation [Will Patterson]:**
1. Be prepared -- know talking points, strengths, weaknesses, have mockups ready
2. Be yourself -- don't imitate other presenters
3. Be authentic -- if you don't like part of the design, say so
4. Show off -- include process images, behind-the-scenes sketches
5. Meet face to face -- video call minimum; read body language

For complete style guide structure and presentation strategy, read `references/04-brand-style-guide.md`.

## LOGO TYPES DECISION TREE

```
What does the brand need?

1. Strong name recognition needed?
   -> Wordmark (Google, Coca-Cola, Visa)
   Best when: name is memorable and distinctive

2. Complex or hard-to-remember name?
   -> Icon/Symbol (Apple, Nike)
   Best when: logo needs to work at very small sizes

3. Want both distinctiveness AND name reinforcement?
   -> Combination mark (Adidas, Slack, Firefox)
   Most flexible approach

4. Long company name needing abbreviation?
   -> Lettermark/Monogram (HBO, CNN)
   Works when initials are memorable

5. Sophisticated, modern, conceptual?
   -> Abstract/Geometric (Mastercard, Airbnb)
   Concept must translate clearly to shape
```

**Logotype archetypes [Will Patterson]:** Monogram, clean combination mark, hand-drawn type, script, luxury (wide-tracked serif), stacked/badge, monoline script, brush script. Type choice has outsized impact on brand perception.

## LOGO DESIGN PRINCIPLES (NON-NEGOTIABLE)

1. **Simplicity** -- recognizable in 1-2 seconds; works at favicon size
2. **Scalability** -- vector-based; works from 1 inch to 10 feet
3. **Versatility** -- works in single color, reversed, on any background
4. **Timelessness** -- not dependent on current trends
5. **Distinctiveness** -- not confused with competitors; ownable
6. **Consistency** -- angles, stroke widths, and colors follow a system [LogoCore]
7. **Vacuum-tight composition** -- less than 25% wasted space in silhouette [LogoCore]

**Design in black and white first.** Add color as the last layer, not the first.

## COLOR FOR LOGOS

**Three rules [LogoCore]:**
1. Use bare minimum colors -- remove gradients and redundant tones
2. Apply bright/contrast colors only to the MOST IMPORTANT element
3. Logo must work on white, black, AND mid-gray backgrounds

**RGB vs CMYK decision:**
- Web-only: use RGB (sRGB color space)
- Even slight chance of printing: design in CMYK
- Strategy: create multiple variations -- RGB with gradients for web, CMYK outline for print
- Include Pantone spot colors for brand color fidelity across printers

For color psychology and associations, read `references/03-logo-production.md`.

## GRIDS AND THE GOLDEN RATIO

[CONTRARIAN] **Grids verify, they don't generate logos [LogoCore]:**
- No grid system is specifically created for logo design
- Use grids to check consistency AFTER designing, not to create the logo
- Plot horizontal and angled guides at key alignment points to verify

**Golden ratio (phi = 1.618):**
- Creates golden rectangle, golden spiral, golden angle (137.5 degrees)
- Useful for verifying proportional relationships in logo elements
- Build logos from repeating geometric ratios for visual consistency

**Grid construction workflow [LogoCore]:**
1. Start from outside, work inward
2. Place horizontal guides at top and bottom extremes
3. Add guides at each significant internal alignment
4. Convert to guides (right-click > Make Guides)
5. Adjust anchor points to snap to guide intersections

## COMMON LOGO MISTAKES

| Mistake | Cause | Fix |
|---------|-------|-----|
| Too complex | Too many details | Remove 30%+ of what you think is essential |
| Doesn't scale | Acute angles, thin intersections | Test at 16px, 32px, 64px while designing |
| Color dependent | Only works in color | Design in black first; monochrome must work |
| Poor letterforms | Generic typeface or inconsistent hand-lettering | Use premium typeface or consistent hand-lettering |
| Confusable | Too similar to competitors | Competitive audit is non-negotiable first step |
| Weak concept | Form without meaning | Start with concept, not form |
| Inconsistent | Varying angles, stroke widths | Use guides to verify; offset paths for consistent strokes |
| Wrong stereotype | Symbol triggers wrong industry association | Research industry associations after designing [LogoCore] |

## FREELANCING & BUSINESS

**Pricing progression [Will Patterson]:**
- Bespoke logotypes typically start around $10,000 minimum
- "Full price or free" philosophy -- never discount
- Value-based: charge based on value gained, not hours spent
- Specialize to charge more: "it's not how, it's who"

**Copyright [LogoCore]:**
- Designer owns copyright by default until transferred via written agreement
- Contract must include: scope, timeline, payment terms, revision limits, copyright transfer, kill fee

**Typeface licensing:**
- Google Fonts: free for all commercial use
- Foundry sites (MyFonts): per-platform licensing (desktop, web, app, book)
- Budget licensing fees into original client quote
- Cross-reference: `typography-systems` for deeper font licensing guidance

## QUICK START SCENARIOS

**"I have 2 weeks to design a logo"**
-> Compress research to 3 days, sketch for 5 days (100+ sketches), vectorize top 3 (3 days), refine and present (2 days), 3 days for revisions

**"Client doesn't know what they want"**
-> Run discovery workshop (1 hour), competitive analysis (2 hours), present 3 mood board directions before sketching, get approval on direction first

**"Logo works large but dies small"**
-> Reduce details by 50%, increase stroke weight, simplify curves, test at target sizes while refining

**"Client loves logo but brand system is weak"**
-> 3 core colors (primary, secondary, accent), 2 typefaces max, simple usage guidelines (1-2 pages), mockups showing application
