# Logo Typography: Logotype Archetypes and Optical Illusions

Type selection and optical correction techniques for professional logotype design.

---

## LOGOTYPE ARCHETYPES [Will Patterson]

| Archetype | Character | Best For |
|-----------|-----------|----------|
| **Monogram** | Single letter or initials as icon | Brands where initials are iconic (e.g., JJ, LV) |
| **Clean combination mark** | Clean font + icon side by side | Most versatile; works with/without name |
| **Hand-drawn type** | Intentional imperfections for human feel | Playful, artisanal, or craft brands |
| **Script logotype** | Flowing connected letterforms | Elegant, personal, boutique brands |
| **Luxury logotype** | Wide-tracked serif letterforms | Premium, high-end, fashion brands |
| **Stacked/badge** | Composed like stamps or crests | Vintage, heritage, established brands |
| **Monoline script** | Consistent stroke weight throughout | Modern, clean, approachable brands |
| **Brush script** | Variable stroke from brush dynamics | Energetic, creative, bold brands |

**Key insight [Will Patterson]:** Logotype design is "inherently more difficult and requires more knowledge in the design and typographic space." Bespoke logotypes typically start around $10,000 minimum because few designers actually design custom logotypes vs just picking fonts.

**Appropriateness principle:** Type style communicates industry, era, and personality before the word is even read.

## TYPEFACE SELECTION FOR BRAND IDENTITY

### Choosing a Companion Typeface [LogoCore]

1. Use gut feeling as first filter -- browse quickly, note what FEELS right
2. Test typeface with actual company name, not lorem ipsum
3. Keep the exact spelling/capitalization the client uses
4. Never add stroke to a typeface to make it bolder -- typefaces are carefully designed
5. Test multiple weights (light, medium, bold) to find optimal pairing with logo symbol
6. Wider tracking = more premium/luxurious feel

### Type-Personality Mapping [LogoCore + Will Patterson]

| Typeface Category | Brand Personality |
|-------------------|-------------------|
| Sans serif | Modern, approachable, tech-oriented |
| Serif | Established, educational, grounded |
| Script | Playful, vintage, human |
| Fantasy/display | Risky -- can shift brand perception to wrong industry |
| Slab serif | Strong, industrial, dependable |
| Geometric sans | Minimal, contemporary, design-forward |

**One typeface can pivotally change the entire direction of the logo** -- type choice has outsized impact on brand perception.

### Licensing Considerations [LogoCore]

| Source | License Type | Notes |
|--------|-------------|-------|
| Google Fonts | Free all commercial use | Desktop, web, app, book |
| MyFonts / foundry sites | Per-platform licensing | Desktop, web, app, book, server |
| DaFont | Often requires commercial license | Read fine print carefully |

Budget typeface licensing into original client quotes. Respect type designers: "There's no reason we should be using their work for free."

## OPTICAL ILLUSIONS IN LOGOTYPE DESIGN

### Overshoot [Will Patterson]

**Problem:** Curved shapes (O, U, C) placed mathematically on guidelines appear SHORTER than flat-topped letters (I, T, H).

**Why:** Humans perceive horizontal contact area more easily than vertical. A circle touching a line at a single tangent point looks shorter than a square sitting flat on the same line.

**Fix:** Extend curved shapes slightly above cap height and below baseline. The overshoot is invisible to viewers because it's designed to be hidden.

**Rule:** Every curved letter must slightly overshoot the guidelines to LOOK like it sits on the line.

### Boning Effect [Will Patterson]

**Problem:** Where a straight line abruptly meets a curve, the shape appears to collapse inward -- like the shape is "sinking in on itself."

**Fix:**
1. Add anchor points at the transition zone
2. Make the curve more gradual at the junction
3. Pull the transition slightly outward
4. Result: shapes that "feel stable" rather than contracting

**Critical for:** Monoline logotypes, icon connectors, anywhere straight meets curve.

### Weight Distribution [Will Patterson]

**Problem:** Even when mathematically equal, strokes can appear different weights at different angles. Horizontal strokes appear heavier than vertical strokes of the same width.

**Diagnostic:** Apply Gaussian Blur at high radius -- reveals dark blobs where weight accumulates.

**Fix:** Make horizontal strokes slightly thinner than vertical strokes so they APPEAR equal.

### Optical Center vs Mathematical Center

**Problem:** The mathematical center of a shape appears too low to human eyes. Text centered mathematically in a circle looks like it's sinking.

**Fix:** Shift content slightly above mathematical center. The amount varies by shape and context -- trust your eye over the numbers.

## LETTERFORM CONSTRUCTION PRINCIPLES

### Consistency Across Letters [Will Patterson]

- Even in sketches, aim for consistent stroke widths, angles, and proportions
- Look for letter-to-letter relationships: does the R share DNA with the A?
- Every letter in the logotype must feel like part of the same family

### Angle Consistency [LogoCore]

- If one letter uses a 10-degree slant, ALL slants should be 10 degrees
- Plot gridlines on both axes to verify
- Use Illustrator guides for precision
- Duplicate guide pairs and distribute across design

### Stroke Width Consistency [LogoCore]

- All similar elements share identical stroke widths
- Fluctuating widths look inconsistent and unprofessional
- Use offset paths to generate consistent outlines from single paths

### Counter Space (Interior Space)

- Counters (interior spaces of letters like O, A, B) should have consistent optical area
- Different geometric shapes (circular O vs triangular A) need different mathematical measurements to LOOK the same
- Larger counters = lighter feel; smaller counters = heavier feel

## KERNING AND SPACING IN LOGOTYPES

### Manual Kerning Rules

1. **Optical kerning, not metric:** Software kerning is a starting point but never final for logotypes
2. **Look at negative space between letters, not letter edges:** The shapes of the spaces between letters must feel equal
3. **Problematic pairs:** L+A, A+V, T+o -- these create large gaps that need manual tightening
4. **Round + round pairs:** O+O, C+O -- these can sit closer than flat-edged pairs

### Tracking Adjustment [Will Patterson + LogoCore]

- Wider tracking = more premium, luxurious feel
- Tighter tracking = more compact, modern, energetic
- ALL-CAPS logotypes almost always benefit from added tracking
- Lowercase logotypes: start at default, adjust by eye

### The Flip Test for Spacing

Flip the logotype upside down (or mirror it) to evaluate letter spacing without reading the word. When you can't read it, your brain evaluates pure shape relationships instead of semantic meaning.

## TYPE SELECTION DECISION FRAMEWORK

### When to Use Each Category

| Brand Context | Recommended Type | Why |
|---------------|-----------------|-----|
| Tech startup | Geometric sans-serif | Clean, modern, forward-looking |
| Law firm | Serif (Didone or Transitional) | Established, authoritative, trustworthy |
| Children's brand | Rounded sans or hand-drawn | Friendly, approachable, playful |
| Luxury fashion | High-contrast serif or wide-tracked sans | Sophisticated, elevated, minimal |
| Craft/artisan | Script or hand-lettered | Human, authentic, personal touch |
| Sports/fitness | Bold sans or condensed | Energetic, strong, dynamic |
| Restaurant (upscale) | Serif or elegant script | Refined, cultured, inviting |
| Restaurant (casual) | Rounded sans or brush | Relaxed, fun, welcoming |
| Financial services | Serif or neutral sans | Trust, stability, competence |
| Creative agency | Display or geometric sans | Bold, distinctive, design-forward |

### Type Testing Checklist

Before finalizing a logotype typeface:
- [ ] Tested with actual company name (not lorem ipsum)
- [ ] Tested at multiple weights (light, regular, medium, bold)
- [ ] Tested with tracking adjustments (tighter and wider)
- [ ] Tested in uppercase AND mixed case
- [ ] Checked licensing for all intended uses (desktop, web, app)
- [ ] Verified letterform quality at small sizes (no blurring, no collapsing)
- [ ] Compared side-by-side with 3-5 alternative typefaces
- [ ] Confirmed no unintended visual associations (wrong industry feel)
- [ ] Checked for problematic letter pairs in the company name (A+V, L+A, T+o)
- [ ] Reviewed with client's exact preferred capitalization

### Custom Logotype vs Existing Typeface

**Use existing typeface when:**
- Budget is limited (under $3,000 for logo)
- Timeline is tight (under 2 weeks)
- Brand needs to feel established/professional quickly
- Typeface licensing covers all needed platforms

**Create custom logotype when:**
- Budget supports bespoke work ($10,000+)
- Brand needs to be highly distinctive
- Existing typefaces don't capture brand personality
- Client values unique, ownable visual identity
- Letterforms need specific modifications for the brand name

## COMMON LOGOTYPE MISTAKES

| Mistake | Cause | Fix |
|---------|-------|-----|
| Using a free font without modification | Looks generic, not custom | At minimum adjust tracking, swap problem letters, or modify terminals |
| Inconsistent stroke weights across letters | Built letters independently | Use offset path; verify with guides |
| Ignoring overshoot on curved letters | Placed mathematically on baseline | Extend O, U, C slightly above/below guidelines |
| Kerning by metrics instead of optically | Trusted software defaults | Flip upside down and evaluate spacing visually |
| Wrong personality for brand | Chose typeface by aesthetics, not fit | Map brand personality to type category first |
| Scaling stroke weights with logo | Strokes become too thin at small sizes | Test at target sizes; consider variable stroke versions |
