# Brand Vibe System & AI Prompts

## BRAND VIBE REPORT TEMPLATE [Lane]

### Required Inputs

Collect as many of these as possible before generating:

```
BRAND VIBE REPORT INPUTS
────────────────────────
Brand name: [name]
Niche/industry: [what category]
What you sell: [products/services]
Target audience: [who, demographics, psychographics]
Founder story: [origin, motivation, personal connection]
Sales messaging: [key selling points, value proposition]
Brands you admire: [competitors or aspirational brands]
Desired feeling: [3-5 adjectives describing the brand energy]
```

### Report Output Structure

The brand vibe report should produce:
1. **Brand name and niche summary** -- one-paragraph positioning
2. **Sales messaging direction** -- key messages and value propositions
3. **Ideal client avatar** -- detailed description of who this brand serves
4. **Founder story** -- narrative that humanizes the brand
5. **Brand energy/feeling** -- emotional descriptors and personality traits
6. **Vibe strategy summary** -- how all elements connect

### Iteration Tips

- Edit the written report before regenerating images
- Text edits are 10x faster than image regeneration
- Save final report to Google Doc for future reference
- Upload to ChatGPT Project for persistent memory

## AI MOOD BOARD GENERATION [Lane]

### Prompt Template for Mood Board Images

```
Create a professional mood board image for [brand name], a [niche] brand.

BRAND VIBE:
[Paste brand vibe report summary here]

VISUAL DIRECTION:
- Color palette direction: [warm/cool/neutral + specific color words]
- Typography feel: [modern/classic/bold/elegant]
- Imagery style: [photography/illustration/mixed + adjectives]
- Overall energy: [3-5 adjectives]

OUTPUT REQUIREMENTS:
- Aspect ratio: 2:3 (portrait)
- Include: color swatches, typography samples, texture/pattern elements, imagery examples
- Style: cohesive, professional, suitable for client presentation
- Mood: [primary emotion to evoke]
```

### Iteration Prompts

| Situation | Prompt Addition |
|-----------|----------------|
| Too cold | "Make it warmer, more inviting, add earth tones" |
| Too generic | "More distinctive, add unexpected elements, more personality" |
| Wrong energy | "More [luxury/playful/corporate/edgy], less [current mood]" |
| Color issues | "More [color], less [color], make the palette more [cohesive/bold]" |
| Direction lost | Start new chat entirely with refined brand vibe report |

### Key Insight: Wide Then Narrow

1. Run 4-6 chats simultaneously with slight variations
2. Compare all results side by side
3. Identify the strongest direction
4. Iterate ONLY on the strongest direction
5. Refine at the vibe report level, not the image level

## AI PRODUCT MOCKUP PROMPTS [Lane]

### Seven-Principle Prompt Structure

```
[1. TASK] Take the uploaded [logo/artwork/design] and create realistic product mockup images.

[2. CONTEXT] I will upload [a logo/brand artwork]. Create lifestyle product photography showing this design applied to real merchandise and environments.

[3. NON-NEGOTIABLES] Do not alter the design in any way. No stretching, recoloring, distortion, or modification of the original artwork. Preserve exact proportions and colors.

[4. THINKING GUIDANCE] Analyze the design's style, colors, and energy. Choose suitable products and environments that complement the brand aesthetic.

[5. OUTPUT DESCRIPTION] Create: close-up shots showing print texture and material quality, wider lifestyle scenes with models or environments, clean studio shots on neutral backgrounds.

[6. BRAND TIE-IN] Reflect the brand's mood in background colors, props, textures, and lighting. Every element should reinforce the brand personality.

[7. VARIATION] Produce several images with different framings, compositions, and product applications. Keep tone unified and brand identity intact. Never compromise geometry or spacing of the original design.
```

### Platform-Specific Tips

**ChatGPT:**
- Best for mood boards and brand imagery
- Most consistent quality
- Good at interpreting written instructions
- Produces better results than image-reference approaches

**Google AI Studio:**
- Free access
- Square output by default
- Decent for product mockups
- More literal interpretation of prompts

**Google Gemini:**
- Stricter content policies
- Better at product mockups than mood boards
- Results can look dated compared to ChatGPT
- May refuse benign requests due to policy filters

## VISUAL BRAND GUIDELINES OUTPUT [Lane]

### What AI Should Extract from Mood Board

When uploading a mood board to generate written guidelines:

1. **Colors** -- specific hex codes for primary, secondary, accent, and neutral colors
2. **Typography direction** -- font style recommendations (serif/sans/display), weight preferences, sizing approach
3. **Imagery style** -- photography style, illustration approach, texture usage
4. **Layout feel** -- density, white space usage, grid suggestions
5. **Design system notes** -- button styles, card treatments, icon style direction

### Using Written Guidelines

Written guidelines serve as reusable instructions for:
- YouTube thumbnails
- Sales pages
- Product packaging
- Marketing materials
- Social media graphics
- Email banners
- Podcast artwork
- Any future brand content

AI is better at reading/interpreting text instructions than analyzing images -- written guidelines produce more consistent downstream results.
