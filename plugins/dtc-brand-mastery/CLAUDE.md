# DTC Brand Mastery v1.0.0 -- Routing & Operations

## Plugin Overview

Complete DTC e-commerce knowledge engine for building, launching, and scaling product brands across Shopify/DTC, Amazon FBA, and TikTok Shop. Built from 9 extraction files covering expert-level training from Gretta Van Riel, Jason Wong, Matt Clark, Melisa Vong, Kian Golzari, Kevin King and 20+ BDSS summit speakers, Incrementum Digital, Escala Academy, and Michelle (TikTok Shop). Provides end-to-end guidance from product ideation through exit.

## Instructor Disagreement Quick Reference

When instructors disagree, present BOTH positions and state when each applies:

| Topic | Disagreement | Resolution |
|---|---|---|
| Minimum markup | Gretta 2.5x / Matt 4x / Jason 5x / Melisa 30%+ | 4x for paid-ads DTC; 2.5x for organic; 30%+ for Amazon; 5x safest default |
| Validation method | Gretta interviews / Matt speed / Melisa data / Jason problem-first / Michelle TikTokability | Platform determines method |
| Pre-launch strategy | Gretta audience-first / Matt speed / Jason anti-delay | Route by available resources and skills |
| Payment terms | Gretta 50/50 / Jason 30/70 / Melisa 100% first order | 30/70 standard; first order may require more upfront |
| Organic vs. paid first | Gretta organic / Matt paid / Jason both | Start where your advantage lies |
| Manufacturing geography | Gretta local first / Jason both / Matt Google for ingestibles / Kian China ecosystem | Ingestibles = domestic; physical goods = China via Alibaba; innovation = local then global |
| Patents vs. speed | Gretta skip patents / Rich Goldstein file early | Do not delay launch, but file provisional for novel IP before disclosure |
| Brand name in title | Amazon instructors = yes / Michelle (TikTok) = maybe not | Platform-specific: Amazon always; TikTok test both; Shopify benefit-driven |
