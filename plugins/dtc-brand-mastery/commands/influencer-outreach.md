# Influencer Campaign Planner

## Step 1: Gather Essential Information

Ask the user these questions before proceeding. Do not skip any.

### Required Inputs
1. **Platform(s):** Instagram, TikTok, TikTok Shop, YouTube, or multi-platform?
2. **Product:** What are you selling? Price point? What makes it visually compelling?
3. **Budget:** Total budget for influencer/creator outreach (cash + product value)?
4. **Goal:** Brand awareness, direct sales, content generation for ads (UGC), or combination?
5. **Timeline:** When do you need content? Ongoing or campaign-specific?
6. **Existing assets:** Do you have product photos, a media kit, or previous influencer content?
7. **Stage:** Pre-launch, recently launched, or established brand scaling?

### Optional Inputs (Ask If Relevant)
- Do you have an existing TikTok Shop with affiliate program set up?
- Are you looking for creators to produce content you own (UGC) or to post on their channels?
- Do you have a landing page or Amazon listing to direct traffic to?
- What is your product cost for samples/gifting?

## Step 3: Tier and Budget Allocation

**Load reference:** `skills/13-influencer-content/ref-creator-outreach-templates.md` for templates.

### Recommended Tier Mix by Budget

| Budget Range | Recommended Mix |
|---|---|
| $0-$500 | Product gifting to 10-20 nano influencers (<10K followers) |
| $500-$2,000 | 5-10 micro influencers ($500-$2K each) + product gifting |
| $2,000-$10,000 | Mix: 2-3 macro + 10-15 micro + nano gifting program |
| $10,000+ | Full program: macro for awareness + micro for conversion + UGC platforms + retainers |

### TikTok Shop Creator Outreach (Volume-Based)
No cash budget required -- commission-based model.

```
CREATOR OUTREACH PLAN

Daily outreach target:          30-50 creators (manual)
                                1,000-2,500 (with bots)
Expected conversion rate:       ~10% (outreach to completed content)

To get 100 pieces of content:   ~1,000 creators contacted
To get 50 pieces of content:    ~500 creators contacted
To get 20 pieces of content:    ~200 creators contacted

Sample cost per creator:        $[product cost + shipping]
Total sample budget needed:     $[sample cost x expected approvals]
```

## Step 4: Outreach Plan

### For Instagram/TikTok Influencers

```
INFLUENCER OUTREACH PLAN

Campaign Name: [name]
Platform(s): [Instagram / TikTok / Both]
Timeline: [start date] to [end date]

TARGET CREATORS:
Tier 1 (Macro, 100K+): [X] creators
  Budget per creator: $[amount]
  Deliverables: [X Reels + X Stories]

Tier 2 (Micro, 10K-100K): [X] creators
  Budget per creator: $[amount]
  Deliverables: [X posts]

Tier 3 (Nano, <10K): [X] creators
  Compensation: Product exchange only
  Deliverables: [X posts -- flexible]

TOTAL BUDGET:
  Creator fees:              $[___]
  Product samples:           $[___]
  Shipping:                  $[___]
  Total:                     $[___]

TRACKING:
  Unique promo codes:        [CREATOR15, CREATOR20, etc.]
  Affiliate links:           [platform if applicable]
  UTM parameters:            [for website tracking]
```

### For TikTok Shop Creators

```
TIKTOK SHOP CREATOR PLAN

Week 1-2: Setup
- [ ] Set up affiliate program in TikTok Shop Seller Center
- [ ] Set competitive commission rate ([X]%)
- [ ] Prepare product samples for shipping
- [ ] Create outreach message templates
- [ ] Set up tracking spreadsheet

Week 2-4: Outreach Ramp
- [ ] Save 30-50 creators daily
- [ ] Create daily Target Collaborations
- [ ] Send personalized DMs to all invited creators
- [ ] Approve and ship sample requests
- [ ] Follow up with creators in Content Pending tab

Week 4+: Optimization
- [ ] Review content completion rates
- [ ] Identify top-performing creators
- [ ] Request spark codes from converting creators
- [ ] Scale outreach volume (consider bots)
- [ ] Set up GMV Max ads with converting content
```

## Step 6: Content Brief

Generate an influencer brief for the campaign.

```
INFLUENCER BRIEF

Brand: [name]
Product: [name and description]
Target audience: [demographics, interests]

Campaign objective: [awareness / sales / UGC / mixed]

Content requirements:
  Platform: [Instagram Reel / TikTok / YouTube Short]
  Format: [video under 30 seconds / photo + caption / carousel]
  Key messages (pick 2-3):
    1. [Main benefit]
    2. [Unique feature]
    3. [Social proof element]
  Hashtags: #[BrandHashtag] + [niche tags]

Do's:
  - Show product in use
  - Mention [key benefit]
  - Be authentic and natural
  - Tag @[BrandAccount]

Don'ts:
  - Don't mention competitors
  - Don't make [specific claims to avoid]
  - Don't use copyrighted music

Compensation: [details]
Deadline: [date]
Usage rights: [brand can reuse for ads for X months]
```

## Step 7: Performance Tracking

Recommend tracking setup:

- **For each creator:** name, content status, spark code status, orders, revenue, commission
- **Weekly review:** content completion rate, cost per content piece, ROAS by creator
- **Monthly review:** top 20% of creators (put on retainer at $500/month for 10-20 pieces)
- **Optimization:** double down on content formats that convert, cut underperformers
