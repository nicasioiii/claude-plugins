## Business Registration Requirements (Michelle)

### For US Corporation or Partnership
- **SS4 Letter from the IRS** showing business name, EIN, and registered business address
  - MUST be the **original IRS-mailed letter**
  - The digital EIN copy IS NOT ENOUGH and will NOT be accepted
  - If lost: call (800) 829-4933 to request a copy
- Business must be registered in the United States
- Must be OPERATED by a beneficial owner within the United States
- International sellers need US-based contacts to represent their business

### Critical Document Rule
**Convert ALL requested documents to JPG format.** The system claims to accept PDF and PNG but reads JPGs best.

### Brand Registry Note
Trademark is NOT required to sell as a brand on TikTok Shop (or Amazon). Proceed with selling and apply for brand authorization later.

## Social Account Setup -- Critical Geo-Locking Rules (Michelle)

### GEO-LOCKING IS PERMANENT
- TikTok social accounts sync with the IP address/geo where the account is created and accessed
- If a VA in another country sets this up without a VPN or US SIM card, the account locks to their country
- An account created in a different country WILL NOT work for linking to TikTok Shop
- If account is created in the US but a VA accesses from outside the US, the account SWITCHES to that country

### Required Setup
1. Download TikTok on a **US-based mobile device** (US phone or phone with US SIM card)
2. Select "Setup" and "Use phone or email"
3. Pick a birthday -- **DO NOT make one up** (ID verification will check later)
4. Set up as personal account
5. Complete identity verification within 24 hours (arrives in TikTok inbox)
6. **ALL team members must ALWAYS use a US VPN** when accessing the account

### Cannot-Do List
- Cannot complete full setup via desktop only -- TikTok requires mobile
- Cannot use same email login for social account and Shop account (separate passwords)

## Account Linking Process (Michelle)

### Prerequisites
- All registration paperwork completed
- TikTok app downloaded and signed into social account on phone
- Social account must be US-based

### Steps
1. Navigate to "Get Started" in TikTok Shop
2. Click "Link Official Account"
3. On phone: TikTok app -> profile -> bars (upper right) -> QR code -> scan
4. Scan QR code shown on desktop
5. Authorize access for brand and shoppable content
6. After linking: gray "TikTok Shop" bar appears on profile
7. Click gray bar -> authorize popup for branded/shoppable content

## Customer Support Setup -- Account Health Critical (Michelle)

### Why It Matters
- Unread/unreplied chats **directly affect brand rating and shop rating**
- Shop rating is **public information**
- Too many unreplied messages -> lose TikTok Shop benefits -> could lose account

### Setup Process
1. Go to Account Settings -> User Management
2. Invite team as **Customer Service Agent** role
3. Also assign **Order Fulfillment Specialist** role
4. Set up FAQ card in chat greeting (shipping, refunds, common questions)
5. Create custom FAQs as needed
6. Set up saved replies in brand's tone of voice

### AI Chatbot (Recommended)
- Activate chatbot **ALL DAY** -- have human check when online
- Leave business hours open (don't restrict to specific hours)
- Has "recommended answer" feature with built-in AI

## Shipping Setup (Michelle)

### Warehouse Setup
- Set up both **pickup warehouse** and **return warehouse**
- If using Amazon MCF: use home/business address (NOT Amazon warehouse address)

### Shipping Template
1. Create ONE shipping template
2. Select **standard shipping**
3. Select ALL states except Hawaii and Alaska
4. Flat rate: **$5-$8** (TikTok restricts higher rates)

### Two Shipping Modes
| Mode | Use When |
|---|---|
| TikTok Shipping | Using 3PL or shipping yourself |
| Seller Shipping | Using Amazon MCF (must create template first) |

### Amazon MCF Fulfillment
Three methods: TikTok for Shopify Channel (free), Aftership ($11-39/mo), WebBee (paid).
**For detailed sync configuration** -> See `skills/shipping-fulfillment/SKILL.md` and its reference file.

## Team Management (Michelle)

### Three Core Functions
1. **Operations Management** -- shop monitoring, sales review, product catalogue
2. **Creator/Affiliate Outreach** -- daily research, outreach, sample management
3. **Customer Support** -- buyer messages, review management

### User Roles
| Role | Access | Use For |
|---|---|---|
| Main Administrator | Full access (everything) | Operations managers |
| Affiliate Manager | Affiliate Center + possibly Order Fulfillment | Creator outreach team |
| Customer Service Agent | Chat and support | Support team |

**Important:** Main Administrator is NOT the same as Owner account -- different permissions.

### Adding Users
1. My Account -> Account Settings -> User Management -> Add User
2. Select role-based access
3. Enter email and language
4. Submit (activation email expires in **120 hours / 5 days**)

### Email Workaround for VAs
TikTok only allows emails not currently associated with another Shop. Use Gmail **+1, +2, +3** format (e.g., name+1@gmail.com).

## GMV Max Ads -- Setup and Optimization (Michelle)

### When to Use What
- **Video Shopping Ads:** When you don't have enough content yet
- **GMV Max:** When you have proven converting content (don't launch without it)

### GMV Max Setup
1. Marketing -> Shop Ads -> Create GMV Max Ads
2. **One product per campaign** (underperformers drag down overall performance)
3. Optimization goal: **GROSS REVENUE** (not ROI target)
4. Start with TikTok's recommended ROI target
5. Budget: TikTok provides recommendation (not guaranteed spend)
6. Schedule: **Run continuously**
7. Video selection: **Auto select** (new affiliate content auto-included)

### ROI Target -- Teeter-Totter Relationship
- Higher ROI target = less budget spent (more selective)
- Lower ROI target = more budget spent (wider reach, higher cost)
- Like Amazon TACOS: lower target sacrifices efficiency for visibility

### Optimization Process
1. Start with TikTok's suggested ROI target
2. If not getting traction: lower ROI target (e.g., from 4.6 to 2)
3. As orders increase: raise ROI target back up toward 5
4. **Prioritize profitability over reach**

### Creative Performance -- Two Tiers
1. **Conversion videos** -- high orders and revenue (scale these)
2. **Awareness videos** -- high impressions and clicks but lower conversion (still valuable for funnel)

### Spark Code Management
- Unauthorized videos = shoppable posts without valid spark codes
- These are converting organic content you can't promote via ads
- **Action:** Contact creator to request or extend spark code
- Check unauthorized videos regularly -- they represent untapped ad potential

## Complete 22-Step Launch Sequence (Michelle)

1. Validate product meets 3/5 TikTokability criteria
2. Review prohibited/restricted products list
3. Complete business registration (SS4 letter, JPG documents)
4. Create US-based TikTok social account (US device/VPN)
5. Verify identity on social account
6. Link social account to TikTok Shop
7. Connect ad account
8. Set up customer support (FAQs, chatbot, saved replies)
9. Set up warehouse (pickup + return addresses)
10. Set up shipping template ($5-$8 flat rate, exclude HI/AK)
11. Choose shipping method (TikTok Shipping or Seller Shipping)
12. Install fulfillment integration (Aftership, WebBee, or Shopify channel)
13. Configure sync settings (inventory only, 1-hour delay, expedited shipping)
14. Run test order to verify full pipeline
15. Import reviews (Amazon -> Shopify via Yotpo -> TikTok Shop)
16. Add team members with role-specific access
17. Begin creator outreach (30-50 daily via Target Collaborations)
18. Set up 3 simultaneous promotions (free shipping + discount + coupon)
19. Launch Video Shopping Ads first (if limited content)
20. Graduate to GMV Max once converting content established
21. Set up weekly customer marketing campaigns
22. Optimize GMV Max (adjust ROI target, review creatives, collect spark codes)

## Anti-Patterns (17 Mistakes to Avoid)

1. Auto-syncing product details/titles/prices from Amazon to TikTok
2. Syncing orders to Amazon immediately (wait 1 hour for cancellation window)
3. Mapping categories from Amazon to TikTok (handle manually)
4. Using digital EIN copy instead of original IRS SS4 letter
5. Letting VAs access TikTok without US VPN (geo-lock risk)
6. Making up a birthday during account creation (ID check will fail)
7. Leaving customer support messages unread (kills brand rating)
8. Launching GMV Max without proven converting content
9. Running multiple products in one GMV Max campaign
10. Ignoring unauthorized videos in GMV Max (untapped spark code potential)
11. Using Shopify integration for product listings (use only for fulfillment)
12. Skipping test orders after fulfillment setup
13. Using free Yotpo plan for review importing (doesn't work)
14. Mismatching SKUs between Shopify and TikTok (reviews fail to link)
15. Charging more than $8 for shipping (TikTok blocks it)
16. Giving team members your owner account (use role-based access)
17. Using emails already tied to another TikTok Shop (use Gmail +1 workaround)
