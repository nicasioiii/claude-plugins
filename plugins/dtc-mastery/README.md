# DTC Mastery v2.0.0

Complete DTC brand building knowledge engine covering the full product-to-exit lifecycle across Shopify, Amazon, and TikTok Shop.

## Skills (15)

### Layer 1: Foundation
| Skill | Description |
|---|---|
| `product-ideation` | PSSP, PRIME, 4-criteria, TikTokability, validation, customer personas |
| `brand-strategy` | Brand pyramid, naming, positioning, tone of voice, archetypes, category-of-one |
| `product-development` | Spec sheets, prototyping, packaging, compliance, 3-stage sampling |
| `sourcing-suppliers` | Alibaba audits, import records, 10-3-1, negotiation scripts, MOQ reduction |
| `quality-inspection` | AQL 2.5, defect classification, failed inspection plans, packaging optimization |

### Layer 2: Platform Setup
| Skill | Description |
|---|---|
| `shopify-store-setup` | 14-step checklist, RPV maximization, upsell funnels, bundles, subscriptions |
| `amazon-listing-optimization` | SEO titles, bullets, backend keywords, A+ content, Rufus/Cosmo, reviews |
| `amazon-ppc-advertising` | Campaign structure, placement strategy, bid optimization, DSP framework |
| `tiktok-shop-setup` | 33-step launch sequence, geo-locking, fulfillment integration, GMV Max ads |

### Layer 3: Growth
| Skill | Description |
|---|---|
| `content-creator-strategy` | Affiliate outreach, UGC, spark codes, content repurposing, influencer partnerships |
| `launch-strategy` | Multi-platform timing, pre-launch audience, giveaways, time-release framework |
| `pricing-unit-economics` | Markup rules, landed cost, RPV formula, break-even ROAS, tariff optimization |
| `external-traffic-list-building` | Blog-to-Amazon funnels, Google Business Profile, attribution links |

### Layer 4: Scale & Exit
| Skill | Description |
|---|---|
| `operations-systems` | 5-level process hierarchy, SOPs, outsourcing, AI automation, technology audits |
| `exit-valuation` | Valuation multiples, IP timing, DTC premium, aggregators, donation tax strategy |
