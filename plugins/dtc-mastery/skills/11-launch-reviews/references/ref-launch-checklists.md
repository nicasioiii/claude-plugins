## DTC 12-WEEK LAUNCH CHECKLIST

### Phase 1: Foundation (Weeks 1-4)
- [ ] Validate product against Four Criteria (big market, healthy margins, messaging potential, media buying potential)
- [ ] Source supplier and order samples
- [ ] Define customer persona (demographics, psychographics, pain points)
- [ ] Set up Shopify store (account, theme, pages, payment)
- [ ] Create product photography (lifestyle + studio)
- [ ] Write product copy using PASTOR framework
- [ ] Set up email capture (popup, promo bar)
- [ ] Configure Klaviyo for abandoned cart flow
- [ ] Create brand assets (logo, color palette, brand guidelines)
- [ ] Set up social accounts (Instagram, TikTok, Facebook)

### Phase 2: Testing (Weeks 5-8)
- [ ] Create 3-5 ad creative variations
- [ ] Launch Facebook/Instagram test ads ($100-500)
- [ ] Build email list (target: 500+ before launch)
- [ ] Begin influencer outreach (50 samples for content)
- [ ] Order inventory (bulk order placed)
- [ ] Set up full email automation (welcome, abandoned cart, post-purchase)
- [ ] Set up customer service system
- [ ] Create landing page with 5 hook variations for A/B testing

### Phase 3: Launch (Weeks 8-12)
- [ ] Soft launch: email list early access with exclusive discount
- [ ] Activate influencer content and shout-outs
- [ ] Run limited paid ads to soft launch audience
- [ ] Collect feedback and iterate
- [ ] Full launch: inventory arrives, scale winning ads
- [ ] Activate full organic social calendar
- [ ] Ask for reviews 3-5 days after purchase
- [ ] Run full test purchase to verify entire pipeline

**Week 12 targets:** 100-1,000+ email subscribers, initial ad data, 50-500 customers.

## TIKTOK SHOP 22-STEP LAUNCH SEQUENCE

1. Validate product meets 3/5 TikTokability criteria
2. Review prohibited/restricted products list
3. Complete business registration (SS4 letter, JPG documents)
4. Create US-based TikTok social media account (US device/VPN)
5. Verify identity on social account
6. Link social account to TikTok Shop (QR code scan)
7. Connect ad account
8. Set up customer support (FAQs, chatbot, saved replies)
9. Set up warehouse (pickup + return addresses)
10. Set up shipping template ($5-$8 flat rate, exclude HI/AK)
11. Choose shipping method (TikTok Shipping OR Seller Shipping)
12. Install fulfillment integration (Aftership, WebBee, or Shopify channel)
13. Configure sync settings (inventory only, 1-hour delay, expedited shipping)
14. Run test order to verify full pipeline
15. Import reviews (Amazon -> Shopify via Yotpo -> TikTok Shop)
16. Add team members with role-specific access
17. Begin creator outreach (30-50 daily, scale to 1,000+ with bots)
18. Set up 3 simultaneous promotions (free shipping + discount + coupon)
19. Launch Video Shopping Ads first (if limited content)
20. Graduate to GMV Max Ads once converting content is established
21. Set up weekly customer marketing campaigns
22. Optimize GMV Max campaigns (adjust ROI target, review creatives, collect spark codes)

## VINE ENROLLMENT & STACKING GUIDE

### Enrollment Requirements
- Amazon Brand Registry required
- Product must be in new or near-new condition
- Fee varies by product category

### Six-Star Review Hack
1. Package product in premium gift box (not standard packaging)
2. Include a bonus complementary product (low cost, high perceived value)
3. Add insert card with tips or instructional video link (value-add only, NOT review solicitation)
4. Vine reviewer expects basic product, receives premium experience
5. Surprise and delight drives 5-star sentiment

### Vine Stacking Method
1. Create multiple size/color variations for your product
2. Each variation qualifies for up to 30 Vine units
3. Example math: 10 colors x 2 sizes = 20 variations x 30 units = 600 Vine reviews
4. Do NOT merge variations immediately
5. Wait to see which variations receive best reviews
6. Merge strategically after review data comes in

## PRODUCT INSERT DESIGN GUIDELINES

### Insert Offer Types
- Free additional product
- Extended warranty
- Discount on next order
- Desirable information (e.g., fitness brand offers meal plans or workout eBooks/videos)

### Design Principles
- Simple, clean design aligned with brand identity
- Clear call-to-action (one primary action)
- QR code prominently placed
- No explicit review solicitation (TOS violation)
- Include customer support email for issues (diverts complaints from public reviews)
- "Our small, family-owned business relies on your feedback" tone

### QR Code Setup Process
1. Go to qr-code-generator.com
2. Create account (free 14-day trial, then paid for dynamic editing)
3. Choose "Website" type
4. Enter destination URL (can be updated later with paid plan)
5. Customize: style, colors, add brand logo in center
6. Download QR code file
7. Give to designer for insert layout

### Insert Printing Options
- **Cheapest:** Ask supplier if they can print inserts in China (or local business card printer)
- **If supplier cannot help:** Search Alibaba for "business card printing" -- affordable in-China shipping
- **US-based:** Use Vistaprint
- Inserts are recommended but NOT required for first order -- do not let them delay launch

## REVIEW IMPORTING PIPELINE (Amazon -> TikTok Shop)

### Step 1: Export Amazon Reviews
1. Install Chrome extension (Helium 10 Review Insights)
2. Navigate to your Amazon product listing
3. Filter for 4 and 5 star reviews only (3 stars optional)
4. Click "Export Table" to download reviews as CSV

### Step 2: Import Reviews into Shopify via Yotpo
**App:** Yotpo Product Reviews & UGC (NOT Yotpo email marketing)

**CSV Template Requirements:**
- Download Yotpo's CSV template from import section
- Date format: YYYY-MM-DD (year-month-day with leading zeros)
- Shopify product handle: found at end of product page URL
- Images: only ONE image URL per review line
  - Workaround: if one review has multiple images, put extras in reviews that have no images
- Public review field: type TRUE (capital letters)
- Review emails: generate placeholder emails (randomemails.com or similar)
- Do NOT copy over instruction row -- keep as reference, delete before import

**Common Import Failures:**
- Date formatting errors (most common failure)
- Missing "TRUE" in public review field
- Incorrect Shopify product handle URL

### Step 3: Import from Shopify to TikTok Shop via Yotpo
1. Go to reviews.yotpo.com (or via Shopify app)
2. Upgrade to PAID Yotpo plan (required -- free plan does not work)
   - Select monthly billing (cancel after import)
   - Lower monthly order estimate to reduce costs
3. Contact Yotpo support -- tell them you want to import Shopify reviews as **verified** to TikTok Shop
4. Yotpo processes reviews (takes ~4 hours)
5. Reviews appear on TikTok Shop within 3 days

### Critical SKU/Barcode Matching
- Yotpo matches products between Shopify and TikTok Shop via SKU and barcode (UPC/GTIN)
- These often do not match between platforms
- Even if you need to create a matching SKU, ensure it matches on both platforms
- Check TikTok Shop: Sales Information -> Product Identifier Code and Seller SKU
- Both must match Shopify for review import to link correctly

### Post-Import: Cancel Yotpo Subscription
- Return to Billing Settings after import completes
- Cancel before next billing cycle if only needed for one-time import

## PRE-LAUNCH AUDIENCE BUILDING

### Gretta's 5-Stage Funnel
1. **Content Community** -- Create content around the problem/niche before product exists
2. **Engaged Audience** -- Grow following; engagement rate matters more than follower count
3. **Email List** -- Convert followers to subscribers; create waitlist
4. **Paying Customers** -- Day-one conversion from list
5. **Brand Advocates** -- Repeat buyers and referrers

### Waitlist Targets
- Minimum viable: 500 email subscribers before launch
- Strong position: 2,000-5,000 subscribers
- The Fifth benchmark: 8,800 waitlist subscribers -> $100K+ launch day

### Audience Building Tactics
- Repost curated content with credit in your niche
- Run co-branded giveaways with similar-sized accounts
- Post daily (consistency = 7x weekly interactions vs. 1x)
- Use content territories (3-5 topic categories aligned with brand)
- Send 50 product samples to influencers before launch for content
- Leverage existing audiences via shout-for-shout exchanges
