# Shopify Store Setup -- Complete Reference

## 14-Step Store Setup Checklist (Matt Clark)

1. Create Shopify store
2. Add first product
3. Choose theme
4. Update company/contact info
5. Activate Shopify Payments
6. Create Privacy, Terms, Refund policies
7. Create Shipping & Returns page
8. Create Contact page
9. Create navigation menu
10. Add custom domain
11. Add remaining products
12. Set up fulfillment (choose from 3 options below)
13. Configure sales tax
14. Activate store

---

## Fulfillment Options Comparison

| Option | Best For | Pros | Cons |
|---|---|---|---|
| **Amazon FBA** | Sellers already on Amazon | 2-day shipping, Amazon handles returns, Shopify-FBA integration exists | Amazon fees, less brand control, FBA inventory limits |
| **Ship yourself** | Very low volume initially | Full control, lowest startup cost | Not scalable, time-intensive |
| **Third-party logistics (3PL)** | Scaling brands | Scalable, professional, brand-controlled packaging | Monthly minimums, setup time, variable costs |

**Gretta's progression:** Self-ship to start --> 3PL when volume justifies (The Fifth Watches, SkinnyMeTea, Drop Bottle all scaled through this path).

**Clark's recommendation:** Amazon FBA integration for Shopify is the easiest path if you already have FBA inventory.

## 6 Sales Optimization Steps (Clark)

Execute after store is live, before driving paid traffic:

1. **Set free shipping threshold** -- drives higher AOV
2. **Add storewide coupon** -- reduces friction for first purchase
3. **Improve one product page** with strong copy (reference PASTOR framework from cro-copywriting-mastery)
4. **Get 3+ reviews on product page** -- minimum social proof
5. **Add sales-boosting guarantee** -- reduce perceived risk
6. **Do live test purchase** -- verify entire checkout flow works

## Bundle Strategy (Clark)

### Basic Bundles
- **App:** "Bundle Products Upsell" by Thematic in Shopify App Store
- **Structure:** Buy 2 = X% off, Buy 3 = Y% off (e.g., 20%, 30%)
- Show on product page (not just checkout)
- Can also manually create bundle as separate product listing

### Super Bundle
- Combine ALL products into one mega-bundle
- Example: LifeBoost 12-bag bundle at $200
- Even if only 1 sale/week, extremely profitable due to high AOV
- Can fulfill manually if needed (worth it at that price point)
- Display near top of collection page for visibility

## Subscription Implementation (Clark)

- Add subscription option to products via Shopify subscription app
- LifeBoost: ~50% of $2.6M/month revenue is subscription
- **Best for:** Consumable products (coffee, supplements, beauty, food)
- **Non-consumables:** Create subscription via complementary consumables, refills, or maintenance products
- Promote subscription through: autoresponder emails, product pages, store banners, package inserts

## Off-Amazon Sales Machine -- Weeks 1-2 (Matt Clark / ASM14)

### Week 1: Maximize Revenue Per Visitor
- Add irresistible guarantee (header, footer, product pages, cart page)
- Display promo bar (free shipping threshold, current offers)
- Add popup for email + SMS capture (Klaviyo; 15% off coupon)
- Set up autoresponder flow (deliver coupon; escalate discount if no purchase)
- Add abandoned cart flow
- Install upsells (Zipify OCU): Pre-purchase > First upsell > Downsell > 2nd upsell
- Create catch-all upsell funnel
- Add basic product bundles on collection pages
- Create super bundle at $100-$200+ price point; display near top of collection
- Execute first customer reactivation campaign (72-hour window; near-break-even discount; promote via every channel)
- Plan one additional promotion per week for next 4 weeks (always give a reason: holiday, anniversary, event)

### Week 2: Create Best Landing Page
- Select ONE product: 4X markup over cost, greatest mass appeal, several hundred units in stock
- Gather maximum proof: written reviews, star ratings, total 5-star count, total customers, years in business, photo/video testimonials, social screenshots, media logos
- Plan offer tiers: 1-unit, 3-unit, 6-unit bundles
- Include guarantee + scarcity element
- Create 5 different hooks with unique headlines
- Build 5 identical landing pages with 1 unique headline each (Clark's 5-hook test method)
- Test on desktop AND mobile

**Part 1 -- Set Up Store:** Products, theme, policies, contact page, domain, fulfillment (FBA/manual/3PL), sales tax, activation

**Part 2 -- Optimize for Sales:** Free shipping threshold, storewide coupon, promo bar app, improve product page copy, 3+ reviews, guarantee, live test purchase

## Post-Purchase Upsell Detail (Gretta + Clark)

**Gretta:** Post-purchase is the most forgotten but arguably most important funnel stage. Capitalize on dopamine/excitement right after purchase. Use Shopify app "Checkout Share" to encourage social sharing. Avoid buyer's remorse with immediate engagement.

**Gretta's tactics:**
- Free gift with purchase (Shopify app "Ultimate Special Offers" -- auto-applies at checkout)
- Automate post-purchase emails: order confirmation, shipping notification, delivery confirmation, review request

**Clark's 4-offer upsell structure:** Pre-purchase popup (20% off same product) > Post-purchase upsell #1 (25% off) > Downsell if declined (30% off or different product) > Upsell #2 if accepted (multi-pack or complementary product)
