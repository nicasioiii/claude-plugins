## Workflow

Strategy/Economics -> Research -> Offer Design -> Funnel Architecture -> Page Copy -> Upsell/Downsell -> Optimize/Scale

## Key Frameworks

- **CFA Three Levels** (Hormozi) -- Client-Financed Acquisition progression
- **15-Factor Scoring** (Georgi) -- Quantitative funnel complexity assessment
- **CONVERSIONS** (Gusten) -- 11-element sales page framework
- **3T Offer Triangle** (Gusten) -- Transformation, Transportation, Transaction
- **Four-Pronged Money Model** (Hormozi) -- Attraction, Upsell, Downsell, Continuity
- **Position/Price/Switch** (Katie) -- Systematic upsell optimization
- **5D CRO System** (Gusten) -- Discover, Design, Develop, Document, Deploy
- **RMBC** (Georgi) -- Research, Mechanism, Brief, Copy

## Instructor Disagreements (Documented)

1. AIDA framework (Georgi supports vs. Gusten replaces) -- resolved with Schwartz as canonical
2. Upsell page length (Katie: short vs. Georgi: longer video) -- route by business model
3. Number of upsells (Gusten: 1-2 vs. Katie: up to 4 vs. Hormozi: 5+) -- route by fulfillment model
4. Discount philosophy (Hormozi: never moderate vs. Katie: 80% off backend) -- route by funnel position
5. When to sell high-ticket (Katie: warm first vs. Hormozi: point of deprivation) -- route by whether next problem is felt
