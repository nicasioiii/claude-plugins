# Value Ladder Design

## Profit Maximizer Path (Gusten)

The complete ascension model from free to backend:

### Level 1: Free
- **Products:** Ebook, checklist, training video, challenge, free trial
- **Delivery:** Automated
- **Purpose:** Get leads and prove your competence
- **Key metric:** Opt-in rate (10-47% cold, 90%+ warm)

### Level 2: Low Ticket ($1-$100)
- **Products:** Templates, workshops, bundles, mini-courses
- **Delivery:** Automated
- **Purpose:** Create impulse buyers and build customer list
- **Key metric:** AOV of $70+ (with bumps and upsells)

### Level 3: Automated Offer ($100-$2,000)
- **Products:** Courses, memberships, software subscriptions
- **Delivery:** Automated (no calls needed up to ~$2,000)
- **Purpose:** Passive income and customer base building
- **Key metric:** Conversion rate from low-ticket customers

### Level 4: Backend ($3,000-$25,000+)
- **Products:** Coaching, consulting, done-for-you, masterminds
- **Delivery:** Application + call
- **Purpose:** Highest revenue per customer, direct access
- **Key metric:** Close rate and LTV

## Seven-Step Profit Maximizer Flywheel (Gusten)

1. **Find** -- New customers through content/ads
2. **Convert** -- Funnel converts visitors to buyers
3. **Upsell** -- Next logical offer on upsell page (one-click charge)
4. **Cross-sell** -- Supporting products within 30-60-90 days via email
5. **Ascend** -- Backend high-ticket offer for deeper access
6. **Retain** -- Monthly subscription/software for recurring revenue
7. **Ignite** -- Turn customers into promoters (affiliates, word-of-mouth, ambassadors)

When step 7 feeds back into step 1, you have a self-reinforcing flywheel that grows without you.

## Hormozi's Complete Money Model Sequence

### The Full Stack (Gym Example -- Day 0-28)

**Day 0:** Attraction offer (free 21-day transformation / 6-week challenge)

**Day 0-2:** Core sale (sell the goal: $500, $1K, $2K packages) + Supplements upsell ($85 GP average)

**Day 2-7:** Meals upsell

**Day 14-21:** Continuity conversion (downsell the upsell into membership)

**Day 21-28:** Prepayment offer on continuity

**For every "no" at each stage:** Run the downsell flow:
- Same product, different terms (payment plan)
- Stripped version (fewer features)
- Free orientation
- Sell supplements anyway (magic pill for those who don't want work)

### Building Order (One Stage at a Time)
1. Get customers reliably
2. Make sure they pay for themselves reliably (CFA Level 2)
3. Make sure they pay for OTHER customers reliably (CFA Level 3)
4. Maximize each customer's long-term value
5. Optimize and add layers

### The $100M Money Model Definition
"A good money model makes more profit from a customer than it costs to get and service them in the first 30 days. A $100M money model makes more profit from ONE customer than it costs to get and service MANY customers in the first 30 days."

## Katie's Offer String (Digital Products)

Complete digital product funnel architecture:

1. **Main Product** (front-end, e.g., $44)
2. **Order Bump** on checkout page ($27-$37)
3. **Upsell 1** immediately after purchase ($67-$97)
4. **Upsell 2** after upsell 1 ($197)
5. **Upsell 3** after upsell 2 ($44-$77)
6. **Upsell 4** after upsell 3 ($44-$77)
7. **Backend email sequences** (multiple product promotions over days/weeks)

**For beginners:** Start with main product + bump + one upsell. Build more over time.

## Starter Advice by Business Stage

### Brand New (No Revenue)
- Pick ONE funnel, ONE offer, ONE avatar
- Build a lead generation funnel + one low-ticket offer
- Get to 30 customers before optimizing anything
- Don't build 4 upsells before you've made a single sale

### $10K-$50K/month
- Your funnel works; now add layers
- Add order bump and first upsell to existing funnel
- Start backend email sequences
- Consider adding a second funnel type (e.g., challenge to feed application funnel)

### Capped at $50K-$100K/month
- You've likely exhausted product-aware and most-aware segments (Georgi's scaling challenge)
- Need to reach problem-aware market with longer funnels and more education
- Consider adding advertorial path before existing funnel
- Test VSL for cold traffic
- Add continuity/subscription for LTV

### $100K+/month
- Full money model with multiple offer layers
- Multiple traffic sources with platform-congruent funnels
- CRO program with systematic testing
- Backend email sequences generating 2x+ front-end revenue
- Continuity/subscription optimized for churn reduction

## The Funnel Ecosystem Business Model (Gusten)

For mature businesses, multiple funnels work together:

- **Traffic funnels** (content, ads) feed into...
- **Lead funnels** (opt-in pages) which feed into...
- **Monetization funnels** (sales pages, VSLs) which feed into...
- **Maximizer funnels** (upsells, backend, continuity) which feed into...
- **Retention systems** (community, content, email) which feed back into...
- **Referral systems** (affiliates, ambassadors) which generate new traffic

Each component reinforces the others. The system grows without linear increases in ad spend.

## Instructor Alignment: Start Simple

All four sources agree: start simple.

- **Hormozi:** Build one layer at a time like layers of paint
- **Gusten:** One funnel, one offer, one avatar
- **Katie:** One bump + one upsell to start; capacity and cash flow must go together
- **Georgi:** Get the economics working on a basic funnel before adding complexity

The temptation to build everything at once is the most common failure mode. Revenue funds the next layer.
