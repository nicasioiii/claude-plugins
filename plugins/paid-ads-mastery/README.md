# Paid Ads Mastery v2.0

Complete paid advertising knowledge engine for Meta, Google, and TikTok Ads.

## Overview

Built from 610+ expert-level training transcriptions across 9 courses, this plugin provides end-to-end guidance from account setup through advanced scaling. It covers tracking infrastructure, creative research, campaign strategy, ad production (traditional and AI), testing methodology, optimization diagnostics, and scaling playbooks.

## Skills (16 Total)

### Layer 1: Setup
- **setup-meta-tracking** -- Meta Pixel, CAPI, server-side tracking, account health, payment hygiene
- **setup-google-ads** -- Google Ads conversion code, Merchant Center approval, product feed optimization

### Layer 2: Research
- **research-creative** -- 4-step research process, code banks, personas, awareness levels, ICE scoring

### Layer 3: Strategy
- **strategy-meta-ads** -- CBO/ABO/ASC, Crazy Method, Blender Method, hot pockets, budget rules
- **strategy-google-ads** -- Search/Shopping/PMax/YouTube structure, bidding, Quality Score, full-funnel
- **strategy-tiktok-ads** -- TikTok creative principles, benchmarks, 3-week iteration loop
- **strategy-cross-channel** -- Multi-platform allocation, push vs pull, brand strategy, funnel mapping

### Layer 4: Create
- **create-ad-copy** -- Ad copy templates, headlines, static design, persuasion psychology
- **create-video-ads** -- Video frameworks (VSL, UGC, mashups), hooks, retention, scripting
- **create-ai-ads** -- Google Flow/Veo, HeyGen, ElevenLabs, CapCut, AI clones
- **create-google-assets** -- RSA headlines, Shopping titles, PMax assets, YouTube 7 elements
- **create-carousel-ads** -- 3 carousel types + hybrid, thumbnail AHQ framework

### Layer 5: Test
- **test-creative** -- 6 testing approaches, kill criteria, volume guidelines, iteration framework

### Layer 6: Optimize
- **optimize-meta** -- CBO optimization loop, diagnostics, creative fatigue, attribution
- **optimize-google** -- Shopping/Search/PMax optimization, POAS vs ROAS, bottleneck analysis

### Layer 7: Scale
- **scale-campaigns** -- Bell Theory, Daily Loop Theory, offer prerequisites, multi-country expansion

## Commands

| Command | Description |
|---|---|
| `/build-campaign` | Design a complete campaign from scratch |
| `/write-ad` | Generate ad copy, video scripts, or creative assets |
| `/research-brief` | Run the 4-step creative research methodology |
| `/audit-campaign` | Diagnose an existing campaign with frameworks |
| `/test-plan` | Design a creative or campaign testing plan |
| `/scale-plan` | Create a scaling roadmap from current performance |
| `/brief-media-buyer` | Generate a comprehensive SOP for team handoff |
| `/creative-iterate` | Produce iteration variations of winning ads |
