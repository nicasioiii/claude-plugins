# Ad Copy, Headlines & Static Creative

Write platform-agnostic ad copy and design static creative concepts. This skill covers Meta primary text, headlines, descriptions, static image ads, and all text-based creative elements.

## PREREQUISITE: ALWAYS RUN RESEARCH FIRST

Before writing any copy, ensure the user has completed creative research (research-creative skill). You need:
- Code bank with customer language
- Persona profiles with tangible/intangible desires and pain points
- Awareness level assessment for the target audience
- Competition breakdown showing what angles are already running

If these are missing, recommend running `/research-brief` first.

---

## AD COPY IMPACT HIERARCHY

Ad copy has 15-30% impact on total ad performance. Allocate effort accordingly:

| Element | Impact | Priority |
|---------|--------|----------|
| Creative (image/video) | 50-70% | Highest -- visual stops the scroll |
| Body text (above "See More") | ~20% | First sentence is most critical |
| Headline (below creative) | ~10% | Reviving dead creatives with new headlines works |
| Description | ~5% | Supports but rarely drives performance |

**The first sentence before "See More" is the single most important copy element.** It must earn the click to expand.

## COPY TEMPLATES (4 PROVEN FORMATS)

### Template 1: Long-Form Copy (Advertorial Style)
- First-person perspective, story-driven
- 300-500 words, reads like a personal account
- Best for: Info products, high-consideration purchases, problem-aware audiences
- Structure: Personal story hook > Problem identification > Failed alternatives > Discovery moment > Product introduction > Benefits > Social proof > CTA with guarantee

### Template 2: Bullet Copy (CLZ Format)
- Hook headline > Selling points as bullets > Testimonial quote > CTA > Scarcity element
- 100-200 words, scannable
- Best for: E-commerce, visual products, solution-aware audiences
- Each bullet starts with a benefit, not a feature

### Template 3: Short-Form Copy
- Headline > 2-3 sentence body > 3-4 bullet points > CTA > Link
- Under 100 words
- Best for: Retargeting, most-aware audiences, sale/promo campaigns
- Include the link in body copy -- older users cannot find the CTA button

### Template 4: Testimonial Ad Copy
- Customer story format, written as if from a real buyer
- Quotation marks around entire copy block
- Best for: Social proof angles, product-aware audiences
- Pull exact language from code bank reviews

## COPY WRITING RULES (NON-NEGOTIABLE)

1. **5th grade reading level** -- people in dopamine-scrolling mode need simple language
2. **Avoid AI-sounding words:** "transform," "revolutionize," "award-winning," "unlock," "elevate," "game-changer"
3. **Use emojis in headlines** for attention (sparingly in body)
4. **CAPITALIZE key words** in headlines for pattern interrupt
5. **Always include the link** in body copy text
6. **One message per ad** -- trying to communicate multiple benefits dilutes impact
7. **Write the first line and headline manually** -- AI can fill the framework, but the hook must be human-crafted
8. **Match copy to awareness level** -- cold audiences need problem-first copy; warm audiences need offer-first copy

---

## SEASONAL/SALE COPY APPROACH

**Offer-centric products** (known brands, commodity items): Lead with the discount -- "60% OFF" in headline and first sentence.

**Problem-solution products** (supplements, info products, tools): Keep evergreen problem/solution copy. ADD discount as a secondary element in brackets: "[50% OFF TODAY] Still struggling with back pain after 8 hours at your desk?"

Never rewrite entire copy for sales. Add discount brackets to first sentence and headline of proven copy.

---

## AD COPY TESTING PROTOCOL

- Test ad copy **once per quarter** (if spending under $10K/day)
- Test more frequently above $10K/day
- Process: Find best headline first (keep it), then test body copy variations
- **Never test headline AND body simultaneously** -- one variable at a time
- Test 10 different ad copies per quarter, find one evergreen winner, use it across all creatives
- Setup: One creative per ad set, duplicate ad set with same audience + creative but different copy only
- 3-4 copy variations per test is optimal

---

## REVIVING DEAD CREATIVES WITH HEADLINES

Once per quarter: take ALL best-performing creatives that stopped working. Change ONLY the bottom headline. This alone can completely revive a creative. Massive ROI for minimal effort.

Headline revival approaches:
- Summarize claims from the video/image differently
- Add new claims or reasons to believe
- Remove friction by addressing common concerns
- Add urgency, scarcity, or exclusivity language

---

## STATIC CREATIVE ITERATION PRIORITY

When iterating static ads, change elements in this order (highest to lowest impact):

1. **Headline** -- always iterate first, highest impact
2. **Body text / testimonial quote** -- second priority
3. **Color scheme** -- surprisingly impactful, test it
4. **CTA button text** -- lowest impact for statics

## THE 10 AD ANGLES

Every ad must choose ONE primary angle:

| # | Angle | When to Use |
|---|-------|-------------|
| 1 | Promotion | Sale periods, price-sensitive audiences |
| 2 | Education | Complex products, unaware/problem-aware audiences |
| 3 | Social Proof | Competitive markets, product-aware audiences |
| 4 | Influencer | Aspirational products, younger demographics |
| 5 | Gifting | Seasonal (holidays, Valentine's), relationship products |
| 6 | New Release | Product launches, novelty-seeking audiences |
| 7 | Emotional Benefit | Lifestyle products, identity-driven purchases |
| 8 | Functional Benefit | Utility products, convenience-seeking audiences |
| 9 | Comparison | Competitive markets, solution-aware audiences |
| 10 | Seasonality | Time-bound relevance, weather/event-driven products |
