# Copy Templates, Testing Protocol & Headline Revival

## LONG-FORM COPY TEMPLATE (ADVERTORIAL STYLE)

**When to use:** Info products, supplements, high-consideration purchases, problem-aware audiences. Advertorials work better for info products; listicles work better for e-commerce.

**Structure (300-500 words):**

```
[PERSONAL HOOK -- first-person story, 1-2 sentences]
I used to [specific problem the reader relates to]. Every [time period], I'd [vivid description of the struggle].

[FAILED ALTERNATIVES -- 2-3 sentences]
I tried [common solution 1], [common solution 2], even [extreme measure]. Nothing worked because [reason -- ties to root cause].

[DISCOVERY MOMENT -- 1-2 sentences]
Then my [friend/doctor/trainer] told me about [product category, NOT product name yet].

[EDUCATION -- 2-3 sentences about the mechanism/root cause]
Turns out, [surprising fact about the real problem]. Most [common solutions] only address [symptom], not [root cause].

[PRODUCT INTRODUCTION -- name the product]
That's when I found [Product Name]. It [unique mechanism that addresses root cause].

[BENEFITS -- 3-5 bullet points]
- [Benefit 1 using customer language from code bank]
- [Benefit 2 with specific number/timeframe]
- [Benefit 3 tied to intangible desire]

[SOCIAL PROOF -- 1-2 sentences]
[Number] people have already [outcome]. [Specific testimonial quote from code bank].

[CTA WITH GUARANTEE]
Try [Product Name] today risk-free with our [guarantee type].
[Direct link to product page]
```

**Pre-test headlines on static creatives BEFORE building the advertorial.** The best-performing headline on a static = the theme for your advertorial.

## BULLET COPY TEMPLATE (CLZ FORMAT)

**When to use:** E-commerce, visual products, solution-aware and product-aware audiences. Scannable format for scroll-happy users.

```
[HOOK HEADLINE -- benefit or curiosity-driven, CAPS + emoji]
[Hook sentence that creates urgency or curiosity]

[SELLING POINTS -- 3-5 benefit-first bullets]
[Emoji] [Benefit 1] -- [brief supporting detail]
[Emoji] [Benefit 2] -- [brief supporting detail]
[Emoji] [Benefit 3] -- [brief supporting detail]

[TESTIMONIAL QUOTE]
"[Exact customer language from code bank]" -- [Name], verified buyer

[CTA]
[Action verb] yours today [urgency element]
[Direct link]

[SCARCITY ELEMENT]
[Limited stock / sale ending / seasonal urgency]
```

---

## SHORT-FORM COPY TEMPLATE

**When to use:** Retargeting, most-aware audiences, sale/promo campaigns. Under 100 words.

```
[HEADLINE -- direct benefit or offer]

[2-3 sentence body connecting problem to solution]

[Emoji] [Bullet 1]
[Emoji] [Bullet 2]
[Emoji] [Bullet 3]

[CTA with link]
[Link URL -- always include in body for older/less tech-savvy users]
```

---

## TESTIMONIAL AD COPY TEMPLATE

**When to use:** Social proof angles, product-aware audiences, when code bank has strong review language.

```
"[Full testimonial in customer's own words, pulled from code bank. Keep the specific details -- numbers, timeframes, emotional reactions. 3-6 sentences.]"

-- [First Name], [Location or context that adds credibility]

[Product Name] -- [one-line value prop]
[CTA + link]
```

## ADVERTORIAL vs. LISTICLE DECISION FRAMEWORK

| Factor | Advertorial | Listicle |
|--------|-------------|----------|
| Product type | Info products, services | E-commerce, physical products |
| Copy style | Personal POV, story-driven | Bullet-pointed, scannable |
| Audience | Problem-aware, unaware | Solution-aware, product-aware |
| Length | 400-800 words | 200-400 words |
| Landing page | Long-form sales page | Product page or short LP |

Always test both formats. Pre-test the headline on static creatives first.

## AD COPY TESTING PROTOCOL (QUARTERLY)

### Step 1: Headline Testing
- Create 5-8 headline variations for your best-performing creative
- Test on static ads first (cheaper, faster data)
- Run for 3-5 days minimum
- Winner = highest CTR + conversion rate combination

### Step 2: Body Copy Testing
- Keep the winning headline locked
- Create 3-4 body copy variations using different templates
- Same audience, same creative, same landing page -- only copy changes
- One creative per ad set for macro-level analysis

### Step 3: Consolidation
- Take the winning headline + winning body copy
- Apply to ALL creatives as your evergreen copy
- This single copy serves as your default for 1 quarter

### Frequency by Spend
- Under $10K/day: Test once per quarter
- $10K-$50K/day: Test once per month
- Above $50K/day: Continuous testing with dedicated copy testing budget

## HEADLINE REVIVAL METHOD (QUARTERLY)

### Process
1. Pull ALL creatives that previously performed well but have stopped (creative fatigue)
2. Change ONLY the bottom headline (the headline element below the creative in Meta)
3. Re-launch in testing campaign
4. Expected revival rate: 20-40% of dead creatives come back to life

### Headline Revival Approaches
- **Summarize differently:** Same claim, new wording
- **Add new claims:** Introduce a benefit not in the original headline
- **Address concerns:** Turn an objection into the headline ("Yes, it works even if you've tried everything")
- **Add urgency:** Time-limited language ("Last chance" / "Ending tonight")
- **Add scarcity:** Stock/quantity language ("Only 147 left")
- **Add exclusivity:** ("Not for everyone -- only for [specific persona]")

### Dynamic Headline Testing Matrix
For each dead creative, create 4 headline variations:
1. Benefit-focused headline
2. Problem-focused headline
3. Social proof headline (number of customers/reviews)
4. Urgency/scarcity headline

## META AD COPY SPECS

| Element | Character Limit | Best Practice |
|---------|----------------|---------------|
| Primary text | 125 chars visible (more behind "See More") | First sentence is everything |
| Headline | 255 chars max | Keep under 40 chars for mobile |
| Description | 125 chars | Supports headline, rarely seen |
| Destination URL | N/A | Always include in primary text too |

---

## COPY QUALITY CHECKLIST (PRE-LAUNCH)

Before launching any ad copy:

1. First sentence creates curiosity or identifies a specific pain point
2. Written at 5th grade reading level (use Hemingway App to check)
3. No AI-sounding words (transform, revolutionize, unlock, elevate)
4. Link included in body text
5. Only ONE core message per ad
6. CTA is specific ("Shop the collection" not "Learn more") unless driving to advertorial
7. Headline uses CAPS and/or emojis for pattern interrupt
8. Copy matches the awareness level of the target audience
9. Customer language from code bank is used (not invented marketing speak)
10. Guarantee or risk-removal element present
