# Google Ads Creative & Asset Production

Create assets specifically for Google campaigns: Search RSAs, Shopping titles, PMax asset groups, YouTube scripts, and Display/Discovery ads.

## RSA HEADLINE FRAMEWORK (THE 7 TYPES)

Every compelling RSA is built from 7 headline types. Each headline has a 30-character maximum.

| # | Type | Formula | Example |
|---|------|---------|---------|
| 1 | Relevance Anchor | Signal match to query (DKI: {KeyWord:Fallback}) | "{KeyWord:Home Security}" |
| 2 | Problem/Pain Recognition | [Tired of/Stop/End] + [Problem] | "Stop Losing Sleep Tonight" |
| 3 | Value Proposition | [Specific Offer] + [Primary Benefit] | "Custom Meal Plans in 5 Min" |
| 4 | CTA | [Action Verb] + [Specific Outcome] | "Get Your Free Trial Now" |
| 5 | USP/Benefit | [Differentiator] + [Quantified Benefit] | "30% Faster Than Competitors" |
| 6 | Social Proof & Authority | [Metric] + [Audience/Context] | "Trusted by 50,000+ Users" |
| 7 | Risk Removal | [Guarantee Type] + [Reassurance] | "Full Refund, No Questions" |

**Urgency** is a MODIFIER, not a core type. Apply to any headline sparingly.

---

## RSA STRUCTURE BEST PRACTICES

- **6-8 headlines per RSA (NOT 15)** -- more headlines = more combinations = data poverty
- 15-headline RSA = up to 2,730 three-headline combinations. Most ad groups never get enough impressions for Google to learn.
- All 7 headline types represented across 6-8 slots
- 25-30 characters per headline
- Headlines must stand alone AND complement each other
- Mix problem AND solution language
- 2-3 descriptions per RSA
- One RSA per ad group (no competing RSAs splitting traffic)

---

## THE 6 MESSAGE ANGLES (OFFER ANGLE EXTRACTION)

Before writing any headlines, extract raw material from these 6 angles:

1. **Problem/Pain Points** -- Specific frustrations, exact language customers use, cost of NOT solving
2. **Value Proposition** -- Main benefit (not just features), clear in 5 seconds
3. **USPs** -- Actually unique (competitors cannot claim), provable
4. **Value Boosters** -- Everything included, specific quantities
5. **Social Proof & Authority** -- Numbers (not "trusted by many"), recognizable names
6. **Risk Removal** -- Specific guarantee/trial terms, addresses main hesitation

## RSA ITERATION LOOP (4 PILLARS)

### Pillar 1: Templatize
- Group ad groups into clusters with similar intent
- Create RSA template assigning each headline slot a specific type
- H1 = keyword slot (changes per ad group), all other slots identical across cluster
- Example: 20 ad groups x 5,000 impressions = 100,000 aggregated impressions for one angle

### Pillar 2: Aggregate
- Pull asset-level performance data across cluster
- Calculate CPI (Conversions Per Impression), RPI (Revenue Per Impression)
- Do NOT use CTR as primary metric (high CTR can attract unqualified clicks)
- Do NOT use Google's "Low/Good/Best" labels (CTR-driven, not outcome-driven)

### Pillar 3: Diagnose (Performance Quadrants)
| | High Impression Share | Low Impression Share |
|---|---|---|
| **High CPI/RPI** | Winners (scale) | Hidden Gems (increase exposure) |
| **Low CPI/RPI** | Silent Killers (remove FIRST!) | Underperformers (replace) |

**Silent Killers are the priority** -- actively hurting performance because Google shows them frequently.

### Pillar 4: Iterate
- Phase 1 (Foundation): Test headline types against each other
- Phase 2 (Growth): Test sub-angles within winning types
- Phase 3 (Mastery): Test specific asset variations within winning sub-angles

## YOUTUBE AD SCRIPTS -- 7 MUST-HAVE ELEMENTS

YouTube ads are fundamentally different from social ads. More similar to short-form infomercials (1.5-3 minutes optimal).

1. **The One Thing (Golden Thread)** -- Single core message woven throughout. After watching, ONE thing they remember.
2. **The Hook (First 5 Seconds)** -- Skip button appears at 5 sec. Goal: get the RIGHT audience to watch more.
3. **Product Demo** -- Show product in natural environment, the way someone actually uses it.
4. **Magical Demo (Optional)** -- Goes above and beyond. Creates "I've never seen anything like this" moment.
5. **Objection Busters** -- Address the "yeah, but..." Price, durability, effectiveness, past bad experiences.
6. **Social Proof** -- Reviews, review counts, UGC clips, expert endorsements.
7. **Offer / CTA** -- If you don't tell them what to do, they will do nothing. Present: offer + next steps.

**Video length sweet spot: 1.5-3 minutes.** Below 30 sec = poor conversions. Above 3 min = higher CPM.

**Winning YouTube ads can run for months or YEARS** without burning out (audiences refresh regularly, unlike social).
