# Meta Crazy Method - Complete Framework

## Overview
- **What it is:** CBO campaign with identical ads duplicated 30-50 times to find multiple "hot pockets"
- **When to use:** Proven asset + trained pixel + need to scale or find additional audiences
- **NOT for testing:** This is a SCALING method for already-working creatives
- **Risk level:** Medium (amplifies results: if good → better; if bad → worse)
- **Budget:** $100-1,000/day depending on asset quantity and CPA

---

## CORE PRINCIPLE: HOT POCKETS THEORY

**The Problem:** Facebook's algorithm learns from initial randomness. Different send distributions create different "hot pockets" within the same audience.

**Real Data:**
- Account spent $1.7M across full year
- Target audience: 150M+ potential (broad targeting)
- Actual reach: Only 20M unique people in 30 days
- Frequency: 8.5 (same 20M people saw ads 8-9 times each)
- **Conclusion:** Meta doesn't distribute evenly across all 150M. It concentrates spend in pockets that convert.

**The Insight:** If one audience has multiple hot pockets, running the same ad 30-50x creates multiple independent chances to find different pockets.

**Why It Works:**
- Each duplicate gets initial random seed (different starting distribution)
- Different seeds find different hot pockets within same 20M people
- Cumulative result: Access same 20M people through different angle each time
- One pocket might hit women 40-55; another hits women 25-35 in same audience

## WHEN TO USE CRAZY METHOD

### DO USE if:
✓ Asset has proven ROAS 1.5+ in previous campaign
✓ You have 50+ assets OR willing to create 30-50 variations
✓ Pixel is trained (you've run 100+ conversions through it)
✓ You want to find new scaling angles for proven creatives
✓ You need to reduce daily randomness (multiple assets = stability)
✓ Budget is $100+ per day (need enough to support multiple assets)

### DON'T USE if:
✗ Asset is brand new (unproven)
✗ Pixel has < 100 conversions (not trained enough)
✗ Only have 5-10 assets total (need 30+)
✗ Want to test creative (wrong tool; use A/B test instead)
✗ Running 6+ Crazy Method campaigns with same creative (cannibalization)
✗ Budget < $100/day (too small to support multiple assets)

## ASSET CALCULATION FORMULA

**How many duplicates do you need?**

```
Number of Assets = Daily Spend ÷ Target Cost Per Purchase

Example 1:
- Daily Budget: $100
- Target CPA: $30
- Assets needed: 100 ÷ 30 = 3.3 → Use 3-4 assets

Example 2:
- Daily Budget: $300
- Target CPA: $50
- Assets needed: 300 ÷ 50 = 6 → Use 6-7 assets

Example 3:
- Daily Budget: $500
- Target CPA: $30
- Assets needed: 500 ÷ 30 = 16.7 → Use 16-17 assets
- Rule: If over 10, divide by 2 → 17 ÷ 2 = 8-9 assets maximum
```

**Rule of Thumb:** Each asset should see 1-3 sales per day on average. If you expect 10 sales/day total, run 5-10 assets (2 sales per asset).

## CAMPAIGN SETUP (STEP-BY-STEP)

### Campaign Level
1. **Campaign Name:** "[Crazy Method] [Asset Name] - [Budget] - [Date]"
   - Example: "[Crazy Method] Winter Coat Video 1 - $300 - 03-06-2026"
2. **Campaign Objective:** Conversions (or Catalog Sales if using DPA)
3. **Campaign Budget Optimization:** ENABLED (critical; don't use ABO)
4. **Daily Budget:** Total daily spend (Example: $300/day)
5. **Bid Strategy:** Automatic
6. **Attribution Window:** 7-day click (or 1-day if testing volatility)

### Ad Set Level Strategy

**Option A: Single Ad Set with All Duplicates (Simpler)**
1. **Ad Set Name:** "All Assets - Open Targeting"
2. **Audience:**
   - Location: Your primary country
   - Age: Match original winning campaign (Example: 25-55)
   - Gender: Match winning campaign
   - Detailed Targeting: NONE (open targeting)
   - Exclusions: None
3. **Placements:** All enabled
4. **Optimization:** Conversions
5. **Create Ads:** Add all 30-50 assets under this single ad set
6. **Structure:** CBO will distribute budget across all assets automatically

**Option B: Multiple Ad Sets (More Control)**
1. Create 3-4 ad sets, each with slightly different demographic narrow
   - Ad Set 1: Ages 25-40
   - Ad Set 2: Ages 40-55
   - Ad Set 3: Ages 55-65+
2. Distribute 10-15 assets evenly across each
3. Same ad can appear in multiple ad sets
4. CBO allocates budget to best-performing sets

**Recommendation:** Start with Option A (simpler). Switch to Option B only if you see dramatic demographic differences in winning assets.

### Creative Requirements

**Critical Rule:** All assets MUST be IDENTICAL except for:
1. Slight demographic narrowing (if testing)
2. Minor copy variation (headline only, same visual/video)

**Examples of SAME:**
- Same video, same copy, same audience ✓
- Same image, copy varies (headline only), same audience ✓
- Same video, different headline, same interest ✓

**Examples of DIFFERENT (Don't do):**
- Same creative, different interests ✗ (use separate campaign)
- Same video, completely different copy ✗ (muddles the test)
- Different videos in same CBO ✗ (confuses which asset won)

**Asset Pool Strategy:**
- If you have 50 proven assets from previous campaigns, use best 15-20
- If you have 5-10 assets, duplicate them (same creative, listed multiple times)
- Facebook will treat each duplicate as separate (forces different learning path)
- Expected outcome: 30-50 assets, only 5-7 profitable; rest fail (normal)

## HOT POCKETS IDENTIFICATION

### How to spot when Crazy Method is working:

**Metric 1: Unequal Asset Performance**
- Asset 1: $0 spend (no sales)
- Asset 2: $1,200 spend, 40 sales
- Asset 3: $45 spend, 1 sale
- Asset 4: $800 spend, 35 sales
- **Pattern:** Wildly unequal spend = hot pockets found ✓

**Metric 2: Frequency Concentration**
- Total reach: 500K people
- Total frequency: 15
- This means only ~500K/15 = 33K people seeing ads; seen 15 times each
- **Pattern:** Same small audience, high frequency = concentrated hot pocket ✓

**Metric 3: Asset Diversity in Winners**
- 3 videos work (different hooks)
- 1 image works (different USP)
- Winners have different themes but same product
- **Pattern:** Diversity in winning assets = multiple distinct hot pockets found ✓

## SCALING WITH CRAZY METHOD

### When the Campaign is Winning (ROAS 1.5+):

**DO NOT scale the entire budget.** Instead:

**Strategy 1: Duplicate the Campaign (Recommended)**
```
Campaign A: Crazy Method - Run 1 - $300/day
Campaign B: Crazy Method - Run 2 - $300/day (duplicate of A, but fresh start)
Campaign C: Crazy Method - Run 3 - $300/day (if still scaling)

Result: Three $300 campaigns = $900/day
Each gets independent Learning Phase
Each finds independent hot pockets
More stable than one $900 campaign
```

**Why this works:** Each campaign independently optimizes. If one hits saturation, others still scaling. Diversification of risk.

**Strategy 2: Add New Assets to Existing Campaign**
- Campaign A running with 30 assets
- Add 15-20 new assets to same campaign
- New assets get fresh start
- Existing winning assets continue
- Expected: Some new assets fail, some become new winners

**Don't:** Scale the entire campaign budget 3x. That usually triggers:
- Re-entry into Learning Phase
- Algorithm confusion (too many assets at once)
- Performance drop

### When the Campaign is Struggling (ROAS < 1.2):

**Kill it.** Don't scale losing assets.

**Diagnosis:** If Crazy Method with proven asset still loses:
1. Account quality issue (new account with poor traffic)
2. Audience wrong (wrong age/gender from original)
3. Pixel no longer trained (if account was dormant)
4. Creative fatigue (asset worked once, no longer converts)

**Next step:** Don't run Crazy Method again until you fix the underlying issue.

## CANNIBALIZATION RISK (CRITICAL WARNING)

### What is Cannibalization?
When multiple campaigns with same creative fight over same hot pockets. Result: All campaigns underperform.

**Example:**
- Campaign A: Crazy Method, Winter Coat Video 1, $300/day ROAS 2.1
- Campaign B: Crazy Method, Winter Coat Video 1, $300/day ROAS 2.1
- Campaign C: Crazy Method, Winter Coat Video 1, $300/day ROAS 2.1
- **Together:** All three running simultaneously = ROAS drops to 1.3 (cannibalization)

### Prevention Rules:

**MAX 3 Crazy Method campaigns simultaneously with same creative**
- 1 original: ROAS 2.1
- 1 duplicate: ROAS 1.9 (slightly lower due to some overlap)
- 1 more duplicate: ROAS 1.5 (overlap increases)
- 4th duplicate: ROAS 1.0 (full cannibalization; all hurt)

**If you want to scale beyond 3 campaigns:**
- Use different creative angles, not same creative
- Campaign A: Hook angle (emotional appeal)
- Campaign B: Benefit angle (practical advantage)
- Campaign C: Social Proof angle (testimonials)
- Each uses different video/image but same product

### Daily Loop Theory & Cannibalization:
Multiple Crazy Method campaigns with same creative WILL cannibalize each other over time. This is why Daily Loop Theory matters:
- Monitor each campaign's ROAS daily
- Pause underperformers when they dip below 1.3
- Rotate assets through pause/unpause cycles
- Keep only best 2-3 active at any time

## DAILY MONITORING

**Every morning:**
- [ ] Check total account ROAS (should maintain 1.5+)
- [ ] Identify top 5 performing assets (which got most spend?)
- [ ] Check if any asset spent $0 (not getting picks; consider pausing)
- [ ] Check if budget fully spent
- [ ] Look for signs of cannibalization (all assets declining together)

**Every Friday:**
- [ ] Identify which assets are genuine winners (2+ sales, positive ROAS)
- [ ] Pause assets with $0 spend for 7+ days (they're not getting chosen)
- [ ] Analyze winner assets: What do they have in common? (Hook? Copy? Audience?)
- [ ] Decide: Scale via duplication? Add new assets? Or kill campaign?

**When to pause individual assets within campaign:**
- Asset has $0 spend for 3+ days (CBO avoiding it; low quality signal)
- Asset is spending but losing (CPA > 2x target despite others working)
- Asset had 3+ sales but all negative ROAS (quality issue)

**When to pause entire campaign:**
- Total campaign ROAS < 1.2 for 2+ days
- Spend declining (reaching audience limits)
- CPA rising steadily week-over-week

## CRAZY METHOD FOR LEAD GENERATION (SPECIAL USE CASE)

### Why Lead Gen Needs Crazy Method:

**The Problem with Small Geographies:**
- Local business targeting 50-mile radius = 2-5M potential audience
- Small audience = few daily leads = algorithm can't optimize
- Pure luck: First lead gets picked by chance

**The Solution: Crazy Method**
- Create 40-50 ad variations (same product, different hooks)
- Distribute across same small audience
- Each variation gets random initial seed (independent luck attempt)
- 40+ attempts > 1 attempt at finding the first lead
- Once first lead converts, algorithm optimizes other 39 variations toward similar people

**Expected Outcome:**
- Days 1-2: Random early leads (luck-based)
- Days 3-7: Algorithm identifies patterns; quality leads improve
- Day 7+: 15-20 assets performing, rest failing (normal distribution)

### Lead Gen Crazy Method Setup:
1. **Audience:** 50-mile radius, age/gender match
2. **Campaign Budget:** $50-100/day (supports 15-25 assets)
3. **Assets:** 40-50 ad variations (copy changes, same video)
4. **Optimization:** Leads (not conversions)
5. **Expected timeline:** 10-14 days to profitability (longer than e-com due to smaller audience)

## TROUBLESHOOTING

### Problem: All 50 assets getting equal spend
**Diagnosis:** CBO not discriminating; algorithm can't find performance differences
**Solution:**
1. Check ROAS on best vs worst assets (should be 3:1 difference)
2. If all exactly same ROAS, assets are too similar; increase variation
3. Or: Audience is genuinely too small; campaign is underfunded
4. Increase budget 20-30% to force algorithm to differentiate

### Problem: Only 2 assets profitable, 48 failing
**Diagnosis:** Normal. This is expected 90% of the time.
**Solution:** This is not a problem; it's working as designed.
- The 2 profitable assets are finding hot pockets
- The 48 failing assets help by preventing cannibalization (diluting bid pressure)
- Continue running; don't kill the campaign if total ROAS is positive

### Problem: Was ROAS 2.0 Week 1, now 1.2 Week 2
**Diagnosis:** Two possible causes:
1. Audience saturation (hot pockets filled up; creative fatigue)
2. Cannibalization from other campaigns running same asset

**Solution:**
1. Check if running other Crazy Method campaigns with same asset; pause them
2. If isolated, audience likely saturated; duplicate campaign instead of scaling
3. Introduce new asset variations to find new hot pockets
4. Or kill and start fresh with different creative

### Problem: No sales Day 1-2
**Diagnosis:** Either pixel issue or very tight budget
**Solution:**
1. Verify pixel is firing (use Pixel Helper extension)
2. Check if budget is sufficient for 30+ assets (might need $200+ per day)
3. Give it to Day 3; Crazy Method needs more time than single-asset campaign
4. By Day 3 you should see sales; if not, pixel issue likely

|--------|---------------|--------------|

## SUCCESS INDICATORS

✓ Campaign working if:
- 5-10 assets generating 80%+ of spend (concentration = hot pockets)
- Overall ROAS 1.5+ maintained for 2+ weeks
- CPA stable or declining
- New assets occasionally breaking through as winners

✗ Campaign should be killed if:
- All 50 assets performing equally (no winners identified)
- Total ROAS < 1.2 after 14 days
- Budget being wasted on 48 loser assets while 2 winners carry campaign
- Pixel not recording (data shows 0 conversions)

## KEY TAKEAWAYS

1. **Crazy Method is scaling, not testing** — only use proven creatives
2. **One audience, many assets** = finding hot pockets in concentrated way
3. **Asset calculation formula** prevents under/overfunding assets
4. **Duplication > scaling** = run multiple campaigns instead of mega-budget
5. **Expect 90% failure** of assets; 5-7 wins = success
6. **Cannibalization is real** = max 3 campaigns with same creative
7. **Hot pockets theory** = small population (20M) sees ads 8+ times (concentrated)
8. **Monitor unequal asset spend** = sign that hot pockets are being found
