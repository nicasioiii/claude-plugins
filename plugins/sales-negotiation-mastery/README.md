## Workflow Layers

**Pre-Call:** sales-mindset-prep --> presence-power --> discovery-qualification (prep questions)

**The Sales Call:**
```
appointment-setting (if setter call)
  --> discovery-qualification (80% of call time)
      + tactical-empathy (woven throughout)
      + tonality-delivery (woven throughout)
  --> pitch-presentation (transition + offer)
  --> closing-techniques (temp check + investment drop)
  --> objection-handling (if objections arise)
  --> negotiation-tactics (if pricing/terms negotiation)
  --> closing-techniques (final commitment)
```

**Post-Call:** pipeline-followup --> appointment-setting (re-book if needed)

**Always-On:** sales-psychology, tonality-delivery, tactical-empathy, sales-mindset-prep
