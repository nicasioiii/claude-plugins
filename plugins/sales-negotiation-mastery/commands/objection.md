# /objection Command

## Purpose

Help the user diagnose, isolate, and resolve a specific objection they are facing.

## Routing

**Primary skill:** `objection-handling`
**Supporting skills:**
- `tactical-empathy` (labeling the emotion, accusation audits, mirroring)
- `sales-psychology` (consistency bias, loss aversion framing)
- `tonality-delivery` (voice control during objection handling)

## Workflow

1. **Get the exact objection.** What did the prospect say, word for word? What was the context? Where in the call did it happen?

2. **Classify it.** Use the taxonomy: Smokescreen, Uncertainty/Fear, Money, Partner/Spouse, Burned in Past, Time, or Logistical?

3. **Check for smokescreen.** "I need to think about it" is NEVER real. Apply the three isolation questions and the Money Aside test.

4. **Check for process breakdown.** 90% of objection problems are actually discovery, transition, or pitch problems (Gordon). Was the Belief Ladder completed? Was the gap built? Did they pre-handle the mental block?

5. **Apply the handling framework.**
   - Default: Gordon's Pacing Method (tie-down #1 --> tie-down #2 --> resolve)
   - Then: ICEL-S framework (Isolate, Confirm, Empathize, Loop, Solve)
   - Label the emotion first (Voss) before any resolution technique

6. **Script the response.** Give exact talk-track language tailored to their specific situation, product, and prospect. Reference `objection-handling/references/objection-scripts.md` for templates.

7. **Recommend risk mitigators if needed.** Payment plans, deposit close, 24-hour refund, 7-day refund, ROI guarantee. Reference `objection-handling/references/risk-mitigators.md`.

8. **Address the root cause.** If this objection recurs across calls, the issue is upstream. Recommend specific discovery or pitch improvements.

## Key Decision Points

- If objection is about **price/terms negotiation** (not fear or belief), route to `negotiation-tactics`
- If objection reveals **incomplete discovery**, recommend looping back to `discovery-qualification`
- If prospect is **genuinely unclosable** (no resources, no need), recommend graceful disqualification
- If dealing with **spouse objection**, use the respect vs. permission distinction and Two Guys reframe
