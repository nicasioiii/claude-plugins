# Closing Scripts: Word-for-Word Language

## Soft/Assumptive Close Scripts

### Script 1: Basic Soft Close
```
"Okay, so here's what we can do. We get you on a deeper dive call into your specific situation.
If we think we can help you, we get you started and roll into everything.
Does that sound good?"
```

### Script 2: Service Introduction Soft Close
```
"What we'd recommend is getting you set up with one-on-one coaching,
where you'll have someone who's already done this walking you through it.
Does that make sense?"
```

### Script 3: Process Overview Close
```
"Here's how it works: First, we map out your current situation.
Then we show you exactly what needs to change.
Then we walk you through the process step-by-step.
Does that sound like something you'd want to do?"
```

### Script 4: Timeline-Based Assumptive
```
"Perfect. So when would work better for your first call, morning or afternoon?"
[They answer]
"Tuesday or Wednesday?"
[They answer]
"Done. [Day] at [time]. I'm sending you the login details now."
```

## Financial Qualification Scripts

### Script 1: Direct Financial Qualification
```
"Our program does come with an initial investment of $3K.
Is that something that would make sense for you financially right now?"
```

### Script 2: Probing Deeper (If Wishy-Washy)
```
"I mean, do you have like $1,500 to $3K right now?
To the best of your knowledge?"
```

### Script 3: Confirmation (If They Said Yes Once)
```
"Just to triple-check, you do have the capital between $1,500-$3K to invest, correct?
Because I don't want to get you on a 30-minute call if this isn't realistic."
```

### Script 4: Payment Plan Introduction (After Resistance)
```
"The full investment is $3K, but we do have payment options.
We can break it into three payments of $1K each.
If we were able to do that, would that work for you?"
```

### Script 5: Capital Confirmation (When They Mention Savings)
```
"Okay, but in terms of liquid capital available—do you actually have $1,500-$3K
that you could invest? Even if you're spending more day-to-day, do you have savings?"
```

## Deposit Close Scripts

### Script 1: Lock-In Close
```
"I love your energy here. To lock this in and make sure we get you scheduled,
let's put down a deposit of $500 today. This holds your spot and shows you're serious."
```

### Script 2: Responding to "Why a Deposit?"
```
"It ensures you'll show up. I don't want to waste your time or mine.
Plus it shows me you're genuinely committed to this.
And honestly, people who invest upfront get better results."
```

### Script 3: Addressing Refund Concerns
```
"What if you change your mind? No problem. 30-day money-back guarantee
if we're not a fit. But this deposit shows you're serious."
```

## Guarantee/Risk Reversal Scripts

### Script 1: ROI Guarantee Close
```
"Here's what I'm going to promise you:
If you don't see results in the first 90 days,
we set up additional one-on-one calls to identify exactly why.
And we figure it out together. No questions asked."
```

### Script 2: Commitment Lock
```
"I don't want you to invest if you're not willing to commit time and go through the process.
But if you do commit, we guarantee we'll get you results.
And if we don't, we double down and figure out what's missing."
```

### Script 3: Confidence Guarantee
```
"Look, we've done this with hundreds of people.
We know the system works.
If it's not working for you specifically, there's a reason,
and we're going to find it and fix it."
```

### Script 4: Simple Money-Back
```
"Worst case scenario: you start, you crush it in the first week,
but then realize it's not for you. You pause your membership,
pay it off, and move on. It's not like you're going to lose everything."
```

## Nurture Close Scripts

### Script 1: Full Nurture Close
```
"You know what, this probably isn't something that makes sense for you right now.
Let's book you on a call when it DOES make sense.

What I'll do in the meantime—I will send you testimonials,
I'll send you case studies, I'll send you whatever value I can.
So you can keep this top of mind.

And whenever you're ready to pull the trigger on this,
we can get you on that call, and then we can get you rock and rolling.
Is that sound fair?"
```

### Script 2: Short Nurture (Time Not Right)
```
"This probably isn't the right time for you.
But let's stay connected. I'll reach out in a month with some resources.
Sound good?"
```

### Script 3: Nurture with Specific Timeline
```
"Let's revisit this in 30 days when you've had more time to think.
In the meantime, I'm sending you three case studies that show exactly how this works.
I'll check in with you on [specific date]."
```

## Handling Common Objections (Pre-Close)

### Objection: "I Need to Think About It"

**Response 1**:
```
"I understand. But you did sign up for this call and watch that webinar,
so something interested you. What was that?
And what would need to change for you to feel confident moving forward?"
```

**Response 2**:
```
"What specifically do you need to think about? Is it the investment?
Is it the time commitment? Let's talk through it right now
so you're not sitting at home wondering."
```

### Objection: "Money Aside" Reframe

```
"Let me ask you this. Money aside, set that away for a second.
When it comes to the program itself, is there anything else
that would be keeping you from jumping in and getting started?"
```

### Objection: "But I Want Multiple Streams of Income"

```
"Yeah, I'm not saying you can't have multiple sources of income.
But doesn't it make sense to go ahead and get to your goal with ONE thing first,
and then worry about adding income streams?

It's kind of like, 'I want to get really good at basketball
AND play football on the side.'
You're splitting your focus. Why not master one first?"
```

### Objection: "I Think I Can Do This Myself"

```
"Yeah, you might be able to. But here's the thing—
we have coaches who've already done this and made it work.
Rather than you spending 6-12 months figuring it out on your own,
they can show you exactly what works in 30 days.
Does that make sense?"
```

### Objection: "Why Should I Choose You?"

(From Luke Alexander—control frame):
```
"I'm not even sure if we can help you yet. That's what this call is for.
I'm just trying to figure out if we're a good fit for each other."
```

## Advanced: Temperature Check + Price Close Combination

### Step 1: Temperature Check (Before Mentioning Price)
```
"Let me ask you something. On a scale of 1 to 10,
where 10 is 'this is exactly what I need right now,'
how does this feel?"
```

### Step 2: Wait for High Number (8-10)
Prospect: "I'd say a 9. This is definitely what I need."

### Step 3: Introduce Price
```
"Perfect. So this is typically a $3K investment.
How does that land for you?"
```

### Step 4: If They Drop Below Their Original Rating
```
"I hear you on the investment. But here's the thing—
just 30 seconds ago you said this was a 9 out of 10,
exactly what you need. The investment hasn't changed what this actually does for you.
What's really coming up for you?"
```

## Challenge Scripts (For Low Commitment)

### Script 1: Direct Challenge
```
"I'm going to be honest with you—sales is also going to be a lot of work.
Are you 100% committed to doing everything you need to be successful?
Yes or no?"
```

### Script 2: Follow-Up if Hesitant
```
"Are you SURE? Because I don't want to waste your time or our trainer's time
on a 45-minute call if you're not all in."
```

### Script 3: Commitment Separator
```
"People that want to make change are going to find time
to do the things that are important to them.
So is this actually important to you or not?"
```

## Reframes When Things Get Tense

### When They Say "This Is Too Much"
```
"What's 'too much'—the investment or the commitment?"
[Listen to answer]

"Here's what I know: You came to this call because something in your current situation
isn't working. This is the tool to change that. What's the real concern here?"
```

### When They Go Silent
```
"What's coming up for you right now? Talk to me."
[Listen. Don't fill silence.]

Then: "So what would need to change for this to feel right?"
```

### When Environment Is Against Them
```
"I know people around you might not understand this.
I get it—my parents were the same way.
But here's the reality: In 90 days you're going to look completely different
in terms of your income and your clarity.
Then they'll understand. We get it. We've seen this a hundred times."
```

## Black Swan Principle: The Rule of Three

(From Chris Voss/Never Split the Difference)

Ask the same question three different ways to get at the real answer:

### Example 1:
```
Q1: "Is price the main thing keeping you from moving forward?"
[Answer]

Q2: "So money aside, is there anything else?"
[Answer]

Q3: "What would need to change for you to feel like this is worth it?"
[Answer]
```

This reveals whether it's actually price or a fear hiding behind price.

### Example 2:
```
Q1: "Do you feel like you have the time to commit?"
[Answer]

Q2: "Realistically, could you find 10 hours a week for this?"
[Answer]

Q3: "What would your schedule look like if you were fully committed?"
[Answer]
```

The third answer usually reveals the truth.

## Getting to "That's Right" (Black Swan)

The goal is to get them to say "That's right" (not just "Yeah, I understand").

### Script:
```
"Let me just make sure I understand where you're at.
You've been wanting to make a change for a while.
You've tried a few things.
You're worried about the investment but you see the value.
You're ready to move but want to make sure it's the right timing.
Is that right?"

[Wait for "That's right"]

"Perfect. So the question is: When is the right timing?"
```

When they say "That's right," they've fully bought in. Now close.
