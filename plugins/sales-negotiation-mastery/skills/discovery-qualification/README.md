## Synthesis Approach

Rather than listing each instructor's approach separately, content is:
1. **Merged where experts agree** (e.g., all emphasize the Doctor Frame)
2. **Presented with both sides where they differ** (e.g., Cole's Problems-First vs Goals-First syntax)
3. **Flagged as contrarian** when insights challenge conventional wisdom
4. **Organized by practical use** rather than by instructor

## Key Principles Across All Content

- **80% of the close happens in discovery** - The pitch is just confirmation
- **Disqualify early** - "Not now" converts better than forcing a close
- **Specificity matters** - Vague problems don't create urgency
- **Emotion drives decisions** - Logic justifies what emotion decides
- **Authority matters** - Know who actually decides
- **Timeline is everything** - "Right now" vs "someday" determines if they move
