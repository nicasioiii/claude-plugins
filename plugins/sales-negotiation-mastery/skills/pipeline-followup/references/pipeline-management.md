# Pipeline Management: CRM, Metrics & Daily Systems

## CRM Hygiene (Non-Negotiable)

**Source:** Alexander. If it is not in the CRM, it did not happen. Many companies will not pay commission until data is entered.

### Minimum CRM Fields

**Contact Info:**
- Name, email, phone
- Company/industry (if B2B)
- Source (how they found you)

**Pipeline Fields:**
- Current stage (Hot / Warm / Cold / Dead)
- Last contact date + method
- Next action + date
- Follow-up sequence status (Touch 1/2/3.../8)
- Original objection (word-for-word)

**Deal Fields:**
- Deal amount / commission value
- Close probability (%)
- Expected close date
- Payment status (if partial/deposit)

**Communication Log:**
- Every call, text, email logged with date + notes
- Objections raised + how addressed
- Key phrases from discovery (ammo for follow-up)

## HotList System (Gordon Method)

### Hot List (Review DAILY)
**Criteria:** Last contact within 48 hours, actively engaged, closing this week.

**Actions:**
- Push for close -- address final objections
- Send contract/payment link
- 1-2 actions per deal per day

### Warm List (Review WEEKLY)
**Criteria:** Last contact within 7 days, in active conversation, closing this month.

**Actions:**
- Execute 8-touch follow-up system
- Send social proof, case studies
- Address secondary objections

### Cold List (Review MONTHLY)
**Criteria:** Last contact 14+ days ago, stalled or in dead list.

**Actions:**
- Run database reactivation campaign
- Send industry update or insight
- Deploy Dean Jackson 9-word email
- One touch per deal per month

## Weekly Pipeline Review Checklist (Friday, 30 Minutes)

### 1. Stage Audit (10 min)
- Are deals in the correct stage?
- Any deals stuck in same stage for 7+ days? Why?
- Move stale deals to dead list

### 2. Bottleneck Analysis (5 min)
- Most deals stuck in follow-up? Not enough touches.
- Most deals stuck post-call? Discovery/pitch is weak.
- Most deals in dead list? Offer/positioning problem.

### 3. Forecast (5 min)
- Which deals close THIS week?
- Which close THIS month?
- Am I on pace for my target?

### 4. Action Plan (10 min)
Write 3-5 specific actions for next week:
1. [Deal name]: [Action] by [Date]
2. [Deal name]: [Action] by [Date]
3. [Process fix]: [Action] by [Date]

## Dead List Management Protocol

**Source:** Alexander. Review every 2 weeks.

### Bi-Weekly Dead List Review
1. Pull all leads with no contact in 14+ days
2. Sort by: most recent contact date, original objection, deal size
3. Select top 10-20 leads for reactivation
4. Choose appropriate re-offer strategy for each
5. Execute outreach over 3-5 days (not all at once)
6. Log responses, move responders back to warm list

### Quarterly Full DR Campaign
1. Pull ENTIRE dead list
2. Segment by objection type
3. Craft segment-specific re-offer or value drop
4. Send batch outreach (personalized, not mass email)
5. Work responses through normal 8-touch system
6. Archive true dead leads (explicit no, competitor, 6+ months silent)

## VSL Funnel Math (Alexander)

Understanding the math of why end-to-end closing matters.

### Typical Funnel Numbers
```
1,000 landing page visitors
    -> 5% opt-in to VSL = 50 viewers
        -> 10% of viewers book = 5 calls
            -> 50% show rate = 2.5 actual calls
```

### Lazy Closer vs End-to-End Closer

| | Lazy Closer | End-to-End Closer |
|---|---|---|
| Touches | 2.5 calls only | All 50 VSL viewers + 5 booked |
| Revenue | ~$5K | ~$25K |
| Pipeline work | Zero | Daily |
| Commission from pipeline | 0% | 50% |

### The Offer Equation
```
Monthly Call Volume x Close Rate x Average Commission = Monthly Income
```

If the math does not work for $10K/month on your current offer, the answer is either:
1. More volume (more calls, more pipeline work)
2. Higher close rate (skill improvement)
3. Better offer (higher commission or price point)
4. New offer entirely

## Show Rate Improvement System

### The 6 Methods (Alexander)

**Method 1: Speed to Lead**
Call and confirm within 3-5 minutes of booking. Run a mini triage:
- Verify contact info
- Confirm they watched the VSL
- Set expectations for the call
- Lock commitment: "Nothing's going to come up, right?"

**Method 2: Address the 4 No-Show Reasons**
| Reason | Solution |
|--------|----------|
| Did not know what the call is about | Confirmation email with agenda |
| Did not know how it would help | Pre-call testimonial video |
| Forgot they had a call | 30-minute reminder text |
| Scared about the call | Pre-call email addressing fears |

**Method 3: Personalized Asset Matching**
If prospect mentioned being scammed before: send student interview of someone who overcame the same fear. Match testimonials to their specific situation.

**Method 4: Post-Booking Breakout Page**
Redirect after booking to a page with 2-3 short videos:
- "What to expect on your call"
- "Common concerns (and why they are normal)"
- "Student success story"

**Method 5: Pre-Call Email Sequence (up to 6 emails)**
1. Booking confirmation + what to expect
2. Brand story / mission
3. Relevant case study
4. "How it works" overview
5. FAQ / common concerns
6. Day-before reminder with agenda

**Method 6: iMessage Automation (SendBlue)**
Automated confirmation from "executive assistant" sent via blue-bubble iMessage. Feels personal, not automated. Dramatically increases open/read rates vs SMS.

## Pipeline Problem Diagnosis

| Symptom | Root Cause | Fix |
|---------|-----------|-----|
| Deals stall in follow-up | Not enough touches, same method | Use 8-touch with varied methods |
| Deals drop after first call | Weak discovery or pitch | Review call recordings, fix front end |
| Lots of deals, nothing closes | Bad qualification | Tighten qualification criteria |
| High no-show rate | Weak booking process | Implement 6 show-rate methods |
| Pipeline empty | Not working dead list | Bi-weekly DR + quarterly campaign |
| Income volatile month-to-month | Pipeline-dependent, not systematic | Build consistent daily pipeline ritual |
