# /link-outreach — Generate Link Outreach Campaigns

## Command Purpose
Create complete link outreach campaigns with personalized templates, prospect lists, and follow-up sequences.

## Invocation
```
/link-outreach [strategy] [niche] [target]
```

**Examples**:
```
/link-outreach HARO "SaaS marketing"
/link-outreach skyscraper "link building" 10
/link-outreach guest-posting "technical SEO" 20
/link-outreach pbn "ecommerce" 5
```

## Strategies Supported

1. **HARO** — Help A Reporter Out daily workflow
2. **SKYSCRAPER** — Reverse-engineer competitor links
3. **GUEST-POSTING** — Pitch guest articles to blogs
4. **PBN** — Private blog network setup checklist
5. **PRESS-RELEASE** — News-based link opportunities

---

## Campaign Outputs

### For Each Strategy, You Get:

1. **Prospect List** (50+ targets min)
   - Domain names
   - Domain rating
   - Niche relevance
   - Contact info (when available)

2. **Outreach Email Template** (personalized)
   - Subject line variations
   - Body copy (generic + customizable)
   - Follow-up sequence (3 emails)

3. **Tracking Spreadsheet** (ready to use)
   - Columns: Target, Status, Sent Date, Response, Link Posted, DR, Notes
   - Formulas for conversion rate tracking
   - Monthly metrics summary

4. **Weekly Execution Plan**
   - What to do each day
   - Time allocation
   - Expected output

5. **Success Metrics**
   - How to measure ROI
   - Expected conversion rates
   - Tracking checklist

## Strategy 4: PBN Setup

### What You Get:
```
PRIVATE BLOG NETWORK SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Pre-Launch Checklist:
□ Budget confirmed ($5K+ recommended)
□ Risk tolerance understood
□ 10+ domains identified for purchase
□ Hosting providers selected (3-5 different)
□ Content calendar created
□ Link strategy defined

Domain Selection (20-30 hours):
Step 1: Find 50 potential auction domains
Step 2: Vet each (age, DR, history, links)
Step 3: Select top 10
Step 4: Purchase from GoDaddy/Register Compass
Step 5: Research old backlinks (archive sites)

Setup Workflow (40-50 hours):
Step 1: Register domains at different registrars
Step 2: Set up hosting (different providers per domain)
Step 3: Install WordPress on each
Step 4: Use different themes per domain
Step 5: Create unique content strategy per blog

Footprint Avoidance Checklist:
□ Different registrars (not all GoDaddy)
□ Different hosting (SiteGround, Bluehost, etc.)
□ Different themes (Astra, Neve, GeneratePress, etc.)
□ Unique content (not copy-paste between sites)
□ Varied publishing schedule (not all Mondays)
□ Different anchor text patterns
□ Mix of dofollow/nofollow
□ Natural internal links within each blog

Content Guidelines:
- Pillar blog: 8-10 foundational articles
- Other blogs: 3-5 articles each
- Each article: 800-1500 words
- Original angle per blog
- Target different long-tail keywords
- Include 1-2 money site links per blog

Link Velocity Strategy (First Month):
Week 1: 1-2 links
Week 2: 1-2 links
Week 3: 2 links
Week 4: 1-2 links
Total: 5-8 links (spread naturally)

Risk Assessment:
- Low risk: Aged domains, different hosts, unique content
- Medium risk: Mix of new/aged, some shared infrastructure
- High risk: All new domains, same hosting, obvious footprints

Expected ROI:
- Setup cost: $2K-5K first year
- Monthly cost: $300-500 (hosting, content)
- Expected links: 50-100/year
- Ranking improvement: Significant (if done right)
- Payback: 2-4 months
- Risk: Deindexing if detected (~5% with proper setup)

Domain Forensics Template:
- Domain name
- Registration date
- Age (years)
- Wayback history (clean? Y/N)
- Referring domains (count)
- Domain Rating (Ahrefs)
- Trust Flow (Majestic)
- Anchor text analysis (old links)
- Purchase price
- Monthly cost
- Content strategy

Maintenance Checklist:
□ Content published on schedule
□ No obvious footprints (different hosts)
□ Links placed naturally (not all week 1)
□ Anchor text varied per domain
□ No cross-linking between PBN sites
□ Renewals set to auto-pay
□ Backups configured
```
