# /local-seo-audit -- Local SEO / GBP Audit

## Command Purpose

Run a comprehensive audit of a local business's SEO presence covering Google Business Profile optimization, citation health, NAP consistency, competitive positioning, and ranking factors. Produces a prioritized action plan with quick wins and long-term improvements.

## Invocation

```
/local-seo-audit [business-name] [location] [primary-service]
```

**Examples:**
```
/local-seo-audit "Smith Plumbing" "Austin TX" "emergency plumber"
/local-seo-audit "Elite Dental" "Vancouver BC" "cosmetic dentist"
/local-seo-audit "Fresh Carpet Care" "Madison WI" "carpet cleaning"
```

## Audit Sections

### Section 1: GBP Profile Audit

**Check each element:**

| Element | What to Check | Status |
|---------|--------------|--------|
| Business name | Contains primary keyword + location? (Service + City format) | Pass/Fail |
| Primary category | Matches core service exactly? | Pass/Fail |
| Secondary categories | All relevant categories selected? | Pass/Fail |
| Business description | Contains primary and secondary keywords naturally? | Pass/Fail |
| Service area | Correctly defined geographic coverage? | Pass/Fail |
| Hours | Complete and accurate? Including special hours? | Pass/Fail |
| Phone number | Trackable number? Consistent with citations? | Pass/Fail |
| Website URL | Points to relevant service page (not just homepage)? | Pass/Fail |
| Photos | Original (not stock)? Geo-tagged? EXIF data injected? OCR-optimized? | Pass/Fail |
| Posts | 5+ per week? Umbrella silo structure? Secondary keywords? | Pass/Fail |
| Reviews | Count vs competitors? Average rating? Response rate? | Pass/Fail |
| Products/Services | Listed with descriptions and pricing? | Pass/Fail |
| Q&A | Seeded with common questions + keyword-rich answers? | Pass/Fail |

### Section 2: Posting Strategy Audit

**Evaluate current posting:**
- Posting frequency (target: minimum 5/week)
- Post types distribution (What's New vs Event vs Offer)
- Keyword usage in posts (primary and secondary targeting)
- Post copy anatomy (headline, secondary terms, brand, owner name, location, POI)
- CTA button usage (do-follow link to service page?)
- Interlinking between posts (umbrella silo structure?)
- Event/Offer alignment with current "What's New" keyword silo

**Generate:**
- 12+ secondary keywords for next quarter
- Monthly silo plan (1 primary focus, 4+ secondary keywords per month)
- Sample post copy following the 6-element anatomy

### Section 3: Image SEO Audit

**Check each image:**
- Original vs stock? (Stock = dead weight)
- OCR text overlays present? (Primary keyword + location)
- Font clarity (sans-serif for OCR readability?)
- File naming convention (keyword-hyphens.jpg vs IMG1234.jpeg)
- EXIF data injection (GPS coordinates, NAP, persona name)
- File metadata (Title, Subject, Tags populated?)
- Geo-tagging (latitude/longitude hardcoded?)

### Section 4: Citation & NAP Audit

**NAP Consistency Check:**
- Primary NAP: Name, Address, Phone -- are they 100% identical across all listings?
- Common inconsistencies: suite numbers, abbreviations (St vs Street), phone format
- Cross-check against GBP, website footer, top 10 citations

**Citation Count Assessment:**
- Current citation count vs competitors
- Benchmark: Google "[competitor name]" in brackets, compare result counts
- Target: beat competitor citation count by 20-30%
- Missing from key directories? (Yelp, Yellow Pages, industry-specific)

**Citation Quality:**
- Country-specific citations used? (Canadian for Canadian businesses, etc.)
- Indexed citation rate? (Target: 70-80% for regular, 90%+ for PBN citations)
- Any duplicate or conflicting listings?

### Section 5: Competitive Positioning

**Map Pack Analysis:**
- Who occupies positions 1-3 for primary keyword + location?
- Their review count and average rating
- Their citation count (Google "[name]" in brackets)
- Their posting frequency and quality
- Their proximity to city center vs your location

**Gap Assessment:**
- Where do you outperform competitors? (Reviews? Citations? Proximity?)
- Where are you behind? (Posting frequency? Image SEO? Link quality?)
- What would it take to break into the map pack? (Citations only? Need backlinks?)

### Section 6: Local Ranking Factor Assessment

**Score each factor (1-10):**
1. NAP consistency / citations -- ___/10
2. Location proximity to searcher -- ___/10
3. Reviews (quantity + quality + recency) -- ___/10
4. GBP optimization (posts, photos, categories) -- ___/10
5. CTR signals (branded search volume) -- ___/10
6. Organic website SEO (on-page, backlinks) -- ___/10

**Overall Local SEO Health Score: ___/60**

### Section 7: Syndication & Link Tier Audit

**Tier 1 Properties:**
- Are GBP posts being syndicated to social profiles? (Facebook, Instagram, LinkedIn, Pinterest, Tumblr, Blogger)
- Are social signals being built on syndicated posts?
- Email/SMS push for high-value offers?

**Tier 2 Automation:**
- GSA contextual link blasts running monthly?
- Linked to Tier 1 assets (NOT directly to GBP)?

**Event Submission Backlinks:**
- Using local event submission sites?
- Linking to GBP Event Post URLs?

## Output: Prioritized Action Plan

### Quick Wins (Week 1-2)
1. Fix NAP inconsistencies across all listings
2. Rename GBP to Service Keyword + Location format
3. Optimize GBP photos (EXIF injection, OCR overlays, proper filenames)
4. Seed Q&A with keyword-rich questions and answers
5. Start 5+/week posting cadence

### Foundation Building (Month 1)
1. Build 50+ citations (indexed via SpeedyIndexBot)
2. Set up umbrella silo posting structure
3. Create image SEO pipeline (EXIF + OCR + geo-tagging)
4. Set up Tier 1 syndication channels
5. Launch review acquisition campaign

### Growth Phase (Months 2-3)
1. Add PBN citations if needed (200 batch)
2. Build social signals on Tier 1 syndicated posts
3. Set up Tier 2 backlink automation
4. Expand to event submission backlinks
5. Begin guest post outreach (if competitive niche)

### Maintenance (Monthly Ongoing)
Follow the 10-step monthly workflow:
1. Keyword research (primary + 12+ secondary)
2. Image creation & SEO
3. Write posts
4. Select interlinking URLs
5. Content calendar
6. GBP scheduler
7. Interlink posts (umbrella silo)
8. Syndicate to Tier 1
9. Social signals
10. Tier 2 links

## Timeline Expectations

- Citation-only campaigns: top 10 within 4-8 weeks
- Full local SEO implementation: sustained ranking dominance in 3-6 months
- Highly competitive niches (lawyers, real estate): 6-12 months with full strategy
