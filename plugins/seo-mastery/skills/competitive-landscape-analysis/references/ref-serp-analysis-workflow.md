# SERP Analysis Workflow -- Complete Reference

## TOP 10 EVALUATION CHECKLIST

For every target keyword you intend to pursue, run this checklist against all 10 page-one results:

### Step 1: Content Type Distribution
Open all 10 results. Classify each:
- Blog post / article
- Product page
- Category/collection page
- Tool / calculator / interactive
- Video (YouTube)
- Forum / Reddit / UGC
- News article
- Homepage

**Decision rule:** Your content type MUST match the dominant type. If 7/10 are blog posts, you need a blog post. If 7/10 are product pages, you need a product page. Fighting the SERP's intent is a losing battle.

### Step 2: Domain Authority Range
Use Ahrefs or SEMrush Chrome toolbar (both free) to check DR/DA of all results:
- All results DR 60+ = very competitive, need significant authority
- Mixed (some DR 20-40) = opportunity for newer sites
- DR under 30 appearing = highly winnable

### Step 3: Content Depth Analysis
For each top result, assess:
- Approximate word count (use Word Counter Plus extension)
- Number of images and whether they are original or stock
- Embedded videos present?
- Tables and structured data?
- Number of H2/H3 sections (content organization depth)

**Benchmark rule:** Your content should match or exceed the depth of the best-ranking result, not the average.

### Step 4: Content Freshness
- Check last-modified dates (look for "updated" dates in content or byline)
- Stale content (2+ years without update) = opportunity to outrank with fresh content
- If top results are all recently updated, the niche is actively maintained by competitors

### Step 5: Schema / Rich Results
Check which SERP features competitors have:
- Star ratings (Review schema)
- FAQ dropdowns (FAQPage schema)
- Product price/availability (Product schema)
- Featured snippet capture
- Image carousels
- Video thumbnails

**Opportunity:** If competitors lack schema but the SERP shows schema-enabled features for similar queries, implementing schema gives you a visual advantage.

### Step 6: Meta Title Optimization Gaps
The fastest competition check:
- Search keyword, read all 10 meta titles
- Count how many have the EXACT keyword in their title
- If fewer than 5/10 have exact keyword in title = significant opportunity
- If titles are generic or brand-focused without keyword = wide open

### Step 7: Internal Link Strength
For each competitor, check in Ahrefs:
- How many internal links point to this ranking page?
- Is the page linked from the homepage or navigation?
- Is it an orphan page with minimal internal links?

**Opportunity signal:** Competing page with few internal links is vulnerable to a well-linked alternative.

### Step 8: Backlink Count per URL
In Ahrefs, check referring domains to the SPECIFIC URL (not just the domain):
- Under 5 referring domains: on-page + internal links can compete
- 5-20 referring domains: need moderate link building
- 20-50 referring domains: need significant link building
- 50+ referring domains: need either outstanding content or time + persistent link building

## CONTENT FORMAT DETECTION

### Matching SERP Intent to Content Strategy

| SERP Pattern | Your Response |
|---|---|
| All listicles ("10 best...", "7 ways...") | Create a better listicle with more items, better research, original data |
| All how-to guides | Create comprehensive step-by-step guide with screenshots/video |
| Mix of blog + product pages | Create BOTH: informational blog post + optimized product/service page |
| Featured snippet present (paragraph) | Structure your content to answer the exact question in 40 words or fewer |
| Featured snippet present (list) | Use a numbered or bulleted list that directly answers the query |
| People Also Ask dominant | Create comprehensive content that answers the PAA questions |
| Local pack (map) showing | Need local SEO strategy (GBP optimization + local content) |
| Video carousel | Create YouTube video optimized for the keyword + embed in article |

## GOOGLE ADS DATA FOR SEO PRIORITIZATION

### Why CPC Data Matters for SEO (Cromwell / Gorrono)
- High CPC = advertisers willing to pay = proven commercial value
- Keywords with zero CPC often have low commercial intent (pure informational)
- DataForSEO SERP parser reveals how many paid results sit above organic results

### Prioritization Framework
| CPC Range | SEO Priority | Reasoning |
|---|---|---|
| $0 | Low priority (unless high volume) | No proven commercial value |
| $0.50-$2.00 | Medium priority | Moderate commercial intent |
| $2.00-$10.00 | High priority | Strong commercial intent, worth competing |
| $10.00+ | Highest priority (if achievable KD) | Exceptional commercial value |

### Using Ads Data for Content Strategy
1. Pull CPC for all target keywords
2. Sort by CPC descending
3. High CPC + Low KD = "golden keywords" (commercial value + achievable ranking)
4. High CPC + High KD = long-term targets (build toward with content cluster)
5. Low CPC + Low KD = quick-win traffic builders (volume play, monetize with ads)

## COMPETITOR CONTENT STRUCTURE ANALYSIS

### What to Document for Each Competitor
1. **Site architecture:** How many categories/sections? How deep is the hierarchy?
2. **Content volume:** Approximate total indexed pages (use `site:domain.com` search)
3. **Publishing frequency:** Check blog archive or Wayback Machine for posting cadence
4. **Content types used:** Info vs commercial ratio? Guides? Tools? Videos?
5. **Internal linking patterns:** Hub-and-spoke? Flat? Random?
6. **Author strategy:** Single author? Multiple? Real bios or generic?
7. **E-E-A-T signals:** Author credentials, citations, expert quotes, about page depth

### Gap Identification Process
1. List all topics/keywords competitor ranks for (Ahrefs Organic Keywords)
2. List all topics/keywords you currently rank for
3. Subtract to find gaps -- keywords they have that you do not
4. Prioritize gaps by: volume, KD, commercial value (CPC), relevance to your business
5. Cluster gaps into content themes for batch production

## AUTHORITY SCORE VERIFICATION

### Domain Authority vs Page Authority
- **Domain authority/DR** = overall site strength. Useful for estimating how hard it is to compete in a niche.
- **Page authority/UR** = strength of a specific URL. More relevant for individual keyword competition.
- A DR 80 site can have weak individual pages (UR 10) on topics it covers thinly.
- A DR 20 site can have strong individual pages (UR 40) in its core niche.

### Reality Check on Third-Party Scores
- DR/DA/Authority Score are approximations based on backlink data
- Google does NOT use these scores (Google has internal signals)
- Useful for RELATIVE comparison, not absolute judgment
- A site with DR 30 + strong topical authority + good on-page can beat DR 60 generalist sites
- Always verify with actual SERP results, not just tool scores
