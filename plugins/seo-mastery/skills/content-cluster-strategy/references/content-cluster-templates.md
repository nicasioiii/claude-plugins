# Content Cluster Templates

Ready-to-use cluster templates organized by business type and topic structure.

## Template 1: Product/Service Review Cluster

**Best for:** E-commerce, affiliate, SaaS, agencies

**Cluster structure:**

**PILLAR: "Ultimate Guide to [Product Category]"**
- Word count: 3,000-4,000
- Targets: "[Product category]" (broad, 500+ monthly searches)
- Contains: Overview, benefits, how to choose, top options, comparison table, FAQ

**SATELLITE 1: "Best [Product] for [Specific Use Case]"**
- Word count: 1,500-2,000
- Targets: "Best [product] for [use case]" (long-tail, 100-300 searches)
- Contains: Problem statement, specific recommendations, detailed comparison, where to buy

**SATELLITE 2: "[Product] Comparison: [Brand A] vs [Brand B]"**
- Word count: 1,500-2,000
- Targets: "[Brand A] vs [Brand B]" (comparison intent, 50-150 searches)
- Contains: Side-by-side comparison, pros/cons, best for X, verdict

**SATELLITE 3: "[Product] Pricing & Cost Breakdown"**
- Word count: 1,200-1,500
- Targets: "[Product] cost," "[Product] price" (decision-phase, 80-200 searches)
- Contains: Pricing overview, cost comparison, ROI, value assessment, where to buy cheapest

**SATELLITE 4: "[Product] Review: [Top Recommendation]"**
- Word count: 1,500-2,000
- Targets: "[Product name] review" (product-specific, 50-150 searches)
- Contains: Product overview, features, performance, pros/cons, where to buy

**SATELLITE 5: "[Product] for [Demographics]: Complete Guide"**
- Word count: 1,500-2,000
- Targets: "[Product] for [demographics]" (audience-specific, 80-150 searches)
- Contains: Demographic-specific needs, recommendations, buying guide, budget options

**Interlinking map:**
```

├─ Links to: S1, S2, S3, S4, S5
└─ Linked FROM: S1, S2, S3, S4, S5

S1 (Use Case Guide)
├─ Links to: PILLAR, S2, S3
└─ Linked FROM: PILLAR, S2

S2 (Brand Comparison)
├─ Links to: PILLAR, S1, S4
└─ Linked FROM: PILLAR, S1, S4

S3 (Pricing)
├─ Links to: PILLAR, S4, S5
└─ Linked FROM: PILLAR, S4, S5

S4 (Top Product Review)
├─ Links to: PILLAR, S2, S3
└─ Linked FROM: PILLAR, S2, S3

S5 (Demographics Guide)
├─ Links to: PILLAR, S1, S3
└─ Linked FROM: PILLAR, S1, S3
```

**Publishing timeline:** Pillar week 1, Satellites weeks 2-6 (staggered or batch)

## Cluster Customization Framework

**Adapt templates by:**

1. **Industry:** Adjust satellite topics to your industry (template 1 for retail, template 2 for services)
2. **Audience:** Adjust depth/sophistication to audience knowledge level
3. **Competition:** Add more satellites if competition is high; fewer if low competition
4. **Keywords:** Replace generic pillar/satellite topics with YOUR actual target keywords
5. **Content type:** Add videos, tools, or interactive elements based on your capability

**Customization example:**

**Template 1 (Product Review) adapted for software SaaS:**
- Replace "Product" with "Software"
- Replace "where to buy" with "free trial" or "pricing plans"
- Add "integrations" satellite
- Add "industry-specific use case" satellites
- Adjust interlinking to match SaaS buying journey
