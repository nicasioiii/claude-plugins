# Internal Linking Playbook -- Complete Reference

## THE 6-STEP INTERNAL LINKING PROCESS (Hockman)

### Step 1: Pick Target Keywords
Select 3-4 target keywords for the page you want to rank higher.

### Step 2: Find Relevant Source Pages (Batch 1)
Find 3-4 pages on your site with topical relevance. Even one paragraph (100 words) about the related topic is enough.

### Step 3: Add Exact-Match Anchors
On those 3-4 pages, add internal links using exact-match keyword anchor text. One keyword per page, no repeats.

### Step 4: Find More Source Pages (Batch 2)
Find 3-4 MORE relevant pages. Add exact-match keywords BUT include filler words in the anchor:
- "my best workout tips"
- "working out tips guide"
- "easy workout tips and tricks"

### Step 5: Find More Source Pages (Batch 3)
Repeat Step 4 with another batch. More filler word variety.

### Step 6: Partial Matches and Rearrangements
After 9-12 exact-match internal links, switch to partial matches:
- "tips on working out" instead of "working out tips"
- "vacuums that are top rated" instead of "top vacuums"

## ANCHOR TEXT RATIOS AT SCALE (Hockman)

For pages with 50+ internal links pointing to them:

| Anchor Type | Ratio | Example |
|---|---|---|
| Exact match | ~30% | "workout tips" |
| Exact match + filler words | ~60% | "best workout tips for beginners" |
| Partial matches / rearrangements | ~10% | "tips for your daily workout" |

## KORAY'S INTERNAL LINK CONFIGURATION RULES

Koray's rules differ from Hockman's ratio approach. Both are valid; Koray's is more precise for semantic content networks.

1. **Most important anchor text in root document.** Root articles carry the most weight.
2. **First anchor has most weight.** Within a page, the first internal link carries more weight than subsequent ones.
3. **One link per heading section.** Space links apart; no clusters of internal links.
4. **Top 9-10 headings matter most.** Bottom sections can be lower quality / less critical.
5. **Fewer links per page = more weight per link.** Over-linking dilutes signal.
6. **Anchor text must match target page H1.** The anchor phrase should appear in both source heading AND target H1.
7. **Max 3 reuses of same anchor across entire SCN.** More than 3 = diminished signal, looks manipulative.
8. **No reciprocal links by default.** Only link back when PageRank return is strategically justified. Example: cold-water article links to root but NOT back.
9. **Justify every link with contextual relevance.** Mention the concept 1-5 times near the link in unique, non-gibberish content.

## INTERNAL LINK POWER -- CASE STUDIES

### Spencer Haws / Niche Pursuits Study
Out of 47 pages tested with ONLY internal link additions (no backlinks, no on-page changes):
- **36 out of 47 pages increased in rankings** (77% success rate)
- Average: 2-3 internal links added per post with keyword-optimized anchor text
- Results:
  - One article: rank 100+ -> **#1** (5 internal links)
  - Another: #32 -> #1
  - Another: #21 -> #1
  - Another: #11 -> #1

### Industry Benchmark
**3 internal links are roughly equal to 1 external backlink** in ranking power.

## TOPICAL RELEVANCE REQUIREMENT (Hockman)

Do NOT link pages with no topical connection.

**Acceptable:** A page with one paragraph (100 words) about the related topic -- that section provides enough relevance to justify an internal link.

**NOT acceptable:** A page about space heaters linking to an air conditioner page with zero overlapping content.

**Casey Keith's method for finding opportunities:**
1. Search `site:yourdomain.com [keyword]` in incognito
2. Google's top result = where it currently sends juice for that keyword
3. Find other articles mentioning the keyword
4. Add internal links from those articles to your target page

## FINDING INTERNAL LINK OPPORTUNITIES

### Method 1: Master Content Database (Authority Hacker)
Track all published articles in spreadsheet. When publishing new content, scan database for related articles to link from/to.

### Method 2: Google Site Search (Authority Hacker / Casey)
`site:yourdomain.com + keyword` -- finds pages mentioning a term that could link to your target page.

### Method 3: Ahrefs Site Audit (Authority Hacker)
Ahrefs Site Audit -> Link Opportunities tab provides automated internal link suggestions.

### Method 4: Link Whisperer (Gorrono)
WordPress plugin that automatically suggests internal links as posts publish. Set to "opportunistic" mode for automatic internal linking based on keyword relevance.

### Method 5: Working Backwards (Authority Hacker)
1. Identify the page you want to push
2. Find high-authority pages on your site: Ahrefs -> Site Explorer -> Pages -> Best by Links
3. Add links FROM those high-authority pages TO the target page
4. Add new sentences if needed to create natural context for the link

## INTERNAL LINKING BEST PRACTICES

### Anchor Text
- Use keyword-rich anchor text (NOT "click here" or "this article")
- Max 5 words per anchor text (Maddy Osman) -- more looks spammy
- Do not hyperlink punctuation
- Vary anchors across different linking pages

### Link Behavior
- Open internal links in same tab (standard behavior)
- Keep all internal links as dofollow (default)
- Boilerplate/navigation links carry less weight than in-content contextual links (Koray)

### Link Direction Strategy
- Link FROM high-authority pages TO pages you want to push
- Link FROM new articles TO existing cornerstone content
- After publishing new content, go back and add links from old relevant articles

### When to Add New Sentences for Links
Sometimes the ideal source page does not mention the keyword you need to link. In this case:
- Add 1-2 new sentences to the source page
- Make them genuinely useful (not just link containers)
- Naturally incorporate the anchor text within the new content
- This is an acceptable practice (Authority Hacker explicitly recommends it)

---

## COMPETITOR INTERNAL LINK PROTECTION (Maddy Osman)

- Never link to brand competitors
- Never link to keyword competitors (using anchor text that competes with your keywords)
- Once you target a keyword on one page, strengthen that page rather than creating a competing page
- Do not link to articles similar to yours -- external links should enhance, not substitute
