# Directory SEO Playbook

## Directory Business Model

### Five Ideal Business Qualities
1. Cheap to start
2. Fully remote
3. Inherently scalable to global audience
4. High net margin
5. Can be sold as an asset

Only other model fitting all 5: SaaS. Directories = "digital real estate" but cheaper.

### Three Directory Types That Survive AI
1. **Community-based** -- marketplace directories or shared-interest hubs
2. **Omnipresent** -- not solely SEO-reliant (YouTube, Reddit, TV, social, LLM SEO)
3. **Specialized data** -- data so hard to get that no one else bothers

### Two Building Approaches
| Style | Description | Best For |
|---|---|---|
| **Strategic** | Run through all 7 filters. Higher win rate. | Non-coders, set-and-forget assets |
| **Shotgun** | Check a few filters, build fast, repeat. | Coders who implement quickly |

## Niche Selection 7-Filter Framework

### Filter 1: Keyword Difficulty + Search Volume
- Target KD under 20 (Ahrefs), ideally under 5
- Check location sub-keywords: consistent 90-150/month per city at KD 0-2 = green flag
- For display ads: aim for 20K-50K monthly visitors threshold
- Ahrefs zero-volume caveat: 0-10 volume keywords often have real traffic

### Filter 2: Competitive Research
- Google keyword + city, find other directories (skip Reddit, Yelp)
- Paste into Ahrefs: monthly visitors, backlinks, referring domains, traffic chart
- Yelp is NEVER a deal-breaker -- topical authority beats Yelp's diluted relevance
- Look for mom-and-pop competitors
- Even 100,000 low-quality backlinks shouldn't scare you

### Filter 3: Problem Solving
- Google Maps: mismatched categories = fragmented results = opportunity
- Reddit: search keyword, look for "Are there any X near me?" posts
- TikTok: viral videos with "where is this?" comments
- Note subreddits for future distribution channels

### Filter 4: Search Intent
- Broad keywords are traps -- seductive volume but foggy intent
- Go deeper to KD 0-5 sub-niches with clear intent
- Location-based keywords have inherently clear intent

### Filter 5: Future Workload
- Low (evergreen): dog parks -- won't change in 10-20 years
- Medium: Amazon bin stores -- review every 6-12 months
- High: AI tool directories -- constant changes
- Ask: "What does maintaining this look like in a year?"

### Filter 6: Improvement Potential
- Data enrichment: add info competitors lack (accessibility, pricing, insider tips)
- User experience: fewer clicks to find info
- Research Google reviews for recurring complaints in your niche

### Filter 7: Monetization Potential
- **Display ads:** 20K-50K visitors = ~$1K/month. Varies by niche RPM.
- **Sponsored listings:** High-AOV businesses will pay ($100-200/month). Check CPC as signal ($3.50+ = valuable leads). Check market size of potential buyers.
- **Affiliate:** Hardest. Requires high-AOV + matched intent. Avoid Amazon affiliate (2-3% commission).
- **Digital products:** Often discovered after traffic (e.g., $19 PDF with insider data).

## Finding Directory Ideas

### Method 1: Organic Frameworks
Start from personal experience, hobbies, pain points.

### Method 2: "Near Me" Keywords in Ahrefs
- Type "near me" into Keyword Explorer > Matching Terms
- Skip first pages (too broad/branded)
- Avoid branded keywords (Walmart, Target) and broad keywords (jobs, food)
- Filter KD to max 5
- Branded directory risks: trademark issues, buyer hesitation

### Method 3: Leveraging Successful Directories
Study successful directories in adjacent niches. Use Ahrefs "top organic competitors" to discover related directory sites.

## On-Page SEO for Directories

### Keyword Clusters
- Never target just one keyword -- build clusters of synonyms and layman terms
- Check Ahrefs parent topics for shared intent
- Cluster hierarchy: highest-volume synonym at top (H1), lower volume in H2/H3

### Header Strategy
- Every page: exactly ONE H1 with target keyword
- H2/H3: insert secondary keywords from cluster
- Pillar page exception: H1 = umbrella keyword, target keywords as H3s
- Rename listings to match search terms, not official business names

### URL Slug Structure (Plan Before Building)
- Individual listings: /state/city/business-name OR /niche/business-name
- Pillar pages: /keyword-state OR /keyword-city
- Include state in URL to avoid duplicate pages (30+ US cities named Springfield)

### Thin Content Avoidance
- Create listing template with consistent structure but personalize each listing's data
- Add city-specific content for location pages
- Balance content volume with value provided

### Images & Alt Text
- Insert keyword in alt text; copy to title, caption, description
- Keep images under 100KB generally
- Google Images TOS: using Google-sourced images is against TOS (risk: penalty)

### Internal Linking
- Top 3 most influential directory SEO tactics
- Creates topical authority (outranked much higher DR sites through topical authority alone)
- Location pattern: homepage > city pages > listings + neighboring cities + blogs
- Pillar page pattern: bottom of each state page links to all other state pages
- Avoid orphan pages; ~25 most popular cities for homepage links

### Additional Elements
- FAQ sections (enable FAQ schema)
- Table of contents (essential for pillar pages)
- Foundational pages (About, Contact) -- trust signals
- Breadcrumbs for site structure understanding
- Social links in footer (only if you'll maintain them)

## Off-Page SEO for Directories

### National SEO, Not Local
- Do NOT buy citations (NAP consistency is for local businesses, not directories)
- In low-competition niches, 2-3-5 backlinks can rank a directory

### Backlink Quality Assessment
Check in Ahrefs: DR, Domain Traffic, Page Traffic (most important), Linked Domains (5,000+ = spam).

### Link Building Methods (Tier List)
- **S:** Link inserts/swaps ($50-100, full quality control)
- **A:** Reddit mentions (free, high ranking posts)
- **B:** Guest posts/HARO, badges, niche edits
- **C:** PBNs (risky for beginners)
- **F:** Web 2.0, blog comments, local citations

### Badge/Award Strategy
1. Create "Top 5/10 Best X in [City]" page
2. Design badge in Canva
3. Generate embeddable HTML code
4. Email businesses with badge
5. Badge links back = backlink
6. Add internal links on award page

### Anchor Text
- Exact match, partial match, branded, generic -- diversify
- First link swap: use exact match. Second: switch it up
- Default destination: homepage. Over time, send some to location pages

### When to Start Link Building
- Finish ALL on-page SEO first
- Verify you're ranking for intended keywords
- Wait for natural plateau, then build with surgical precision

## 12-Step Building Roadmap

1. Find and validate niche (7 filters)
2. Plan monetization (before building)
3. Scrape data (Outscraper for Google Maps)
4. Clean data (verify contextual relevance)
5. Enrich data (add value beyond Google Maps)
6. Buy domain (only AFTER data is ready)
7. Build website (WordPress + Elementor Pro, or AI tools)
8. Implement on-page SEO
9. Set up GA + GSC (submit sitemap)
10. Launch with Reddit post
11. Wait 1-2 months
12. Build backlinks to push past plateau

## Common Mistakes
1. Buying domain before data is ready
2. Not planning monetization before building
3. Choosing too-competitive niches (stay under KD 20)
4. Trusting KD scores blindly
5. Buying Fiverr backlinks
6. Buying citations for directories (wrong SEO game)
7. Not setting up slug structure upfront
8. All-exact-match anchor text
9. Ignoring internal linking
10. Shiny object syndrome
