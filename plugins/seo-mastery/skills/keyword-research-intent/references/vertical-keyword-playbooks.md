# Vertical-Specific Keyword Playbooks

Industry-specific keyword strategies and patterns that differ by niche type.

## E-Commerce Vertical (Amazon, Shopify, Direct Sales)

**Business model:** Direct product sales with margin per transaction.

**Revenue per ranking:** HIGH (direct sales)

**Keyword characteristics:**
- Searchers are ready to buy
- Keywords almost always commercial intent
- Long-tail is critical: "best [product] for [use case]"
- Price keywords valuable: "cheap [product]," "[product] on sale"
- Comparison keywords high-value: "[brand] vs [brand]"

### E-Commerce Keyword Strategy

**Keyword breakdown:**
- 50% specific product keywords ("buy [product]")
- 25% "best X for Y" buying guides
- 15% price/deal keywords ("[product] discount," "cheap")
- 10% comparison keywords ("[brand] vs [brand]")

**Tool-specific keywords:**
- Product name + "review"
- Product name + "price"
- Product name + "cheap"
- Product name + "discount"
- "Best [category] under $X"
- "[Category] for [use case]"
- "[Category] alternative to [competitor]"

**Example: Athletic Shoe Ecommerce**

High-value keywords:
- Best running shoes for flat feet
- Nike Pegasus review
- Asics Gel-Kayano price
- Running shoes under $100
- Nike vs Asics running shoes
- Best trail running shoes for beginners
- Where to buy Saucony Ride 16

**Monetization:** Direct sales commission (high margin)

**Ranking difficulty:** Often competitive (other ecommerce sites)

**Timeline:** 8-16 weeks to ranking (commercial keywords slower)

### Ecommerce Content Structure

**Best-of article** (targets "best [product] for [use case]"):
- Introduction: Problem statement (Why does this use case matter?)
- Buying guide: What to look for
- Comparison table: Top 5-7 products side-by-side
- Individual reviews: 200-400 words per product
- Deals section: Where to buy + discount links
- FAQ: "Is [product] worth it?" "What's the difference between [model A] and [Model B]?"

**Product review** (targets "[Product] review"):
- Product overview: Specs, price, availability
- Design & comfort: Real user experience
- Performance: How does it perform?
- Pros and cons: Honest assessment
- Where to buy: Direct links
- Verdict: Final recommendation

**Comparison article** (targets "[Brand] vs [Brand]"):
- Side-by-side specs: Visual comparison
- Design: Differences in build quality
- Performance: Real-world testing
- Price: Value for money
- Who should buy which: Decision framework
- Verdict: Which is better and when

## SaaS Vertical (Software Sales, Subscriptions)

**Business model:** Monthly/annual subscriptions or contract deals.

**Revenue per ranking:** VERY HIGH (long LTV)

**Keyword characteristics:**
- Keywords are highly competitive (SaaS budgets large)
- "Free trial," "pricing," "alternatives" are gold
- Problem-focused keywords drive awareness
- "[Tool] vs [competitor]" shows buying intent
- "Best [category] for [industry]" shows specialization

### SaaS Keyword Strategy

**Keyword breakdown:**
- 40% problem/awareness ("how to [solve X]," "what is [problem]")
- 30% "alternatives" and comparisons ("[tool] alternative," "[tool] vs [tool]")
- 15% pricing/free trial ("[tool] free trial," "[tool] pricing")
- 10% "[tool] review"
- 5% navigational ("[tool] login," "[tool] features")

**High-value keyword patterns:**
- "[Problem] solution"
- "[Problem] software"
- "Best software for [use case]"
- "Cheapest [category] for small business"
- "[Competitor] alternative"
- "[Tool] review for [industry]"
- "[Tool] free tier"
- "[Tool] integrations"

**Example: Project Management SaaS**

High-value keywords:
- Project management software for small business
- Asana alternative free
- Monday.com vs Asana pricing
- Best project management tool for remote teams
- Jira alternative low cost
- Monday.com integration Slack
- Is Monday.com worth the price

**Monetization:** Affiliate partnership (commission per trial/sign-up) OR content + lead generation

**Ranking difficulty:** VERY HIGH (major SaaS competitors with huge budgets)

**Timeline:** 12-24 weeks to ranking (competitive market)

**Win strategy:** Go niche—target "best [category] for [specific industry]" instead of "best project management tool"

### SaaS Content Structure

**Problem-focused guide** (targets "[Problem] solution"):
- Define problem: What is the problem? Why does it matter?
- Problem impact: Real cost (time, money, productivity lost)
- Solution overview: General solution approach (not tool-specific yet)
- Solution categories: Different ways to solve (manual, spreadsheets, software)
- Software comparison: 3-5 tools that solve this problem
- Implementation guide: How to actually implement the solution
- FAQ: "Can we solve this without software?" "What's the ROI?"

**Comparison article** (targets "[Tool] vs [Tool]"):
- Overview: What does each tool do?
- Pricing: Side-by-side cost comparison
- Features: Detailed feature matrix
- Best for: Which tool for which use case
- Integrations: Which tools integrate with what
- User experience: Which is easier to use
- Support quality: Which has better support
- Verdict: When to choose which tool

**Free alternative article** (targets "[Tool] alternative free"):
- Original tool: Why people use [tool]
- Problems with [tool]: Cost, complexity, feature gaps
- Free alternatives: 5 free tools that do similar things
- Feature comparison: Free vs paid tools
- Trade-offs: What you lose going free
- Setup guide: How to get started with free option
- When to upgrade: When to move from free to paid

**Pricing-focused review** (targets "[Tool] pricing"):
- Pricing overview: All pricing tiers
- Which plan for which use case
- Cost per user calculation
- ROI calculation: Will this tool pay for itself?
- Hidden costs: Setup, training, overages
- Free tier options: Is there a free option?
- Money-back guarantee: Risk-free trials
- Verdict: Is it worth the price?

## Agency/B2B Services Vertical (Consulting, Agencies, Professional Services)

**Business model:** Lead generation → sales call → contract deal

**Revenue per ranking:** VERY HIGH (LTV from contracts)

**Keyword characteristics:**
- Keywords are highly specific ("best [service] for [industry]")
- "How to" and problem keywords drive early awareness
- "Near me" and local keywords important
- "For [industry]" filtering critical
- "[Service] cost" indicates buying intent

### B2B Services Keyword Strategy

**Keyword breakdown:**
- 40% problem-aware keywords ("how to [solve X]," "why [problem]")
- 30% "for [specific industry]" keywords
- 15% "near me" and local keywords
- 10% "[Service] cost/pricing"
- 5% "best [service]"

**High-value keyword patterns:**
- "[Service] for [industry]"
- "Best [service] company"
- "[Service] near me" (local)
- "[Service] cost"
- "How to [achieve outcome]"
- "[Service] agency [industry]"
- "Affordable [service]"
- "[Service] ROI"

**Example: SEO Agency**

High-value keywords:
- SEO for ecommerce brands
- Best SEO agency for healthcare
- Local SEO services near me
- SEO cost pricing
- How to improve Google rankings
- SEO for small business
- Technical SEO agency
- SEO ROI calculator

**Monetization:** Lead generation → sales call → contract deal

**Ranking difficulty:** Medium-to-high (other agencies competing)

**Timeline:** 12-20 weeks to ranking

**Win strategy:** Own specific niches ("SEO for medical practices," "SEO for law firms")

### B2B Services Content Structure

**Problem awareness article** (targets "how to [achieve outcome]"):
- Problem definition: What outcome are they trying to achieve?
- Why it matters: Business impact of achieving/not achieving
- Common mistakes: How most companies mess this up
- Best practices: How to do it right
- Metrics: How to measure success
- DIY vs professional: When to DIY vs hire expert
- Next steps: If they decide to hire help (soft CTA)

**Industry-specific guide** (targets "[Service] for [industry]"):
- Industry overview: What's unique about this industry
- Industry challenges: Specific pain points
- How [service] solves these challenges
- Industry best practices: Specific approaches that work
- Case studies: Real examples from this industry
- Industry-specific metrics: KPIs that matter
- Getting started: Steps to implement
- Finding a [service] provider: What to look for (CTA)

**Local services page** (targets "[Service] near me"):
- Service overview: What you provide
- Local coverage area: Where you serve
- Local examples: Companies you've worked with
- Local market insights: What works in your area
- Getting started locally: Consultation CTA
- Testimonials: From local clients
- Contact information: Clear CTA

**Pricing/cost article** (targets "[Service] cost"):
- Factors that affect pricing: What drives cost
- Typical price ranges: Budget, mid-market, premium
- Price comparison: DIY vs hiring vs hiring agency
- What's included: What you get for your money
- ROI calculation: Will it pay for itself
- Hidden costs: Often-overlooked expenses
- Price vs quality: Cheapest isn't always best
- Getting a quote: CTA to schedule consultation

## Affiliate/Content Vertical (No Direct Service)

**Business model:** Advertising + affiliate commissions (80%+ of revenue)

**Revenue per ranking:** MEDIUM (multiple revenue streams)

**Keyword characteristics:**
- ALL keywords are "best X," "X review," "X vs X"
- High search volume (1000+/month) preferred
- Long-tail critical for targeting "best X for [use case]"
- "Cheap," "budget," "affordable" modifiers important
- "Where to buy" and "[Brand] deal" keywords monetize well

### Affiliate Keyword Strategy

**Keyword breakdown:**
- 60% "best X" and "top X" keywords
- 20% "X review" keywords
- 10% "X vs X" comparison keywords
- 10% "buy X," "price," "deal" keywords

**High-value keyword patterns:**
- "Best [product]"
- "Top [products] for [use case]"
- "[Product] review"
- "[Brand] vs [brand]"
- "[Product] price"
- "Cheap [product]"
- "[Product] deals"
- "[Product] where to buy"

**Example: Fitness Affiliate Site**

High-value keywords:
- Best protein powder 2026
- Best home gym equipment
- Best kettlebells for beginners
- Whey vs casein protein comparison
- Best resistance bands for strength training
- Creatine supplement review
- Budget gym equipment under $100

**Monetization:** Affiliate commissions (primary), display ads (secondary)

**Ranking difficulty:** Medium-to-high (affiliate sites abundant)

**Timeline:** 8-16 weeks to ranking (volume helps)

**Win strategy:** Own specific niches or use cases ("Best kettlebells for women," "Best protein powder for gut health")

### Affiliate Content Structure

**Best-of article** (targets "best [product]"):
- Introduction: Why this product category matters
- Buying guide: How to choose [product]
- Comparison table: Top products side-by-side
- Individual reviews: 300-400 words per product
- Special categories: "Best budget," "Best premium," "Best for [use case]"
- FAQ: "What's the difference between [model A] and [model B]?"
- Final verdict: Our #1 pick (with affiliate link)

**Product review** (targets "[Product] review"):
- Product overview: What it is, price, where to buy
- Features: Detailed breakdown
- Build quality: Construction and durability
- Performance: Real-world testing
- Pros and cons: Honest assessment
- Similar products: Alternatives
- Where to buy: Multiple affiliate links
- Final verdict: Should you buy it

**Comparison article** (targets "[Product] vs [Product]"):
- Overview: What each product is
- Price comparison: Cost analysis
- Features: Side-by-side features
- Build quality: Which is more durable
- Performance: Which performs better
- Best for: When to choose which
- Value for money: Which is better value
- Verdict: When to choose each

## Cross-Vertical Principles

**These rules apply across ALL verticals:**

1. **Intent always determines monetization strategy**
   - Commercial intent keywords = direct sales/affiliate potential
   - Informational keywords = authority/ad revenue potential

2. **Keyword difficulty is relative to domain authority**
   - New site: Target KD <30
   - 6-month site: Target KD <45
   - 12+ month site: Target KD <60

3. **Volume-to-effort ratio matters**
   - Below 50/month = only pursue if KD <15 or unique opportunity
   - 50-200/month = good sweet spot for long-tail
   - 200+/month = mainstream keyword (higher competition)

4. **Niche specificity reduces competition**
   - "Best dog food" = massive competition
   - "Best dog food for sensitive stomach" = niche opportunity
   - Rule: More specific = lower competition, higher intent

5. **Monetization must match keyword intent**
   - Don't sell to info-seekers (they'll leave)
   - Don't write educational content for sales keywords (doesn't match intent)
   - Match content type to search intent
