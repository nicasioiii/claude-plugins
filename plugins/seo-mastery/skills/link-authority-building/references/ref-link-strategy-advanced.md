# Advanced Link Strategy

## Entity Stack Construction

### Components (Build from Day 1)
- Social profiles: Facebook, Twitter/X, LinkedIn, Instagram, Pinterest, YouTube
- Citations and directories: Yelp, Yellow Pages, industry-specific directories
- Google properties: GMB listing, Google Sites, Google Drive stacks
- Branded properties: Tumblr, WordPress.com, Medium, Blogger

### Trust Signals (External) -- Floate
- Domain age (continuous registration; drops/expirations count against you)
- Active social profiles with engagement (not just Zapier automation)
- Social signals (external mentions, shares, likes)
- WHOIS information: use complete registration data rather than private WHOIS (private WHOIS can be a trust-negative signal)
- Good hosting / IP neighborhoods: avoid cheap shared hosting; get dedicated IP
- On-page trust signals: SSL, legal pages, contact info, blog comments activity, testimonials, "As Featured In" logos, certifications

### Social Signals Service
- Signal Boy ($19-$29/link package) for daily social signals
- Purpose: build trust element that offsets link velocity and enables more aggressive dofollow anchors

## NoFollow Strategy Deep Dive

### Strategic Importance
- 2019: Google made nofollow a "hint directive" -- the key change enabling this entire strategy
- NoFollow links can now pass juice at Google's discretion based on page authority, link authority, and social signals
- Higher ratio of nofollow + trust signal links = ability to use more aggressive dofollow tactics

### NoFollow Link Building
- 100+ SOPs for building nofollow backlinks on DA/DR 80-99 sites
- Generic enough to link to any niche
- Average build time per link: 2-3 minutes
- A VA could build the entire database in ~5 hours (~$25 at $5/hr)
- Cost per link: under $2

### Disavow Methods
- **Link Research Tools** -- best link disavow tool; nearly automated; best for 10-20+ websites
- **Manual review** -- for individual sites or small batches
- Always include nofollow links as an option in disavow tools (contrarian per Floate)

## Anchor Text Ratios

### Floate's Guidance
- Link velocity matters MORE than anchor text right now
- Higher authority link source = less important anchor text ratios become
- Higher authority YOUR site = less important anchor text ratios become
- Recent trends favor more aggressive anchor text than conventional wisdom suggests
- Focus on creating UNIQUE anchor text on every single backlink rather than hitting exact ratios
- Internal links: much more aggressive exact-match keywords allowed vs. external

### Safe Distribution (Synthesized)
- Naked URL: ~30-35%
- Branded: ~30-35%
- Partial Match: ~10%
- Generic: ~15-20%
- Exact Match: use carefully, balanced by diversity

### Finding Your Ideal Ratio
1. Check current anchor distribution in Ahrefs
2. Compare to top 5-10 competitors
3. Match their distribution
4. Avoid anchor repetition -- never use the same exact anchor text from multiple sources

### Combining Anchor Types for Uniqueness
- "Check out the most awesome page from [Brand Name]" = branded + generic + partial match
- Each link should use a creative, unique combination

## Link Velocity Framework

### Core Rules (Floate)
- Velocity depends on: (a) number of pages receiving links, (b) number of existing links
- Think of velocity as page-by-page, not site-wide
- More pages on site = higher link velocity is safe (spread across pages)

### Velocity by Site Age
- Brand new sites: go very slow (1-3 links/month)
- Penalty-recovering sites: go very slow
- Aged sites with existing profiles: can handle more velocity
- Sites with natural link growth history: match or slightly exceed natural rate

### Dead Page Problem
- Building 10 links in 2 weeks to a page dead for 2-3 years = terrible signal
- Old pages getting sudden new links without content updates = red flag
- Always update content before building new links to aged pages

### Freshness Algorithm Impact on Niche Edits
- Google's freshness algorithm scores pages with single-link insertions lower if the page hasn't been meaningfully updated
- When buying niche edits, request content additions, imagery, or media updates alongside the link

## Tiered Link Building

### NoFollow Tiered Building (Activating Hint Directive)
- Most nofollow links just need an indexing service (Omega Index: ~10,000 links for under $100)
- Quality nofollow links from large publications need tier 2 links to activate juice passing

### Targets for Tiered Building
- Large publication nofollow links (The Sun, Forbes, etc.)
- HARO placement links
- Charity and sponsorship links
- Wikipedia links

### How It Works
- Build links + social signals pointing at the PAGE containing your nofollow link
- Goal: grow that page's authority so Google's hint directive passes juice through it
- Higher authority the host site (DR 95+): cheap/spam tier 2 links work
- Lower authority host: invest in quality tier 2 links

### Tier 2 Sources
- Cheap niche edits (Rhino Rank, Get Me Rankings)
- Niche-specific PBN link providers (vet with examples; find in Facebook groups, forums)
- Social signals to the page: Signal Boy, $19-$29/package

### Quality Tiered Method (SEO Gems)
- Find DR20-DR40 sites in Ahrefs "Best by Links"
- Identify pages with 50-150 existing referring domains
- Pay the site owner to insert a link on that page ($60-$70)
- The existing 50-150 referring domains ARE your tier 2 for free
- This is the most cost-effective tiered link building approach

### Contrarian Observation
- Fresh guest posts with nothing else were more affected by algorithm updates than niche edits
- Low-quality tiered link building was heavily hit by updates
- Quality tier 2 > volume of cheap tier 2

## Niche Edit Strategy

### Core Rules
- Way cheaper than guest posts
- Safe to use 5-10 niche edits diversified across link profile
- Freshness warning: pages updated with ONLY a link insertion score lower
- Best value: high-quality niche edits on pages with existing high referring domains

### When Buying Niche Edits
- Request content updates alongside the link (not just link insertion)
- Verify the page has real organic traffic (not just high DR)
- Check the page is topically relevant to your content
- Confirm the page has been updated recently or will be updated

## Infographic Link Building

- Repurpose existing blog content into infographics using Canva
- Include embed code below infographic (Siege Media Embed Code Generator auto-includes backlink)
- Submit to infographic directories (99Signals maintains updated list)
- Search: [keyword] "submit infographic" for additional submission sites
