# Citation & Directory Strategy: Building Local Authority

**Length:** ~300 lines
**Purpose:** Complete workflow for citation research, submission, and indexation

---

## Part 1: What Are Citations?

**Citation** = Online mention of NAP (Name, Address, Phone)
- With or without backlink
- Across directories, listings, social profiles
- Signals to Google that your business exists and is legitimate

**Citation Authority:**
1. **Tier 1:** Authority directories (Yelp, YP, Google Places equivalent)
2. **Tier 2:** Industry-specific (Healthgrades for medical, Angi for home services)
3. **Tier 3:** Local data aggregators (Factual, Localeze)
4. **Tier 4:** Niche directories (review sites specific to your industry)

**Impact on ranking:** Each citation signals **local authority + consistency**. Google's algorithm works like this: "If Business X appears in 100 directories with same NAP, it must be real and established."

## Part 3: NAP Standardization (Foundation)

Before submitting any citations, **create master NAP format.**

### Master NAP Document Template

```
BUSINESS NAME: [Exact Legal Name]
ADDRESS: [Street Address, City, State ZIP]
PHONE: [Format: (555) 123-4567 OR 555-123-4567]
WEBSITE: [https://yourdomain.com]
EMAIL: [contact@yourdomain.com]
HOURS: [Mon-Fri 9am-5pm / Sat 10am-4pm / Sun Closed]
```

**Standardization rules (non-negotiable):**
- Name: Always exact legal name (no abbreviations, no keyword variations)
- Address: Full street address; consistent capitalization & abbreviation (St vs Street)
- Phone: Choose format once (parentheses or dashes) and use everywhere
- Website: HTTPS only; consistent with/without www
- Hours: 12-hour format with am/pm; consistent day naming
- State: Always two-letter abbreviation (CA, TX, NY)

**Example of BAD inconsistency:**
```
GBP: "Joe's Carpet Cleaning Service"
Yelp: "Joes Carpet Cleaning"
YP: "Joe's Carpet Cleaning Services" (extra "Services")
Facebook: "Joe Carpet Cleaning" (different name entirely)
```

→ Google sees these as potentially different businesses = authority splits

## Part 4: Citation Submission Workflow

### Step 1: Audit Existing Citations

Before submitting new, check what already exists.

**Search string:** `[business phone] [business name]` in Google
- Example: `(608) 555-1234 "Joe's Carpet Cleaning"`
- Results show all indexed citations of your business

**Audit checklist:**
- [ ] GBP already claimed?
- [ ] Which directories have old/incorrect info?
- [ ] Are there duplicate listings (duplicate GMB)?
- [ ] Any rogue listings with wrong address?

**Action:** For each existing listing, **claim and update** before creating new citations.

### Step 2: Manual Submission (Start Here)

For first 10-20 directories, submit manually to ensure quality control.

**Process per directory:**
1. Search business name + city in directory
2. If found: Click "Claim Business" → update info
3. If not found: "Add Business" → fill all fields
4. Match **exact NAP** from master document
5. Add website URL (HTTPS)
6. Complete verification (email PIN, phone call, postcard—varies)
7. Publish

**Time estimate:** 3-5 minutes per directory × 20 directories = 1-2 hours

**Recommended order (easiest to hardest):**
1. Facebook (instant)
2. Foursquare (instant)
3. Bing Places (instant)
4. Apple Maps (instant)
5. YP (15 min)
6. Yelp (20 min + verification)
7. Angi (20 min + verification)
8. Industry-specific (varies)

## Part 5: Citation Indexation (Critical Step)

**Citation created ≠ Citation indexed**

Indexation = Citation appears in Google's index. Without it, citation doesn't help ranking.

### Check Indexation Status

**Method 1: Google Search**
```
Search: "[business name] [phone number]" (in quotes)
Look for: Indexed citations showing up in results
Timeline: 12-24 hours for initial indexation
```

**Method 2: Google Search Console**
```
Go to: Links report
Look for: New referring domains (should see cite directories)
Timeline: 24-48 hours after citation creation
```

**Method 3: Ahrefs / Semrush**
```
Check: Backlink profile growth
Look for: 2-3 new referring domains per week (cite directories)
Timeline: 1-2 weeks for all backlinks to propagate
```

### Indexation Acceleration Tools

If citations not indexing naturally, use accelerators:

| Tool | Cost | Method | Speed |
|------|------|--------|-------|
| **Speedy Index Bot (Telegram)** | $0.004 per link | Submits to Speedy Index API | 12-24 hours |
| **Speed Links** | $0.02 per link | Similar to Speedy Index | 12-24 hours |
| **GSC / Index URL API** | Free (yours only) | Direct Google submission | Instant (if own site) |

**Process:**
1. After citations created, compile URLs
2. Submit to indexation tool
3. Wait 12-24 hours
4. Re-check Google Search for indexed citations
5. Expected: 15-40% initial indexation, 60-80% after 1-2 weeks

## Part 6: Citation Quality Hierarchy

Not all citations equal. Google weights citations by:

1. **Directory authority** (Yelp > YP > local niche directory)
2. **Relevance** (Healthgrades for doctor > YP for doctor)
3. **Consistency** (exact NAP match > partial match)
4. **Recency** (updated this month > updated 2 years ago)
5. **Links** (citation with do-follow backlink > citation-only)

### Citation Hierarchy (Recommended Priority)

```
Tier 1 (MUST):
  Google Business, Bing, Apple Maps, Facebook, Yelp, YP

Tier 2 (SHOULD):
  Foursquare, Angi, industry-specific (5-10 directories)

Tier 3 (NICE):
  Niche local directories, chamber sites, local aggregators
```

**Strategy:** Nail Tier 1 (10 directories) before adding Tier 2-3.

## Part 7: Citation Consistency Checks (Ongoing)

Set quarterly audits to maintain consistency.

### Quarterly Audit Checklist

- [ ] Search "[business name] [phone]" - verify no rogue listings
- [ ] Spot-check top 10 directories - verify NAP accuracy
- [ ] Update outdated hours in major directories (seasonal changes)
- [ ] Check for duplicate GMB listings (claim and consolidate if found)
- [ ] Monitor Yelp reviews (respond to all)
- [ ] Update photos/description where possible
- [ ] Re-index new citations if submitted
- [ ] Compile indexation report (how many of our citations are indexed?)

### Common Consistency Issues & Fixes

| Problem | Cause | Fix |
|---------|-------|-----|
| Rogue listing with wrong address | Competitor or old submission | Claim it; merge or suppress in Google |
| Duplicate GMB | Accidental creation or API quirk | Use "Link your locations" tool to consolidate |
| Inconsistent phone format | Manual entry variations | Claim each; standardize to master format |
| Different business name | Someone else submitted | Claim listing; change name to exact match |
| Address variation ("Street" vs "St") | Copy-paste errors | Claim; correct to match master format |

## Part 9: Citation ROI Tracking

Monitor citation impact on traffic and leads.

### Metrics to Track

| Metric | Tool | Interpretation |
|--------|------|-----------------|
| **Indexed citations** | Google Search | % of submitted citations actually indexed |
| **Referring domains** | Ahrefs / GSC | How many directory domains link to you |
| **Local search rankings** | GBP Analytics | Position in local 3-pack |
| **Traffic from citations** | GA4 (referral source) | Clicks from directories to your site |
| **Calls from citations** | Call tracking / GBP Analytics | Calls initiated from directory listings |
| **Cost per citation** | Calculation | Total spend / number of indexed citations |

### ROI Benchmark

**Typical results after 60 days:**
- 30-50% of submitted citations indexed
- 5-20 new referring domains
- 20-40% increase in local search visibility
- 2-3x increase in calls/leads from local search

**Cost:**
- Manual (10 directories): 3 hours × $50/hr labor = $150
- Tool-based (50 directories): $50-100 tool cost + 2 hours labor = $150-200
- Full-scale (350 directories): $200 tool cost + 5 hours labor = $400

**Payback:** Typically 30-60 days (single new client often pays back entire investment)
