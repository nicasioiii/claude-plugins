# GBP Optimization

## Primary Keyword Foundation

### Formula
**Service Keyword + Location Modifier** (e.g., "Carpet Cleaning Madison WI")
- Do not get creative with naming -- match core service to city
- Adding city/state modifier triggers local relevance and attracts high-intent users
- This single alignment is often the difference between page-two obscurity and top-three map pack

### Primary Category Selection
- Choose the most specific GBP category matching your primary service
- Google allows one primary category and multiple secondary categories
- Primary category has the strongest ranking influence
- Change primary category to test ranking impact -- results visible within 1-2 weeks

## Secondary Keyword Engine

### Six Sources
1. **SEMrush:** Input primary + city, filter by relevance
2. **"People Also Search For":** Scroll to bottom of SERP, mine intent data
3. **Google Search Console:** Export CSV, isolate service+location keywords, deploy into posts
4. **GBP Performance Data:** "How people discovered you" section shows raw search terms
5. **Google Trends:** Input core service WITHOUT location modifier, filter by country, check "Rising"
6. **Google Keyword Planner:** Sort by Top of Page Bid (high CPC = commercial intent = competitor spend)

### Content Silo Strategy
- Target variations per post ("rug cleaning," "stain removal") instead of repeating primary
- Develop monthly content silos: one secondary keyword per post for topical variety
- Build a list of 12+ secondary keywords to sustain monthly posting cycles
- Track which secondary keywords drive GBP impressions and clicks monthly
- Retire underperforming keywords and replace with new variations quarterly

## GBP Posting Strategy

### Frequency Rules
**Minimum 5 posts/week (~20/month)**
- "What's New" posts: 3-5/week (daily workhorses)
- Event posts: 1 every 1-2 weeks (2-4/month)
- Offer posts: 1 every 1-2 weeks (2-4/month)

### Why 5+ Posts Per Week
- Google rewards consistent posting activity as a freshness signal
- Competitors posting 1-2x/month cannot match the topical breadth of 20+ monthly posts
- Each post creates an additional internal link within GBP's ecosystem
- Posting frequency compounds over months -- a 6-month archive of 120+ posts creates massive topical depth
- Low-effort competitors drop off after 1-2 months; consistency wins

### Post Type Details
- **What's New:** Target secondary keywords. These are the core SEO signal posts.
- **Event posts:** Set start/end dates correctly to prevent early expiration. Target primary service keyword. If the end date is set far enough in the future, event posts persist without expiring.
- **Offer posts:** Drive immediate conversions. Include specific offer details with CTA.
- Event and Offer posts must target the SAME primary service keyword as the current "What's New" silo.

### Post Copy Anatomy (6 Elements)
1. **Headline with keyword** at top -- first line of the post acts as the headline
2. **Layer secondary terms** naturally (no stuffing) -- weave related terms into the body
3. **State brand name** explicitly -- reinforces entity signals to Google
4. **Include owner's name** -- strengthens entity signals (real person = trust)
5. **Target location** -- city + neighborhood/locality drill-down for hyper-local relevance
6. **Anchor to Points of Interest** -- reference nearby landmarks, intersections, or well-known venues for geographic association

### Example Post Structure (Carpet Cleaning)
```
Professional Carpet Cleaning in East Madison, WI

Looking for deep carpet cleaning near the Capitol Square area?
[Brand Name] owner [Owner Name] uses hot water extraction
to remove embedded dirt and allergens from residential carpets.

Our Madison carpet cleaning team serves East Madison, Willy Street,
and the Atwood neighborhood. Located minutes from Olbrich Gardens.

Book your carpet cleaning appointment today!
[CTA Button -> Service Page URL]
```

### Additional Copy Rules
- Use TopicalRelevance.com to identify related entities and terms
- Google recognizes emojis as valid entities -- use relevant icons sparingly (1-2 per post max)
- Never put phone number in post body (Google rejects as policy violation)
- Use CTA buttons for phone numbers and URLs
- 1,500 characters per post -- don't force length; focus on relevance + conversion
- Each post should be self-contained and valuable even without clicking the CTA

## Umbrella Silo Strategy (Interlinking)

### Monthly Execution
- One core service keyword per month
- Each "What's New" post targets a separate secondary keyword
- All posts within the silo link to each other AND to the target service page
- At month end, you have 15-20 interlinked posts all pointing to one service page

### The Loop (Spokes)
Link Post 1 -> Post 2 -> Post 3 -> Post 4 -> back to Post 1. Closes the circuit.

**Building the loop:**
- When publishing Post 2, go back and edit Post 1 to add a link to Post 2
- When publishing Post 3, edit Post 2 to link to Post 3
- At the end of the month, edit the final post to link back to Post 1
- This creates a closed-circuit internal linking loop within GBP

### The Money Link (Hub)
Every post links to the SAME target page (service page) via CTA button. CTA button is the ONLY do-follow link available -- funnels maximum SEO value.

### Rules
- Use WhereGoes tool to verify no broken redirects in the loop
- Do NOT interlink Event or Offer posts -- standalone signals for homepage/landing pages
- Random linking wastes link equity -- create closed loops
- Start a new silo each month targeting a different service page

## Internal Linking (Google Ecosystem)

Select 1-2 Google properties per post:
- Google Sites, Google Maps/My Maps, Drive/Cloud stacks (PDFs, price lists), Reviews, YouTube

### External Property Syndication
Connect 2-4 external properties per post:
- Social profiles (Facebook, Twitter, LinkedIn)
- Top citations (Yelp, Yellow Pages) for NAP consistency
- Branded Web 2.0s (Tumblr, Blogger, WordPress)
- Alternate map embeds (BatchGeo)

### Syndication Tier Strategy
- **Tier 1:** GBP post links to your website + Google properties
- **Tier 2:** Syndicate GBP post content to social profiles with links back to the GBP post or website
- **Tier 3:** Build social signals (likes, shares, comments) to Tier 2 syndicated posts
- This creates a link pyramid amplifying each GBP post's authority

## Image SEO for Local

### Core Rules
- Stock photos are dead weight. Use original images (before/after, team photos)
- If no originals, use AI tools with text overlays containing service keywords
- Upload 5-10 images per week to GBP (minimum) to maintain freshness signals
- Mix image types: service photos, team photos, location photos, before/after shots

### OCR Exploitation
- Google uses OCR to read text within images -- most competitors overlook this
- Add keyword overlays with bold, clear sans-serif fonts (fancy scripts confuse OCR)
- Inject primary service keyword + location directly onto image
- Example overlay: "Affordable Carpet Cleaning Madison WI" in white text on a dark banner
- Google reads logos via OCR -- adding brand logo reinforces keyword-brand association

### EXIF Data Injection (Step-by-Step)
1. Open TheExifer.com and upload your image
2. In the Description/Caption field, enter: target keyword + full NAP + website URL
3. In the Artist/Author field, enter: owner/persona name
4. In the Copyright field, enter: brand name
5. Hardcode GPS coordinates (latitude/longitude) of business location in the GPS fields
6. Set rating to 5 stars within file data as a trust signal
7. Download the EXIF-injected image and upload to GBP

### File Metadata (Windows/Mac Properties)
- Right-click > Properties > Details: populate Title, Subject, Tags with primary keywords + NAP + persona name
- For AI/web-sourced images: use GeoSetter tool to add latitude/longitude, city, neighborhood
- Export as JPEG for optimal compression and EXIF compatibility (PNG strips EXIF data)

### File Naming
- Rename to keyword (carpet-cleaning-madison-wi.jpg)
- Use hyphens (read as spaces by search engines)
- Never upload generic names like IMG1234.jpeg
- Include location in every filename for local relevance

### Geo-Tagging with GeoSetter
- Free desktop tool for batch geo-tagging images
- Drag pin on map to exact business location
- Adds latitude, longitude, city, state, country to image metadata
- Process: open image in GeoSetter > navigate map to location > click to set coordinates > save
- Batch process multiple images at once for efficiency

## GBP Performance Tracking

### Key Metrics to Monitor Monthly
- **Search queries:** Which terms triggered your GBP listing
- **Profile views:** Total views of your GBP profile
- **Direction requests:** Users requesting directions (high-intent signal)
- **Phone calls:** Calls initiated from GBP listing
- **Website clicks:** Clicks to your website from GBP
- **Photo views:** How often your photos are viewed vs competitors
- **Post engagement:** Clicks on CTA buttons per post type
