# Local Ranking Factors & Advanced Tactics

## Ranking Factors Hierarchy (Jacky Chou)

1. **NAP consistency / citations** -- cannot be ignored
2. **Location proximity** to searcher
3. **Reviews** (quantity + quality)
4. **GBP optimization** (posts, photos, categories)
5. **CTR signals**
6. **Organic website SEO** (supports but secondary)

## Citation Building

### Jacky Chou Method
1. Buy 50 citations for $5 (Fiverr)
2. Index via Telegram indexer (SpeedyIndexBot)
3. Wait 2 weeks
4. ~50% of campaigns need only citations
5. Optional: 350 citations for $60 as bulk option
6. Citations should be country-specific (Canadian for Canadian businesses)

### Competitor Benchmarking
Google "[competitor name]" in brackets, check number of results. If they have ~1,300 results, you can beat them with 300-400 citations assuming vicinity match.

### PBN Citations
- Include NAP on top + naked URL + description matching GMB + AI-generated keyword content
- 200 PBN sites per batch
- 90%+ indexation rate (vs. 70-80% for regular citations)
- Never point PBNs directly to money site -- only tier citations
- Do NOT tier PBN citations (ruins the network)

### Citation URL Risk Levels
- **Low risk:** Naked URLs
- **Medium risk:** Nofollow links
- **High risk:** Dofollow (not recommended for organic side)

### Guest Posts for Local
- Only needed for highly competitive terms (accident lawyers, real estate agents)
- Include map embed + NAP + dofollow links
- Help with both local AND organic rankings (unlike citations which are local-only)

## Location Proximity Strategy

### Direct Correlation
Distance between GMB and searcher directly correlates to ranking.

### Location Options
- Own office (ideal)
- Co-working space (verify address doesn't have many existing GMBs)
- Buy a GMB location ($200-$400 from vendors -- risky)

### Competition Factor
- Low competition: can rank from 40+ km away
- High competition: must be in population-dense area
- Search target keyword to see recommended area (Google shows a circle)

## Reviews Strategy

### White Hat Methods
1. Always prompt clients for review
2. **Review funneler:** QR code -> landing page. 4+ stars -> Google review. 3 or fewer -> contact form
3. Always follow up with low-star reviewers -- if unresolved, they'll return

### Gray Hat
- Offer gift/incentive for 5-star review (comp drinks, keychain, raffle entry)
- Value of a 5-star review: approximately $10-$20

## CTR Manipulation (Advanced/Gray Hat)

### Formula
Impressions x CTR (based on position) = natural daily clicks / 30 = per-day number

### Example
Position 9, keyword 1,000 searches/month: 1000 x 1.9% = 19 clicks/month / 30 = 0.6 clicks/day. Only send 0.3 clicks/day.

### Rules
- **Always undershoot, never overshoot.** Undershooting = just another click. Overshooting = penalty.
- Use partial match keywords: "Vancouver realtor Indexy" boosts keyword association
- Manual CTR: search "[keyword] [business name]" and drive to office via Google Maps
- Ask employees/friends to do the same

## Rank and Rent Model

### Setup
- **Domain:** EMD works best (vancouverrealtor.com). No hyphens + .com preferred.
- **Hosting:** Cheap (Namecheap). Multiple sites: Cloudways + Cloudflare proxy.
- **Tech stack:** WordPress only.
- **Design/CRO:** CTA above fold on mobile and desktop.
- **Expectations:** Supplementary income, not $100K/month.

### Ranking Process
1. Buy 50 citations for $5. Index.
2. DO NOTHING for 4 weeks.
3. Expected: top 10 within 4-8 weeks for non-competitive keywords.
4. Only buy backlinks if not top 10 after 8 weeks.
5. If position ~10 after niche edit, buy 5 more links.

### Pricing Formula
Search Volume x CTR x Form Fill Rate x Lead-to-Client Ratio x AOV x Margin x Your Cut

**Benchmarks:**
- CTR at position 1: ~30%
- Form fill rate: ~15% (good landing page)
- Lead-to-client ratio: 5% (conservative)
- Margin estimate: 25%
- Your cut: 25-50%

### Cold Email Pitching
- Buy burner domain (try-[brand].com). Max 3 email accounts per domain.
- Configure SPF/DKIM/DMARC. Warm up 3+ weeks.
- Find leads: Outscraper (scrape by city), Fiverr for persona lists.
- Keep copy SHORT: "I rank on the front page for [keyword]. Would you be interested?"
- Results: 2% reply rate, 3 opportunities per campaign.
- Always follow up at least twice. LinkedIn automation may outperform email.

## Syndication & Backlink Tiering

### Tier 1 Syndication
After GBP post goes live, share to:
- Facebook & Instagram (share with images)
- LinkedIn (longer versions as LinkedIn Articles)
- Pinterest (specific boards linking back to services)
- Tumblr, Blogger, WordPress (short summaries + "Read More" link)
- Email & SMS (push high-value offers/events)

### Social Signals for Tier 1
Buy signals to make syndicated content look natural:
- Facebook: Shares (most important) + Likes
- Tumblr: Reblogs + Likes
- Pinterest: Repins (Saves)
- LinkedIn: Likes + Connections
- Vendors: SMMKings, JustAnotherPanel, YoYo Media

### Tier 2 Backlink Automation
- Link to Tier 1 assets, NOT directly to GBP
- Use GSA Search Engine Ranker "Contextual Link Blasts"
- Execute once/month for previous month's posts

### Event Submission Backlinks
- Local event submission sites are high DA, geo-specific "hidden gems"
- Point links to GBP Event Post URL for entity connection
- Find: "[City]" + "submit an event" / "events calendar" / "add event"
- Event title = exact same as GBP Event Post

## Monthly Local SEO Workflow

1. Keyword research (primary + 12+ secondary keywords)
2. Image creation & SEO (geo-tag, EXIF, OCR overlays, filenames)
3. Write posts (keyword-based)
4. Select URLs (interlinking targets)
5. Content calendar (spreadsheet)
6. GBP scheduler tool (automate publishing)
7. Interlink "What's New" posts (umbrella silo)
8. Syndicate to Tier 1 properties
9. Social signals to syndicated URLs
10. Push Tier 2 links to syndicated URLs

### Timeline Expectations
- Sustainable ranking dominance: 3-6 months of consistent implementation
- Rank and rent with $5 citations: top 10 within 4-8 weeks for non-competitive keywords
