# On-Page Optimization Skill

## What This Skill Covers

On-page SEO is the direct optimization of a single page's content, structure, and markup to rank better in Google. This skill covers the five critical elements that signal relevance to search engines:

1. **Title Tags & Meta Descriptions** - CTR and relevance signals
2. **Header Structure (H1-H3)** - Content hierarchy and keyword signals
3. **Schema Markup** - Rich snippets, knowledge graphs, E-E-A-T signals
4. **Image Optimization** - Alt text, filenames, lazy loading
5. **Internal Linking** - Authority distribution and topical signals

## Why On-Page Optimization Matters

**Ranking Impact**: On-page optimization accounts for ~5-10% of ranking lift when done correctly. While backlinks matter more, on-page is the foundation that makes backlinks work.

**CTR Impact**: Title tags and meta descriptions directly control click-through rates from search results. A poorly written title can reduce clicks by 30-50%.

**User Experience**: Proper heading structure and image optimization improve readability and reduce bounce rates, creating a positive feedback loop for rankings.

## Core Principle: Tier 1 vs Tier 2 Signals

**TIER 1 (Highest Impact):**
- Domain name
- URL structure (includes target keyword)
- Title tag (includes target keyword)
- H1 tag (includes target keyword)

**TIER 2 (Secondary):**
- H2-H6 tags
- Alt text
- Image filenames
- Meta description
- Body text keyword distribution

**Tier 3 (Minimal):** Font styling, CSS classes, keyword density (keyword stuffing is penalized)

Focus on Tier 1 first. These four elements account for most on-page ranking signals.

## Implementation Workflow

### Step 1: Audit Current State
Use Google Rich Results Test and your CMS to identify:
- Missing or duplicate title tags
- Meta descriptions that Google rewrites
- Missing H1 tags or multiple H1s per page
- Thin/missing alt text on images
- Broken internal linking structure

### Step 2: Prioritize Pages
Start with:
1. **Money pages** (product pages, high-intent pages that drive revenue)
2. **Pages ranking 11-50** (just outside top 10; easiest to move to top 3)
3. **Pages with high impression count but low CTR** (title/description rewrite opportunity)

### Step 3: Optimize Systematically
Use the reference guides in order:
1. Meta tag optimization first (fastest implementation)
2. Heading structure second (requires content review)
3. Schema markup third (validates with tools)
4. Image optimization fourth (bulk task)
5. Internal linking audit last (comprehensive site view)

### Step 4: Validate & Monitor
- Test schema with Google's Rich Results Validator
- Monitor CTR in Google Search Console month-over-month
- Track ranking changes for optimized pages

## Quick Start Checklist

Use this for rapid implementation on any page:

- [ ] Title tag: 50-60 characters, primary keyword first, benefit/unique angle, matches H1
- [ ] Meta description: 160 characters, keyword + secondary keywords naturally, compelling CTA
- [ ] H1 tag: One only, includes primary keyword, 20-70 characters
- [ ] H2-H3 structure: Logical hierarchy, 2-5 H2s, secondary keywords distributed
- [ ] Alt text: All images described, keyword naturally included (not stuffed)
- [ ] Image filenames: Match target keywords, hyphens between words
- [ ] Internal links: 3-5 contextual links to related pages, descriptive anchor text
- [ ] Schema markup: Article, Organization, or Service schema added and validated
- [ ] URL: Includes target keyword, lowercase, hyphens as separators, under 75 characters

## Common Mistakes to Avoid

1. **Keyword Stuffing in Headers** - "Best SEO Tools Best SEO Best Practices SEO" looks spammy, confuses Google
2. **Multiple H1 Tags** - Only one H1 per page; use H2-H6 for subsections
3. **Generic Alt Text** - "Image 123" provides no value; describe what's actually shown
4. **Ignoring Title Rewrite Signals** - If Google rewrites your title, adjust it to match your H1
5. **Duplicate Meta Descriptions** - Each page should have a unique meta description
6. **Overoptimized URLs** - Avoid "best-seo-tools-for-content-writers-guide-2024" (too long)
7. **Forgetting to Validate Schema** - Test with Google Rich Results Validator, not just schema.org validator
