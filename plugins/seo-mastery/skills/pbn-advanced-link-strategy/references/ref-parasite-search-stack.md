## AI Content Generation for Parasites

### Claude Prompt Engineering
1. Paste the social media prompt into Claude (preferred over ChatGPT for writing style)
2. Provide: company name, target keyword, target URL
3. Claude visits the URL, scrapes content, understands key points
4. Feed minimum 20 examples of existing posts (X, LinkedIn, etc.)
5. Claude analyzes writing style: tone, sentence length, vocabulary, humor, emoji usage
6. Generates 3 short-form posts (280 chars for X) and 3 long-form posts (~1,000 chars)

**Critical:** Minimum 20 post examples required for accurate style replication.

**Article prompt:** Requires 5 blog post URLs as writing samples (articles need more training material).

## Medium as a Parasite Property

**Key discovery:** Medium articles require 10+ real followers to index.

**If articles fail to index:** Delete old article, rewrite, publish as new with new URL.

**Content strategy:** Use article prompt with 5 blog post URLs as writing samples. Add outgoing links to Wikipedia (authority signal), own social profiles. Include one contextual link to target page.

## YouTube SEO for Parasite Campaigns

**Title:** Target keyword at the BEGINNING. Then make it clickable.

**Description structure:**
1. Target keyword as first line
2. Clickable URL to target page
3. Descriptive text
4. Timestamps (first timestamp 00:00 with target keyword as label)
5. Hashtags

**Thumbnails:** Include target keyword as readable text (Google reads text in images).

**Playlists:** Name playlist your target keyword -- playlists can rank on page one.

## Image SEO for Parasite Campaigns

- Save images as JPEG (not PNG) -- only JPEG allows metadata editing via right-click properties
- Name file your target keyword (e.g., "georadius-schema-generator.jpg")
- Add target keyword to Title/Subject field, brand name to Author field
- Google only shows same image once in image search -- create multiple variations
- Add keyword text + brand logo overlay to screenshots/product images

## Search Stacks (Derek Pierce)

### Core Philosophy
Build a network of properties that collectively dominate page one for target keywords.

### Stack Components
- Main money site
- Google Sites page
- Amazon S3 hosted static pages
- Claude Code-generated artifacts (published to GitHub Pages)
- Social media profiles (all parasite platforms)
- PBN sites
- Press releases
- YouTube channel + videos
- Secondary sites (built rapidly with templates + AI)

### Claude Artifact Templates
Three templates for different use cases:
1. "Best of" / Affiliate content (top products with CTA buttons, star ratings, tables)
2. Review posts (keyword header, CTA button, product details)
3. Informational/guide pages (structured content with FAQ schema)

### Secondary Site Rapid Deployment
- Use LocalWP/Flywheel templates + Claude Code for quick WordPress setup
- Deploy multiple secondary sites targeting keyword variations
- PBN itself can target long-tail versions while linking to money site for main term

## Standard Parasite SEO Types (Floate)

### Standard Parasite
Use high-authority third-party platforms (Gumroad, Medium, YouTube) to rank pages. For parasite pages needing links: 1-2 strong niche edits or social signals usually sufficient.

### Reverse Parasite
If you OWN a large high-authority site, you are running reverse parasite SEO. Internal linking between owned properties functions as an internal PBN.

### Local Parasite
Create assets that parasite another business's branded search terms. Name lead-gen asset after a nearby well-known business. Works best within same city. Strategy for lead generation, not client work.

## Web 2.0 PBN Strategy

- Use free platforms (WordPress.com, Blogger, Tumblr, Wix, Weebly, Medium)
- Each Web 2.0 maps to only ONE PBN (no cross-linking)
- Make them look legitimate: About page, contact form, regular content
- Use varied anchor text types across Web 2.0 links
