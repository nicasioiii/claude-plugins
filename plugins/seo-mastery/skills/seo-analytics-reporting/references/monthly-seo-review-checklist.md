# Monthly SEO Review Checklist (30-60 minutes)

**Length:** ~150 lines

## Week 1 (Data Collection)

### Data Gathering (15 minutes)
- [ ] Pull Looker Studio Executive dashboard (auto-refreshed)
- [ ] Export Google Search Console Performance report (CSV)
  - Date range: Last 30 days
  - Filter: Impressions > 0
  - Sort by: Clicks (highest first)
- [ ] Export GA4 Traffic report
  - Metric: Organic sessions + conversions
  - Comparison: Last month
- [ ] Check rank tracking tool (Ahrefs/SEMrush)
  - New keywords in top 10
  - Lost keywords from top 10
  - Ranking improvements (moved up 5+ positions)

### Calculation (10 minutes)
- [ ] Calculate organic revenue = organic sessions × CR × AOV
- [ ] Calculate ROAS = organic revenue ÷ SEO budget
- [ ] Calculate organic CPA = SEO budget ÷ conversions
- [ ] Calculate YoY growth = (this month - last year) ÷ last year

### Quick Wins Identification (10 minutes)
- [ ] Found 3 keywords gaining momentum (+10 positions this month)
- [ ] Identified pages with 10x+ traffic vs previous month
- [ ] Found keywords in positions 6-10 that can move to #1 with small optimization

## Week 2 (Analysis)

### Traffic Analysis (15 minutes)
- [ ] Organic traffic MoM change: ___ % (target: +10-15%)
- [ ] Conversion rate: ___ % (target: 2-5% depending on industry)
- [ ] New users vs returning: ___ % new (target: 40-60% new)
- [ ] Top 5 traffic sources by keyword: Document
- [ ] Traffic by page type distribution: Blog _% | Product _% | Service _%

### Ranking Analysis (10 minutes)
- [ ] Total top 10 keywords: ___ (track monthly for trend)
- [ ] Top 3 keywords: List and position
- [ ] Biggest ranking improvements: Document 5 keywords that moved up
- [ ] Biggest ranking losses: Document 5 keywords that moved down
- [ ] Keywords in "opportunity zone" (positions 11-20): Count = ___

### Conversion Analysis (15 minutes)
- [ ] Highest-converting keywords: List top 5
- [ ] Lowest-converting keywords that rank: Document for optimization
- [ ] Highest-converting pages: List top 5
- [ ] Pages with traffic but zero conversions: Mark for testing
- [ ] Customer LTV by traffic source: Organic = $ ___ (vs. paid)

### Content Performance (10 minutes)
- [ ] Top 5 articles by organic traffic this month
- [ ] Articles published this month: Count = ___
- [ ] Articles updated this month: Count = ___
- [ ] Articles that need updating (stale info): List 3

### Technical Health (10 minutes)
- [ ] Core Web Vitals: LCP: ___ms | FID: ___ms | CLS: ___ (all "Good"?)
- [ ] Indexation rate: __% (target: 90%+)
- [ ] Crawl errors in GSC: Count = ___ (any major issues?)
- [ ] Page speed issues: Any new problems this month? Y/N

## Common Issues During Review

| Issue | Fix |
|-------|-----|
| "Data doesn't match last month" | Check date ranges, filter settings |
| "Traffic up but revenue down" | Lower conversion rate (traffic quality issue) |
| "Rankings not improving" | Check keyword difficulty, backlinks, content quality |
| "Can't explain traffic spike" | Check for viral content, PR coverage, algorithm update |
| "New pages not ranking" | 1. Too new (need 2-4 weeks) 2. Weak internal links 3. Technical issues |

## Monthly Review Frequency

- **Monthly:** Full review (30-60 min)
- **Weekly:** Quick pulse (10 min) - is traffic on track?
- **Daily:** Real-time monitoring (5 min) - any issues?
