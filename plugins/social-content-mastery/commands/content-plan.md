# /content-plan

## Skills Activated
- `content-strategy`

## Prerequisites
Ask if the user has a brand foundation (niche, pillars, avatar). If not, recommend running `/build-brand` first.

## Workflow

### Step 1: Gather Inputs

Ask the user:
1. What are your content pillars? (If defined. If not, help define 3-5.)
2. What platform(s) are you creating for?
3. What's your current posting frequency and capacity?
4. Are you in the first 100 days (daily creation) or past it (batch creation)?
5. What formats have you tried? Any winners?

### Step 2: Generate Ideas Using Endless Idea Generator

Apply the 4A Framework + 10 Proven Approaches + Credibility layer to the user's pillars. Generate 30+ specific topic ideas.

Use the Lean Content Multiplication method:
- Cross pillars with content buckets (Futurism, Story, Actionable Advice)
- Apply Magical Multipliers (Tips, Stats, Steps, Lessons, Benefits, Reasons, Mistakes, Examples, Questions, Personal Stories)
- Consider audience archetypes if the user serves multiple segments

### Step 3: Assign Content Types

For each topic, recommend:
- Content type: Attraction (80%), Nurture (20%), Conversion (0% unless launching)
- Format: Based on user's winning format or equal-split testing protocol
- Platform-specific adjustments if user specifies a platform

### Step 4: Build the Calendar

Create a 30-day content calendar with:
- Day number
- Content pillar
- Topic/idea
- Format (reel, carousel, thread, long-form, etc.)
- Suggested hook direction (1-line hook concept)
- Content type (attraction/nurture/conversion)

### Step 5: Repurposing Plan

Map each piece of content to 1-2 repurposing opportunities:
- Which pieces can become threads/carousels on other platforms?
- Which hooks can be tested in different formats?
- Reference the 3-tier repurposing strategy (Basic, Advanced, Pro)

### Step 6: Batch Creation Schedule

If the user is past 100 days, provide a weekly batch creation schedule:
- Monday: Idea generation
- Tuesday: Copywriting
- Wednesday: Copy review
- Thursday: Design/filming
- Friday: Scheduling

### Output

Deliver:
- 30-day content calendar (table format)
- 5-10 hook concepts for the first week
- Repurposing map for top 5 pieces
- Batch creation schedule (if applicable)
- Recommended next step: run `/write-hook` or `/write-script` for specific pieces
