# /optimize-profile -- Platform-Specific Profile Optimizer

## INTAKE QUESTIONS (Ask All Before Auditing)

### Question 1: Platform Selection
**Which platform profile are you optimizing?**

| Choice | Skill Activated |
|--------|----------------|
| A) Instagram | `instagram-growth` + `brand-foundation` |
| B) LinkedIn | `linkedin-growth` + `brand-foundation` |
| C) TikTok | `tiktok-growth` + `brand-foundation` |
| D) Twitter/X | `twitter-growth` + `brand-foundation` |
| E) YouTube | `youtube-growth` + `brand-foundation` |
| F) All platforms (unified audit) | All platform skills + `brand-foundation` |

### Question 2: Current Profile
**Share your current profile details:**
- What is your current bio/headline?
- What is your profile picture? (describe it)
- Do you have a banner/cover image?
- What content is pinned/featured?
- What link is in your bio?

### Question 3: Brand Context
**Tell me about your brand:**
- What do you do? (in one sentence)
- Who do you help?
- What transformation do you deliver?
- What makes you different from others in your space?
- Do you have an existing brand foundation? (If yes, reference it. If no, suggest `/build-brand` first.)

### Question 4: Goals
**What's the primary goal of your profile?**
- A) Get more followers (growth-focused)
- B) Generate leads/clients (conversion-focused)
- C) Build authority/credibility (positioning-focused)
- D) Sell products/services (sales-focused)
- E) Build community (engagement-focused)

## OUTPUT FORMAT

After gathering intake answers, produce a profile optimization report:

### Section 1: Profile Audit (Current State)
Rate each element on a scale of 1-5 and explain why:

| Element | Score | Assessment | Fix Priority |
|---------|-------|-----------|-------------|
| Bio/Headline | X/5 | What works, what doesn't | High/Med/Low |
| Profile Picture | X/5 | Assessment | Priority |
| Banner/Cover | X/5 | Assessment | Priority |
| Pinned/Featured Content | X/5 | Assessment | Priority |
| Link/CTA | X/5 | Assessment | Priority |
| Username/Handle | X/5 | Assessment | Priority |
| Overall First Impression | X/5 | Assessment | -- |

### Section 2: Optimized Profile (Recommendations)

#### Bio/Headline
- Provide 2-3 rewritten bio options using platform-specific frameworks
- For LinkedIn: use headline template from `linkedin-growth`
- For Instagram: use the 4-line bio framework from `instagram-growth`
- For Twitter: use Big Idea principle from `twitter-growth`
- For TikTok: use the mission statement format from `tiktok-growth`
- For YouTube: use the transformation statement from `youtube-growth`

#### Profile Picture
- Specific recommendations based on platform norms
- Lighting, framing, expression guidance

#### Banner/Cover Image
- What to include (value prop, CTA, brand colors)
- Platform-specific dimensions and best practices

#### Pinned/Featured Content
- What to pin and why
- How to organize highlights (Instagram)
- Playlist strategy (YouTube)

#### Link & CTA Strategy
- What to link to (landing page, lead magnet, link tree)
- CTA placement and wording

### Section 3: Platform-Specific Extras

**Instagram:** Highlights strategy, grid aesthetic, Reels pinning
**LinkedIn:** Skills SEO, About section framework, banner templates
**TikTok:** Backend setup checklist, Q&A feature, connected platforms
**Twitter/X:** Pinned tweet strategy, pseudonymous best practices
**YouTube:** Channel description, playlist organization, channel trailer

### Section 4: Action Items
Numbered, prioritized list of specific changes to make, starting with highest-impact items.
