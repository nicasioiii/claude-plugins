# /write-script

## Skills Activated
- `write-scripts`
- `write-hooks`

## Workflow

### Step 1: Gather Context

Ask the user:
1. What topic/idea is this content about?
2. What platform is this for?
3. What format? (Short-form video, long-form video, LinkedIn post, Twitter thread, Instagram caption, carousel copy)
4. What is the goal? (Educate, inspire, entertain, convert)
5. Do you have a hook already? (If yes, use it. If no, generate one.)
6. Who is the target audience?
7. What is the ONE takeaway for the viewer?

### Step 2: Select Format and Structure

Based on the format:

**Short-form video (Reels/TikTok/Shorts):**
- Use the Video Flow Formula: Hook > Conflict > Resolution > CTA
- Or Educational format: Hook > Thing 1 > Thing 2 > Thing 3 > CTA
- Target: 15-60 seconds

**Long-form video (YouTube):**
- Use H.O.T. formula: Hook > Outcome > Testimonial/Trust
- Then deliver value using Educational or Challenge structure
- Target: 8-15 minutes

**LinkedIn post:**
- Use 4 LinkedIn Writing Rules
- First 200 chars = hook + cliffhanger
- Body in 1/3/1 rhythm
- Headers must be tangible, specific, actionable, skimmable
- End with Post Endings framework (6 categories)
- Target: ~2,800 characters

**Twitter thread:**
- First tweet = sales pitch (must earn attention)
- Each tweet reveals NEW information (fast Rate of Revelation)
- End with TL;DR + CTA
- Target: 3-7 tweets

**Instagram caption:**
- First line = hook (before fold)
- Body delivers value
- End with question or CTA to drive comments
- Target: Concise, scannable

### Step 3: Write the Hook

If user doesn't have a hook, generate one using write-hooks principles:
- Apply 7 Spoken Archetypes
- Aim for max alignment across 4 components
- Score with CGOVE
- Offer 2-3 hook options

### Step 4: Write the Body

Apply the relevant structure from Step 2. Key principles:
- **1/3/1 rhythm** for all text-based content
- **Rate of Revelation** -- every sentence must add NEW information
- **Golden Intersection** -- combine personal story with actionable advice when possible
- **AIDA flow** -- Attention > Interest > Desire > Action
- Every sentence justifies its existence
- Embellish with opinion and personality
- Viewers could ask ChatGPT -- they watch/read YOU for the human element

### Step 5: Write the CTA

Select appropriate CTA type:
- Follow / Comment trigger / Save / Share / Link / Engagement / DM / Subscribe
- Maximum 2 seconds at end of video, or in caption
- Match CTA to content goal

### Step 6: Apply Post Ending (for text posts)

Use one of the 6 ending categories:
1. Connect with successful outcome
2. Connect with bad outcome
3. Challenge
4. Motivate
5. Summarize
6. Ask for a sale

### Step 7: Review and Compress

- Cut all unnecessary words (compression process)
- Check Rate of Revelation -- delete repeated information
- Read aloud for flow
- Verify hook is the strongest version
- Platform-specific formatting (line breaks, character limits, safe zones)

### Output

Deliver:
- Complete script/post with hook, body, and CTA clearly labeled
- Estimated duration (for video) or character count (for text)
- 1-2 alternative hooks if applicable
- Visual/audio direction notes (for video)
- Recommended post ending category (for text)
- Next steps: `/film-plan` (for video) or scheduling recommendations
