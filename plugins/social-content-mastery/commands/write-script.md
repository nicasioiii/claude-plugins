# /write-script -- Complete Script & Post Generator

## INTAKE QUESTIONS (Ask All Before Writing)

### Question 1: Content Type
**What are you creating?**

| Choice | Script Approach |
|--------|----------------|
| A) Short-form video (Reel, TikTok, Short) | 30-60 second script, tight structure, visual notes |
| B) Long-form video (YouTube) | Full script with sections, B-roll notes, music cues |
| C) LinkedIn text post | Text post with hook, sections, CTA, engagement question |
| D) LinkedIn carousel text | 6-10 slide text with hook slide and CTA slide |
| E) Caption for existing video | Description with hook, value, CTA |

### Question 2: Video Format (for video content)
**Which format does this follow?**
- A) Educational / How-to
- B) Challenge / Three-act structure
- C) Vlog / Day-in-the-life
- D) Reaction / Commentary
- E) Life lesson / Personal story
- F) News story / Commentary
- G) Comparison / Tier list
- H) Not sure -- help me choose

### Question 3: Topic & Main Point
**What is this content about?**
- What is the ONE main point?
- What does the viewer gain from watching/reading?
- What is your personal experience or angle on this topic?

### Question 4: Target Viewer
**Who is this for?**
- Describe your ideal viewer in one sentence
- What awareness level are they at? (Unaware / Problem-aware / Solution-aware / Expert)

### Question 5: Scripting Level (for video)
**How tightly do you want this scripted?**
- A) Fully scripted (every word, teleprompter-ready)
- B) Guided talking points (key beats scripted, delivery improvised)
- C) Loose outline (general direction, mostly improvised)

### Question 6: CTA Goal
**What do you want the viewer to do after?**
- A) Comment a keyword for a free resource
- B) Follow for more content
- C) DM you for information
- D) Click a link (bio, description)
- E) Just engage (like, share, save)
- F) No hard CTA -- just provide value

## GENERATION PROCESS

### For Short-Form Video Script (Choice 1A)
1. Load `write-scripts` SKILL.md
2. Load `write-hooks` SKILL.md
3. Select format structure from the 8 viral formats
4. Generate hook using /write-hook process (or provide 2 options)
5. Build body using chosen format structure:
   - Educational: Hook -> Thing 1 -> Thing 2 -> Thing 3 -> CTA
   - Challenge: Hook -> Setup -> Conflict -> Resolution
   - Story: Hook -> Emotional spike -> Story -> Takeaway
6. Apply 4 storytelling revision rules
7. Add visual notes: B-roll suggestions, shot type changes, text overlays
8. Add audio notes: music mood, SFX cues, silence moments
9. Target: 200 words for 40-60 seconds

### For Long-Form Video Script (Choice 1B)
1. Load `write-scripts` SKILL.md
2. Load `ref-storytelling-frameworks.md`
3. Apply Gusten Sun 5-part structure: Hook -> Main Big Idea -> Bullet Points -> Stories -> CTA
4. Apply But-So Foreshadow framework for body
5. Include storytelling elements (4 essential elements, emotional journey)
6. Match scripting level to user preference
7. Add section-by-section visual and audio notes

### For LinkedIn Text Post (Choice 1C)
1. Load `write-scripts` SKILL.md
2. Load `ref-linkedin-writing.md`
3. Select content style (Steps, Stats, Mistakes, Lessons, Examples)
4. Load `write-hooks` SKILL.md for LinkedIn hook (first 200 chars)
5. Build post using E02 structure:
   - Hook (200 chars) -> Introduction -> Sections -> Conclusion
6. Alternate formatting (bullets, paragraphs) across sections
7. Stay under 2,800 characters
8. Add engagement question at end
9. Suggest CTA approach (autoplug, comment trigger, or profile link)

### For LinkedIn Carousel (Choice 1D)
1. Load `ref-linkedin-writing.md`
2. Apply 10-slide framework
3. Write text for each slide (one idea per slide, minimal words)
4. Suggest visual approach per slide

### For Video Caption (Choice 1E)
1. Apply caption structure: Hook -> Value -> CTA
2. Simple sentences, list format
3. Include CTA with question before it

## OUTPUT FORMAT

### For Video Scripts
1. **Hook**: Spoken hook + text overlay + visual suggestion + audio cue
2. **Full Script**: Complete text with section markers
3. **Visual Notes**: B-roll suggestions, shot changes, text overlays per section
4. **Audio Notes**: Music mood per section, SFX cues, silence moments
5. **Script Stats**: Word count, estimated duration, reading pace note
6. **Revision Checklist**: Results of 4 storytelling revision rules applied
7. **Filming Tips**: Scripting level reminder, energy/delivery notes

### For LinkedIn Posts
1. **Hook**: First 200 characters (before "See more")
2. **Full Post**: Complete text with formatting applied
3. **Character Count**: Total characters (must be under 2,800)
4. **Content Style**: Which of the 5 styles used
5. **Engagement Question**: End-of-post question
6. **CTA**: Autoplug or comment trigger
7. **2 A/B Hook Variations**: Alternative hooks to test

### For All Types
- **"You/Your" Check**: Confirmed within first two sentences
- **One Main Point**: Clearly identified
- **Platform Rules**: All platform-specific formatting applied
