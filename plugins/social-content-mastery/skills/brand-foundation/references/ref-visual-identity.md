# Visual Identity Reference

## RULE OF TWOS -- BEGINNERS (Frabrand, E01)

Start with absolute minimums to avoid overwhelm:
- **2 fonts maximum** -- one for headers, one for body text
- **2 main colors** -- one primary, one accent
- **2 main imagery styles** -- e.g., photography + flat illustration

This constraint forces consistency. Most beginner brand inconsistency comes from using too many visual elements.

---

## RULE OF THREES -- INTERMEDIATE (Frabrand, E01)

After establishing basics, expand slightly:
- **Maximum 3 colors** -- 1 main + 2 accent (60/30/10 ratio)
- **Maximum 3 font types** -- header, subheader, body
- **Maximum 3 visual styles** -- e.g., photography, icons, textures

---

## COLOR SYSTEM

### Selection Process

1. Choose a primary color that reflects your brand personality
2. Use contrast checker tools (Coolors, Huemint) to find complementary accents
3. Test readability on mobile screens in bright sunlight
4. Document hex codes and usage rules

### Usage Guidelines

| Application | Color Rule |
|---|---|
| Background | Primary or white/off-white |
| Text | High contrast against background (dark on light or light on dark) |
| CTAs/highlights | Accent color only -- used sparingly for maximum impact |
| Borders/dividers | Muted version of primary or neutral gray |

### Platform Consideration

Design for mobile consumption in bright sunlight. Colors that look vivid on your monitor may wash out on a phone screen outdoors. Test with brightness at maximum.

## FONT SYSTEM

### Font Pairing Rules

1. One serif + one sans-serif is the safest combination
2. Never pair two similar fonts -- contrast is the point
3. Header font can be decorative; body font must be highly readable
4. Google Fonts and Font Squirrel for free commercial-use fonts
5. Test at small sizes (captions on phone screens)

### Readability Hierarchy

- Headers: 20-30% larger than body text
- Subheaders: 10-15% larger than body
- Body: minimum 14pt equivalent for mobile readability
- Captions: bold weight to compensate for small size

## IMAGERY STYLE

### Photography vs. Graphics

- Photography: more authentic, personal, trust-building
- Graphics/illustrations: cleaner, more professional, easier to template
- Most personal brands use photography for A-roll content and graphics for educational content

### Consistency Rules

- Same filter/preset across all photos
- Same lighting style (warm vs. cool, bright vs. moody)
- Same composition tendencies (centered vs. rule-of-thirds)
- Create presets in editing apps and reuse consistently

## DESIGN PRINCIPLES FOR SOCIAL MEDIA (Frabrand, E01)

### Core Principles

- **Functionality over form** -- unless you are selling design services
- **Clarity over cleverness** -- if they don't understand it, they scroll
- **White space is as important as filled space** -- let elements breathe
- **Start with few elements, add only if absolutely necessary**
- **Left-align text for anything over 2 lines** -- matches Western reading pattern
- **High contrast is essential** -- test with contrast checker tools

### F-Pattern Reading

People scan content in an F pattern: top-left > right > down-left > right (shorter scan) > down. Place most important information along this F shape.

### Visual Hierarchy

The order elements pull attention is determined by:
1. **Placement** -- center draws first
2. **Size** -- larger text = more attention
3. **Weight** -- bold vs. regular
4. **Color contrast** -- high contrast = more attention

### Mobile-First Design

- All social content is consumed primarily on mobile
- Design at mobile viewport first, not desktop
- Touch targets for interactive elements: minimum 44px
- Safe zones: avoid placing critical elements in corners where platform UI overlays (like/share/comment buttons)

### Instagram Technical Specs

- 4:5 ratio recommended (1080 x 1350 pixels)
- Profile grid shows 1:1 ratio -- plan first slide to look good with top/bottom cut off
- Plan for 135px cut on top and bottom in grid view
- Cover image selection matters for grid aesthetics

## BRAND STYLE GUIDE PROCESS (Dickie Bush, E02)

### Step 1: Define Brand Parameters

- Target audience description
- 3-5 personality adjectives
- Visual direction (modern/classic, bold/subtle, minimal/detailed)
- Tone of voice (professional/casual, serious/playful)

### Step 2: Visualize with Gamma

Create a 10-slide brand identity deck:
1. Brand name and tagline
2. Mission statement
3. Target audience
4. Brand values
5. Brand personality
6. Color palette with hex codes
7. Typography system
8. Imagery style with examples
9. Do's and don'ts
10. Application examples (post mockups)

### Step 3: Apply with Canva Templates

- Create master templates for each content format (posts, carousels, stories, banners)
- Lock brand colors and fonts into Brand Kit
- Design 3-5 template variations per format
- Share templates with team via Canva Teams

## BRAND MOTHERSHIP DOCUMENT (Frabrand, E01)

### What It Contains

Single reference document combining:

**Section 1: Brand Core**
- Mission statement
- Big Promise
- 3-5 brand values
- Brand archetype(s)
- Spectrum positioning

**Section 2: Brand Persona**
- Tone of voice guidelines (with examples)
- Words we use vs. words we avoid
- Communication style rules
- Content personality traits

**Section 3: Brand Visuals**
- Color palette with hex codes and usage rules
- Typography system with font names and hierarchy
- Imagery style guidelines with reference examples
- Logo usage rules (if applicable)
- Template links

### How to Use It

- Share with contractors, editors, designers before starting work
- Reference when content creation feels "off-brand"
- Update quarterly as brand evolves
- Single source of truth for brand consistency across collaborators
