## 7 LinkedIn Headline Variations (220 chars max, E02)

The headline is the single most visible element across LinkedIn -- it appears on every post, every comment, every search result.

| # | Pattern | Template |
|---|---|---|
| 1 | **The Outcome** | I help [audience] use [mechanism] to [achieve outcome with number] |
| 2 | **The Aspirational Brand** | [Unique brand]. Building to [$goal] and sharing everything along the way |
| 3 | **The Credible Talker Bio** | I talk about [topic]. [Result for # of audience]. [Credible Experience] |
| 4 | **The Credible Category Bio** | [Qualitative Credibility]. [Numbers]. [Category POV] |
| 5 | **The Specialist Bio** | [What you do] for [audience]. [Position] @ [company]. [Description]. [Social proof] |
| 6 | **Specific Transformation** | I help [even more specific audience] achieve [specific transformation] \| [methodology] |
| 7 | **Role + Niche** | [Role] \| I help [niche audience] solve [specific problem] so they can [outcome] |

### Headline Rules
- Write 3 variations, each <= 220 characters
- Include audience + mechanism + result
- Choose the clearest one (not the cleverest)
- Test for 2-4 weeks, then iterate based on profile visit data

## About Section (7 Psychological Frameworks, E02)

Write 250-300 words using one of these frameworks:

### 1. Problem-Agitation-Solution
Problem -> agitate (FOMO, costs of inaction) -> solution + social proof + CTA

### 2. Achievement-Realisation-Results
Authority marker -> personal pivot/realization -> results you create for others + CTA

### 3. Story-Driven Relatability
Backstory -> "I've been there" connection -> how you help now -> social proof + CTA

### 4. Contrarian Insight
Common assumption -> flip it with evidence -> fresh perspective + proof + CTA

### 5. WHO-WHAT-HOW
Who you help -> what you help achieve -> how (your process/method) + proof + CTA

### 6. Future Pacing
Paint desirable future -> identify gap between now and then -> your method as bridge + CTA

### 7. Authority-First Showcase
Big wins upfront -> unique insight/philosophy -> tangible client benefits + CTA

### Writing Style
"Write like you're explaining your work to someone at a barbecue, not pitching them."

### About Section CTAs
Turn bottom half into a map directing people to multiple destinations:
- Justin Welsh model: extended bio on top, links to different resources below
- Include lead magnet link, best content link, booking link

| Bucket | Purpose | Examples |

| **Adjacent Skills** | Reinforcing skills | Storytelling, Leadership, Personal Branding, Public Speaking |
| **Industry Keywords** | Terms prospects search | B2B Marketing, SaaS Growth, Creator Economy |

### Skills Rules

- Every skill must be specific, search-friendly, relevant to positioning
- 20-25 skills total across all 4 buckets
- Remove generic skills that dilute positioning

## Connection Request Strategy (E02)

### LinkedIn Allows 200 Connection Requests Per Week
- This is a unique LinkedIn advantage: grow audience at will through deliberate connection strategy
- 200 requests/week = potentially 200 new followers every week

### Strategic Connection Building
Build two lists:
1. **25 creators/accounts** similar to you (serve same audience, different offers)
2. **25 big influencers** your buyers follow (their comment sections are where your buyers hang out)

### Connection Request Rules
- Send 5-10 personalized requests per day
- Reference a post or specific challenge in the request message
- Never spam connect with random people -- untargeted connections poison your test group
- Your network IS your niche: algorithm distributes content to connections first

## LinkedIn-Specific Posting Rules (E02)

### The First 200 Characters Are Everything
- LinkedIn truncates posts to first ~200 characters before "See more" button
- These 200 characters are the sales pitch, hook, and consideration vehicle
- If nobody clicks "See more," the algorithm thinks nobody is interested and post dies
- First 200 characters must be both a HOOK and a CLIFFHANGER

### Post Structure Framework (E02)
- **Hook** (first 200 chars): ~200 characters, must compel "See more" click
- **Introduction** (~300-400 chars): Opening hook, the "promise" (3-5 sentences/bullets), the "solution" (1-2 sentences), transition sentence
- **Each main section** (~300-450 chars): Declarative opening (the "what") -> 3-5 tactical pieces (the "how") -> one-sentence insight (the "why")
- **Conclusion** (~100-200 chars)
- **CRITICAL:** Alternate format between bulleted lists and paragraph style across sections
- Total: stay under 2,800 characters (of LinkedIn's 3,000 max). Non-negotiable

### Content Format Quick Reference (E02)
| Format | Length | Dwell Time | Best For |
|---|---|---|---|
| Short text post | 50-120 words | Low-Med | Testing topics, outlines as content |
| Long text post | 200-450 words | Med-High | Deep value, frameworks, stories |
| Carousel (PDF) | 6-10 slides | High | Step-by-step, frameworks, high saves |
| Infographic | 1 page | High | Process flows, checklists, comparisons |
| Photo + text | Any length | Med | Authenticity, scroll-stopping |
| Short video | 30-60 seconds | High | Algorithm boost, personality, reach |
| Screenshot + text | Any length | Med | Social proof, results, authenticity |

### Posting Cadence (E02)
- Post 3-5 days per week
- Mix: 2 short posts (50-120 words), 1 long post (200-450 words), 1 visual (carousel/infographic) every 1-2 weeks
- Video when possible (algorithm currently prioritizing)
- Maintain predictable cadence/windows each week -- teach the feed when to expect you
