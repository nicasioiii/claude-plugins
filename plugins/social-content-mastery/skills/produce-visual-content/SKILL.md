## Core Design Principles for Social Media (E01 Frabrand)

### The Three-Limit Rule
- Maximum **3 colors** (1 main + 2 accent)
- Maximum **3 font types** (1 heading + 1 body + 1 accent/highlight)
- Maximum **3 visual styles** (maintain consistency, not chaos)

### Design Philosophy
- **Functionality over form** -- unless selling design services
- **Clarity over cleverness** -- viewers scan in milliseconds
- **White space is as important as filled space** -- crowded = ignored
- **Start with few elements, add only if absolutely necessary**
- **Design for mobile consumption in bright sunlight**

### Reading Patterns
- **F-pattern reading:** People scan top-left > right > down. Place most important information top-left.
- **Left-align text for anything over 2 lines** (matches Western reading: left-to-right, top-to-bottom)
- **People don't read -- they scan.** They cluster 2-4 words, then 4-5 clusters into paragraphs. Vary sentence length to maintain novelty.

### Visual Hierarchy
The order in which elements pull attention is determined by:
1. **Placement** -- Center draws first
2. **Size** -- Larger text = more attention
3. **Weight** -- Bold vs. regular
4. **Color contrast** -- High contrast = more attention

### Contrast Rules
- High contrast is essential -- test with contrast checker tools
- Dark text on light background or light text on dark background
- Avoid medium-contrast combinations (grey on white, light blue on white)
- Test readability on phone screen in outdoor lighting conditions

## Instagram Technical Specs

| Format | Dimensions | Aspect Ratio | Notes |
|---|---|---|---|
| Feed Post (square) | 1080 x 1080 | 1:1 | Classic, works for all content |
| Feed Post (portrait) | 1080 x 1350 | 4:5 | **Recommended** -- takes up more screen space |
| Reels / Stories | 1080 x 1920 | 9:16 | Full-screen vertical |
| Carousel slides | 1080 x 1350 | 4:5 | Same as portrait post |
| Profile grid view | Shows 1:1 crop | 1:1 | Plan first slide to look good with top/bottom cut off |

**Grid Planning Note (E01 Frabrand):** Profile grid shows 1:1 ratio. When posting 4:5 images, 135px is cut from top and bottom in grid view. Design first slides with this crop in mind.

### Safe Zones
- Respect platform UI element positions (like/share/profile/comment buttons)
- Keep critical text and visuals away from bottom 15% and top 10%
- Test by overlaying platform UI screenshot before publishing

## Carousel Creation Framework

### Why Carousels Work
- Higher average engagement than single images on Instagram and LinkedIn
- Multiple slides = more time on post = better algorithm signal
- Educational content naturally fits the slide format
- Encourages saves (valuable reference material)

### Carousel Structure (7-10 slides recommended)
| Slide | Purpose | Content |
|---|---|---|
| 1 (Cover) | Hook + stop the scroll | Bold headline, curiosity element, branded design |
| 2 | Context/setup | Why this matters, problem statement |
| 3-8 | Value delivery | One key point per slide. Clear, scannable. |
| 9 | Summary or takeaway | Recap the key points |
| 10 | CTA | Save, share, follow, comment, link in bio |

### Carousel Design Rules
- **One idea per slide** -- don't overload
- **Consistent visual template** -- same font, colors, layout across slides
- **Slide 1 must work as a standalone image** -- it appears in the feed grid
- **Use numbered slides** (1/10, 2/10) to show progress and encourage swiping
- **Large, readable text** -- minimum 24pt on mobile (test by shrinking preview)
- **Brand watermark or handle** on every slide (people screenshot and share)

### Carousel Cover Slide Formulas
- **"X Things That [Outcome]"** -- listicle format, curiosity
- **Bold Statement + "Swipe >"** -- direct instruction to engage
- **Before/After Preview** -- show transformation
- **Question** -- "Are you making these mistakes?"
- **Statistic** -- shocking number + context

## Infographic Design

### When to Use Infographics
- Data-heavy content (statistics, research, comparisons)
- Process/workflow visualization
- Timeline or chronological content
- Comparison charts (vs. format)

### Infographic Design Rules
- **Top-to-bottom flow** -- guide the eye downward
- **Section headers** -- break content into scannable chunks
- **Icons > text** -- visual representations where possible
- **Consistent color coding** -- categories get their own colors
- **Source citation** -- credibility matters for data-driven content
- **Brand footer** -- logo, handle, website at bottom

### Tools for Infographic Creation
- Canva (templates available, drag-and-drop)
- Figma (more control, steeper learning curve)
- Piktochart (infographic-specific)
- Adobe Illustrator (professional)

## Text-on-Screen Design

### Text Overlay Principles
- **Maximum 7 words per text element** on screen at any time
- **High contrast** between text and background
- **Background dim/blur** behind text for readability if placed over video/image
- **Consistent positioning** -- same text placement across content for brand recognition
- **Animation** -- subtle entrance animations increase engagement (slide in, fade in, pop)

### Text Hook Structures (Visual Hooks for Video)
Text that appears on screen in the first 1-3 seconds of a video:

| Type | Example | When to Use |
|---|---|---|
| Question | "Why do 90% of creators fail?" | Curiosity, pain point |
| Statement | "This changed everything." | Bold claim, authority |
| List preview | "5 mistakes killing your content" | Promise of value |
| Contrarian | "Stop doing this immediately." | Pattern interrupt |
| Result | "$10K in 30 days. Here's how." | Social proof, aspiration |

### Font Selection for Social
| Category | Best For | Examples |
|---|---|---|
| Bold sans-serif | Headlines, hooks, CTAs | Montserrat Bold, Bebas Neue, Impact |
| Clean sans-serif | Body text, captions | Inter, Open Sans, Lato |
| Script/handwritten | Accent text, personal touch | Caveat, Pacifico (use sparingly) |

## Visual Branding Consistency

### Brand Visual Checklist
- [ ] Consistent color palette across all content (max 3 colors)
- [ ] Consistent font family (max 3 fonts)
- [ ] Logo or handle visible on all shareable assets
- [ ] Consistent layout templates for recurring content types
- [ ] Profile grid aesthetic planned (for Instagram)
- [ ] Thumbnail style recognizable (for YouTube)

### Building Visual Templates
1. **Design 3-5 base templates** in Canva, Figma, or your tool of choice
2. **Create for each recurring format:** Carousel template, quote post template, tip post template, thumbnail template
3. **Lock brand elements:** Colors, fonts, logo placement are fixed. Only content changes.
4. **Save as brand kit** in Canva or equivalent tool
5. **Document in brand style guide** for consistency across team members

### Visual Identity Connection
The visual content you produce must align with your brand foundation:
- **Rule of Twos/Threes** (from brand-foundation): 2-3 colors, 2-3 fonts, consistent imagery style
- **Brand archetypes** influence visual style: Sage archetype = clean, minimal. Rebel = bold, high-contrast.
- **Color psychology matters:** Blue = trust, Red = urgency, Green = growth, Yellow = optimism

## Platform-Specific Visual Notes

### Instagram
- 4:5 portrait (1080x1350) for maximum screen real estate
- Grid aesthetic matters for profile visitors
- Carousel posts get highest engagement of static formats
- Visual consistency builds recognizable brand in Explore page

### LinkedIn
- Carousels (uploaded as PDF documents) perform exceptionally well
- Professional, clean design aesthetic
- Data-driven infographics get high engagement
- Blue and neutral tones tend to match platform feel (but brand colors override)

### Twitter/X
- Images with text overlays get more engagement than text-only
- 16:9 or 4:3 aspect ratios for images
- Simple, bold graphics with minimal text
- Memes and visual metaphors perform well

### YouTube
- Thumbnails are CRITICAL -- see ref-thumbnail-design.md for complete system
- Channel art: 2560x1440, but safe area is 1546x423 (center)
- End screen design: plan last 20 seconds of video for end screen overlay
